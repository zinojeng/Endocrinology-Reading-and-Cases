## 🧠 一頁總複習（跨章高頻記憶點）

### 致病機轉兩條路線
- 🔥 **致癌基因路線 (proto-oncogene, gain-of-function)**：MEN2 / MEN3 = ***RET*** 活化 →「開火」。
- 🛑 **抑癌基因路線 (tumor suppressor, loss-of-function, two-hit)**：MEN1 (*menin*)、MEN4 (*CDKN1B*/p27)、MEN5 (*MAX*) →「踩煞車」。
- 口訣：**「2、3 開火，其餘踩煞車」**。

### 五型 MEN 速覽
| 型別 | 基因 (染色體) | 核心腺體 |
|------|---------------|----------|
| MEN1 | *MEN1* (11q13) | **3P**：Parathyroid · Pancreas · Pituitary |
| MEN2A | *RET* (10q11) | MTC + Pheo + Parathyroid |
| MEN3 (MEN2B) | *RET* (10q11) | MTC + Pheo + mucosal neuroma / marfanoid（**無 PHPT**） |
| MEN4 | *CDKN1B* (12p13) | MEN1-like（副甲狀腺 + 腦下垂體為主） |
| MEN5 | *MAX* (14q23) | Pheo/PPGL 為主 |
