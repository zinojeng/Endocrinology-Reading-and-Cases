# Ch.44 單基因型與其他免疫內分泌症候群（Monogenic & Other Immunoendocrine Syndromes）

> 來源：Williams Textbook of Endocrinology, Ch.44 The Immunoendocrinopathy Syndromes — 單基因型免疫內分泌症候群（IPEX / CTLA4 / LRBA / STAT3-GOF / Anti–insulin receptor Ab / Wolfram / Omenn / 染色體疾病）

---

## 🧬 一、IPEX Syndrome（Immunodysregulation Polyendocrinopathy Enteropathy X-Linked）

### 基本資料
- **MIM**：<u>340790</u>、<u>300292</u>；首次描述於 <u>1982</u> 年。
- **遺傳模式**：<u>X-linked recessive</u>（X 性聯隱性）→ 主要影響**男嬰**。
- **致病基因**：<u>*FOXP3*</u> 突變。
- 是一種罕見、會造成**多重自體免疫疾病**並導致**早夭**的失調。

### 🔬 致病機轉（核心觀念）
- *FOXP3* 編碼 **forkhead class of winged helix transcription factor**（forkhead box protein 3）。
- FOXP3 表現於 <u>CD4⁺/CD25⁺ regulatory T cells（Treg）</u>。
- 這些 Treg 能**抑制其他 T 細胞的活化**（suppress activation of other T cells）。
- 👉 *FOXP3* 突變 → **無法生成 Treg** → 失去免疫耐受 → IPEX。
- **scurfy mouse** 是關鍵動物模型：
  - 胸腺移植自 scurfy mouse 進入免疫缺陷鼠 → **傳遞疾病**；移入免疫健全鼠 → **不傳遞**。
  - 注射正常 T 細胞可**救回（rescue）**表現型。
  - 👉 證明這是一種「regulatory cell（調節型細胞）」相關的疾病。
- Linkage analysis：X 染色體 <u>Xp11.1–q13.3</u>（17-centimorgan 區段）與 IPEX 相關。

### 🩺 臨床三主徵（必背）
1. **極早發 Type 1A diabetes**（very early onset T1DM）
2. **嚴重 enteropathy**（腸病變）→ failure to thrive（生長遲滯）
3. **Dermatitis**（皮膚炎）

- ⚠️ 若不積極治療，通常在**出生後頭一兩年內死亡**。
- 其他表現：atopy、thrombocytopenia、hemolytic anemia、hypothyroidism、lymphadenopathy、nephropathy、alopecia。
- **免疫評估**：<u>IgE 上升</u> + <u>eosinophilia（嗜酸性球增多）</u>。

### 基因型–表現型關聯
- 較**輕**表現型 → point mutation 或不導致 FOXP3 完全缺失的小片段缺失。
- 同一突變內表現型仍有顯著差異 → 提示其他基因 / 環境因素參與。

### 💊 治療
- **針對各組成疾病**支持治療：
  - 嚴重 enteropathy → **bowel rest + parenteral nutrition**（腸道休息＋全靜脈營養）。
  - Diabetes → insulin therapy。
- **穩定期免疫抑制**：calcineurin inhibitors。
- **根治性治療**：<u>Hematopoietic Stem Cell Transplantation（HSCT）</u>
  - HSCT 可**逆轉 enteropathy 與其他自體免疫成分**。
  - ⚠️ 但**已建立的 T1DM 與甲狀腺疾病一般不會緩解**（部分接受 conditioning immunotherapy + HSCT 者有 T1DM 逆轉案例報告）。
  - 腸道免疫系統重建**可能比骨髓重建更晚**發生。
- **未來方向**：將功能性 *FOXP3* gene 送入淋巴球的**基因治療**（gene therapy），輔助或最終取代 HSCT。

### 🧠 IPEX 表型口訣
> **IPEX** ≈ 「**I**mmune dysregulation, **P**olyendocrinopathy, **E**nteropathy, **X**-linked」
> 臨床記：**「3 大 D」= Diabetes（T1DM）+ Diarrhea/enteropathy + Dermatitis**，加 **高 IgE / 嗜酸性球**。

### ⚠️ IPEX vs APS（最容易考的鑑別）

| 比較項目 | **IPEX** | **APS-I (APECED)** | **APS-II** |
|---|---|---|---|
| 致病基因 | <u>*FOXP3*</u> | *AIRE* | 多基因（HLA 等） |
| 機轉 | **Treg 生成缺陷**（周邊耐受失敗） | **胸腺中央耐受**失敗（周邊抗原表現↓） | 多基因免疫失調 |
| 遺傳 | **X-linked recessive** | autosomal recessive | 多基因 / 偶有家族 |
| 發病年齡 | **新生兒 / 嬰兒期極早發** | 兒童期 | 成人為主 |
| 標誌組合 | **T1DM + enteropathy + dermatitis** | 念珠菌感染 + 副甲狀腺低下 + 腎上腺低下 | Addison + 自體免疫甲狀腺 ± T1DM |
| 特殊檢驗 | 高 IgE、嗜酸性球↑ | — | — |
| 預後 / 治療 | 早夭；**HSCT** 根治 | 支持治療 | 荷爾蒙補充 |

📊 **表格解讀（IPEX vs APS）**
- 最關鍵的觀念分流：**IPEX = 周邊耐受（Treg）缺陷**；**APS-I = 中央耐受（AIRE / 胸腺）缺陷**。兩者都是「rare disease 教我們免疫耐受」的代表（見 Conclusion）。
- 看到**男嬰** + **頑固性腹瀉（enteropathy）** + **極早發糖尿病** + **濕疹 / 高 IgE** → 直接想 IPEX。
- 看到**念珠菌 + 副甲狀腺低下 + 腎上腺低下**三聯 → 想 APS-I。

> 📌 **重點 (High-Yield)**：
> 1. IPEX = *FOXP3* 突變 → **CD4⁺/CD25⁺ Treg 生成缺陷** → 失去周邊免疫耐受。
> 2. 臨床三主徵：**極早發 T1DM + 嚴重 enteropathy + dermatitis**；免疫上見 **IgE↑、eosinophilia**。
> 3. **X-linked recessive**，不治療多在頭 1–2 年內死亡；根治靠 **HSCT**，但**已成形的 T1DM/甲狀腺病通常不可逆**。
> 4. 與 APS-I 區分：**IPEX→周邊 Treg；APS-I→中央 AIRE/胸腺**。

## 🧬 二、IPEX-like 疾病（非 *FOXP3* 突變）

- 臨床特徵類似 IPEX 但**無 *FOXP3* 突變**，突顯 **Treg 功能與免疫耐受**的核心地位。
- 相關突變基因：
  - <u>*IL2RA*</u>（即 **CD25 deficiency**）
  - <u>*STAT5b*</u>
  - <u>*STAT-1*</u>
  - <u>*ITCH*</u>

> 📌 **重點 (High-Yield)**：IPEX-like 疾病（*IL2RA*/CD25、*STAT5b*、*STAT-1*、*ITCH*）共同主題＝**Treg 細胞群異常**。其中 **CD25 deficiency（IL2RA）** 也以 T1DM + enteropathy 表現，最易與 IPEX 混淆。

## 🧬 三、CTLA4、STAT3、LRBA 突變

| 疾病（表中名稱） | 基因 / 遺傳 | 核心機轉 | 臨床特徵 |
|---|---|---|---|
| Autoimmune lymphoproliferative syndrome type V | **CTLA4 haploinsufficiency**（loss-of-function, 雜合） | **Treg 功能缺陷** | 自體免疫甲狀腺炎、自體免疫血球減少、低丙種球蛋白血症 |
| （LRBA 症候群） | **LRBA**（雜合突變） | 可能影響 **CTLA4 在 T 細胞內的正確 intracellular trafficking** | 類似 CTLA4 症候群（較不明確） |
| Infantile-onset autoimmune disease 1 | **STAT3**（dominant activating / GOF，雜合） | 可能因 **T 細胞不當活化**（STAT3 為 effector cytokine 訊號傳遞者） | 甲狀腺炎、血球減少、**T1DM**、**short stature（矮小）** |

- ⚠️ **CTLA4 / LRBA 相關的自體免疫，程度比 IPEX 輕得多**（much milder than IPEX）。
- STAT3 為 **dominant activating（功能獲得型, GOF）**突變 → 與多數「失功能致病」相反，易混淆。

### 🧠 記憶法
> - **CTLA4 / LRBA** → 想成「**Treg 的剎車壞掉**」：CTLA4 是 Treg 抑制功能的關鍵分子，LRBA 負責把 CTLA4 送到對的地方。
> - **STAT3-GOF** 的招牌附加特徵 = <u>short stature（矮小）</u>，幫你和其他 IPEX-like 區隔。

> 📌 **重點 (High-Yield)**：
> 1. **CTLA4 haploinsufficiency**（=ALPS type V）→ Treg 功能缺陷；**LRBA** 突變影響 **CTLA4 的細胞內運輸**，兩者表型相近但**比 IPEX 輕**。
> 2. **STAT3 為 dominant activating（GOF）突變** → 嬰兒期自體免疫，標誌附加 **short stature**。

## 🩺 四、Anti–Insulin Receptor Autoantibodies（Type B Insulin Resistance）

### 基本資料
- 又稱 **type B insulin resistance** 或 ***acanthosis nigricans***。
- 極罕見：報告病例 <u>< 100 名</u>。
- 病因：**anti–insulin receptor antibodies + anti-insulin antibodies**（抗胰島素受體抗體＋抗胰島素抗體）。

### 🩺 臨床
- 約 **1/3** 患者合併其他自體免疫病（如 **SLE、Sjögren syndrome**）。
- 其他關聯：arthralgia、vitiligo、alopecia、自體免疫甲狀腺病、secondary amenorrhea、自體免疫家族史。
- 常見實驗室異常：**ANA 陽性、ESR↑、hyperglobulinemia、leukopenia、hypocomplementemia**。

### ⚠️ 極端的胰島素阻抗（重點數字）
- 胰島素阻抗**極為深重**：每天靜脈給予多達 <u>175,000 U</u> 胰島素仍可能無法降血糖。
- 儘管高血糖且阻抗嚴重，**ketoacidosis（酮酸中毒）罕見**。
- 病程多變：部分**自發緩解**；部分反而出現**嚴重低血糖**（抗體在體外有類胰島素效應）。
- **Acanthosis nigricans**：因皮膚（組織學正常）肥厚與皺褶造成，與**胰島素阻抗狀態**相關（無抗體的其他重度阻抗也會有黑色棘皮症）。

### 💊 治療
- NIH 發展的方案（已成功治療此罕病）：
  - <u>Rituximab</u>（針對 B 淋巴球）
  - ＋ <u>cyclophosphamide</u>
  - ＋ <u>pulse corticosteroids</u>

### 🧠 記憶法
> **「Type B = B 細胞（抗體）打你的 receptor」**——所以治療要打 **B 細胞（rituximab）**。
> 對照：**Type A** insulin resistance＝受體基因缺陷（無抗體）；**Type B**＝**抗受體抗體**。

> 📌 **重點 (High-Yield)**：
> 1. Type B insulin resistance＝**抗胰島素受體抗體**致極重度阻抗（可需 **175,000 U/day IV** 仍無效），但 **DKA 罕見**。
> 2. 1/3 合併自體免疫病（**SLE / Sjögren**）；招牌皮膚徵 = **acanthosis nigricans**。
> 3. 治療＝**Rituximab + cyclophosphamide + pulse corticosteroids**（NIH 方案）。

## 🧬 五、Wolfram Syndrome（DIDMOAD）

### 基本資料
- **MIM**：<u>222300</u>（chromosome 4）、<u>598500</u>（mitochondrial）。
- **遺傳**：**autosomal recessive**（體染色體隱性）。
- 致病基因：<u>*WFS1*</u>，位於 <u>4p16.1</u>，編碼 <u>wolframin（100-kDa 跨膜蛋白）</u>。

### 🧠 DIDMOAD 口訣（必背四聯）
> **D-I-D-M-O-A-D**
> - **DI** = **D**iabetes **I**nsipidus（尿崩症）
> - **DM** = **D**iabetes **M**ellitus（**非自體免疫型**糖尿病）
> - **OA** = progressive bilateral **O**ptic **A**trophy（雙側進行性視神經萎縮）
> - **D** = sensorineural **D**eafness（感音性聽損）
- 此外：**神經與精神障礙**顯著，可致嚴重失能；MRI 可見**腦部萎縮性變化**。

### 🔬 機轉
- Wolframin 定位於 **endoplasmic reticulum（ER, 內質網）**，分布於**神經與神經內分泌組織**。
- 誘發 ion channel activity → 細胞內 **Ca²⁺ 上升** → 參與**細胞內鈣恆定**。
- *WFS1* 突變 → wolframin **穩定性下降**。
- 病程為**緩慢進行的神經退化過程**；胰臟 β 細胞為**非自體免疫性選擇性破壞**（幹細胞模型顯示與 β 細胞 **ER stress 增加**相關）。

### 基因型–表現型
- **嚴重表現型**（定義：**第一個十年內**即出現神經疾病）見於：**截短型蛋白（truncated proteins）** 與 **carboxy-terminus（羧基端）突變**。

### 🩺 臨床要點
- **兒童期發病的 DM 通常是第一個表現**。
- **DM 與 optic atrophy 出現於所有病例**；其他特徵表現變異。
- 糖尿病**病程長短**與**微血管併發症**發生相關。
- 其他內分泌：可有 **ACTH deficiency、GH deficiency**。
- 案例報告：兩名相關孩童併 **megaloblastic + sideroblastic anemia**，對 **thiamine（維生素 B1）治療有反應**，且 thiamine 治療使**胰島素需求明顯下降**。

> 📌 **重點 (High-Yield)**：
> 1. Wolfram = **DIDMOAD**（DI + DM + Optic Atrophy + Deafness），**AR 遺傳**，*WFS1*（4p16.1）→ **wolframin（ER 蛋白）**。
> 2. **DM 與 optic atrophy 100% 出現**；糖尿病為**非自體免疫**（β 細胞 ER stress / 選擇性破壞），且常為**首發**表現。
> 3. **截短型 / C 端突變 → 嚴重表現型**（首十年內神經疾病）。

## 🧬 六、Omenn Syndrome

- **MIM**：<u>603554</u>。
- 性質：**primary immunodeficiency syndrome**（原發免疫缺陷），帶**自體免疫表現**，主要侵犯**皮膚與腸胃道**。
- 機轉：與 **T-cell receptor 重組（recombination）下降**相關的突變。
- 與 IPEX/APS 連結的關鍵：一研究顯示兩名患者**胸腺中 *AIRE* gene 表現下降**，並伴隨**周邊抗原表現下降**。

> 📌 **重點 (High-Yield)**：Omenn syndrome＝**TCR 重組缺陷**的原發免疫缺陷，侵犯**皮膚＋腸胃道**；可見**胸腺 *AIRE* 表現↓ → 周邊抗原表現↓**（與 APS-I 中央耐受機轉相呼應）。

## 🧬 七、Chromosomal Disorders（染色體疾病）

| 疾病 | MIM | 相關自體免疫風險 |
|---|---|---|
| **Down syndrome（Trisomy 21）** | <u>190685</u> | **Type 1 diabetes、thyroiditis、celiac disease** |
| **Turner syndrome** | — | **thyroid disease、celiac disease** 風險增加 |

- 🩺 **建議**：對 **trisomy 21** 與 **Turner syndrome** 患者應**定期篩檢**相關自體免疫疾病。

> 📌 **重點 (High-Yield)**：Down（trisomy 21）→ **T1DM + 甲狀腺炎 + 乳糜瀉**；Turner → **甲狀腺病 + 乳糜瀉**；兩者皆需**定期篩檢自體免疫病**。

## 📊 八、總整理：「基因 → 蛋白 → 表現型」對照表（必背核心）

| 症候群 | 🧬 基因 | 🔬 蛋白 / 機轉 | 遺傳 | 🩺 招牌表現型 |
|---|---|---|---|---|
| **IPEX** | <u>*FOXP3*</u> | forkhead box protein 3；**CD4⁺/CD25⁺ Treg 生成缺陷** | **X-linked recessive** | **極早發 T1DM + enteropathy + dermatitis**；IgE↑、eosinophilia |
| **CD25 deficiency**（IPEX-like） | *IL2RA* | CD25（IL-2 受體 α）；Treg 異常 | — | T1DM + enteropathy |
| **ALPS type V**（CTLA4） | *CTLA4* | **CTLA4 haploinsufficiency**；Treg 功能缺陷 | 雜合 LOF | 自體免疫甲狀腺炎、血球減少、低丙球；**比 IPEX 輕** |
| **LRBA 症候群** | *LRBA* | 影響 **CTLA4 細胞內運輸** | 雜合 | 類 CTLA4 症候群 |
| **Infantile-onset AI disease 1** | *STAT3* | **STAT3 GOF（dominant activating）**；effector cytokine 傳遞 | 雜合活化型 | 甲狀腺炎、血球減少、T1DM、**short stature** |
| **Wolfram (DIDMOAD)** | <u>*WFS1*</u> | **wolframin**（ER 跨膜蛋白，Ca²⁺ 恆定）；非自體免疫 β 細胞破壞 | **AR** | **DI + DM + Optic Atrophy + Deafness** |
| **Omenn** | （TCR 重組相關） | TCR recombination↓；胸腺 *AIRE*↓ | — | 皮膚 + GI 自體免疫 + 免疫缺陷 |

📊 **表格解讀（基因→蛋白→表現型）**
- **共同主軸**：除 Wolfram（純內質網/退化性、**非免疫**）外，其餘多圍繞 **Treg / 免疫耐受**。把它們想成「**Treg 軸的不同壞點**」：
  - *FOXP3*（造不出 Treg）→ 最嚴重（IPEX）。
  - *IL2RA/CD25*（Treg 存活訊號）。
  - *CTLA4*（Treg 的抑制武器）/ *LRBA*（運送 CTLA4）。
  - *STAT3-GOF*（T 細胞訊號失控，**唯一 GOF**，注意方向相反）。
- **最易混淆三點**：① IPEX vs CD25 deficiency（表型重疊，差在基因）；② STAT3 是**活化型**而非失功能；③ **Wolfram 與免疫無關**（非自體免疫糖尿病，勿歸入 Treg 軸）。

> 📌 **重點 (High-Yield)**：
> 1. **Treg 軸壞點記憶串**：*FOXP3*（造）→ *IL2RA*（養）→ *CTLA4*/*LRBA*（用/送）→ *STAT3-GOF*（失控）。
> 2. **唯二例外**：**STAT3 為 GOF（活化型）**；**Wolfram 為非免疫性 ER/退化病（DIDMOAD）**。

## 🧠 九、Conclusion（章節結語精要）

- 透過研究 **APS-I 與 IPEX** 等罕病，逐步釐清兩大耐受機制：
  - **胸腺對周邊抗原的表現**（central tolerance，*AIRE*）。
  - **regulatory T cell 的發育**（peripheral tolerance，*FOXP3*）。
- 這些罕病的教訓有助於理解**更常見**的自體免疫內分泌疾病的病理生理，並可能催生**免疫學的預防與治療方法**。

> 📌 **重點 (High-Yield)**：罕病（**APS-I = AIRE/中央**；**IPEX = FOXP3/Treg/周邊**）是理解正常免疫耐受與常見自體免疫病的鑰匙——這正是本章貫穿全篇的核心命題。
