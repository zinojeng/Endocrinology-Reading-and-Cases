## 🧠 一頁總複習（高頻記憶點）

- **APS-I (APECED)**：<u>AIRE</u> 基因突變、**體染色體隱性 (AR)**、兒童期發病。三主徵 = **chronic mucocutaneous Candidiasis + Hypoparathyroidism + Addison disease**（「**2 of 3** 即診斷」）。
- **APS-II (Schmidt)**：**多基因 + HLA 關聯**、成人女性多。Addison + 自體免疫甲狀腺病 + T1DM；<u>不會有念珠菌與副甲狀腺低下</u>（與 APS-I 的關鍵區別）。
- **IPEX**：<u>FOXP3</u> 突變、**X-linked**、Treg 缺陷 → enteropathy + T1DM + 皮膚炎，嬰兒期致命。
- **Wolfram (DIDMOAD)**：<u>WFS1</u> → **D**iabetes **I**nsipidus + **D**iabetes **M**ellitus + **O**ptic **A**trophy + **D**eafness。
- 記憶軸：**APS-I = 單基因 AR (AIRE)**；**APS-II = 多基因 HLA**；**IPEX = 單基因 X-linked (FOXP3)**。
