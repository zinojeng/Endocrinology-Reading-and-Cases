## 0. 章節定位與大架構 🩺

自 **Addison** 首次描述合併 vitiligo 與 hyperpigmentation 的原發性腎上腺功能不全以來，immunoendocrinopathy syndromes 同時推進了 endocrinology 與 immunology 的理解。

- **多數**自體免疫內分泌病（如 type 1 diabetes、autoimmune thyroid disease）是**單獨**出現。
- 但有兩個界定清楚的 polyendocrine syndrome：

| 症候群 | 致病基因/特色 | 遺傳模式 | 常見度 |
|---|---|---|---|
| **APS-I** | *AIRE* gene 缺陷 | <u>Autosomal recessive (體染色體隱性)</u> | 罕見 |
| **APS-II** | 強烈關聯 <u>HLA 區域 (染色體 6p21.3)</u> + 多基因 | 多基因，無單純遺傳模式 | 較常見但定義較鬆 |

- 本章為簡化，**APS-II** 涵蓋了部分臨床醫師再細分的：
  - **APS-II**：Addison disease + type 1 diabetes 或 thyroid autoimmunity
  - **APS-III**：thyroid autoimmunity + 其他自體免疫病（但**不含** Addison disease 或 type 1 diabetes）
  - **APS-IV**：兩種或以上其他 organ-specific autoimmune disorders
- **APS-II 別名**（反映歷史重要性）：Schmidt syndrome、polyglandular autoimmune disease、polyglandular failure syndrome 等。這些舊名的缺點：未涵蓋腺體可「亢進」也可「低下」、未納入非內分泌病（pernicious anemia、celiac disease）。
- **IPEX**（immunodysregulation polyendocrinopathy enteropathy X-linked）：由 *FOXP3* 基因突變引起；*FOXP3* 是 regulatory CD4⁺ T cells 發育與功能的核心。

> 📌 **重點 (High-Yield)**：
> - **APS-I = AIRE = autosomal recessive**；**APS-II = HLA/多基因**。
> - APS-II 是個「大傘」，把 APS-III、APS-IV 都收進來。
> - **IPEX = FOXP3 = X-linked**，破壞 regulatory T cell。

## 1. Autoimmunity Primer：免疫耐受的基礎 🔬

自體免疫 = **免疫耐受 (tolerance, 分辨自我 vs 非自我的能力) 崩解**。崩解可發生在：
- **中央 (central)**：生成器官層級（thymus、bone marrow）
- **周邊 (peripheral)**：標的器官或淋巴組織

### 1.1 兩大免疫臂：T cell vs Autoantibody (B cell)

| 比較項 | **Autoantibody (B cell)** | **T lymphocyte** |
|---|---|---|
| 辨識對象 | **完整分子**（可溶性或細胞表面）；辨識 **conformational determinants (立體構型)** | **peptide fragments（胜肽片段）**，常 <u>8–12 個胺基酸</u> |
| 呈現方式 | 直接結合 | 由 **MHC** 呈現於另一細胞表面 |

📊 **記憶法**：抗體看「**形狀**」（立體構型），T cell 看「**片段**」（被 MHC 端上桌的小胜肽）。

### 1.2 MHC = 熱狗堡比喻 🌭

- Histocompatibility molecule 與 T-cell receptor 結合時需先**載入抗原胜肽**。
- 比喻：**抗原胜肽 = 熱狗**，**MHC 分子的 groove (凹槽) = 麵包**。
- MHC 高度**多型 (polymorphic)**：凹槽襯邊的胺基酸不同 → 決定能結合並呈現哪些胜肽。

### 1.3 CD4 vs CD8 / Class I vs Class II

| | **CD4⁺ T cell** | **CD8⁺ T cell** |
|---|---|---|
| 配對 MHC class | **Class II**（HLA-**DP / DQ / DR**） | **Class I**（HLA-**A / B / C**） |
| 抗原來源 | 細胞外蛋白（extracellular），被 APC 攝取 | 內生性表現蛋白（endogenous） |
| 表現細胞 | APC：macrophage、dendritic cell、B lymphocyte | 幾乎所有**有核細胞** |
| 結果 | 協調免疫反應 | 釋放 cytotoxic chemicals **殺死標的細胞** |

🧠 **口訣**：
- **「外 → 二 → 四」**：細胞**外**抗原 → Class **II** → CD**4**。
- **「內 → 一 → 八」**：細胞**內**抗原 → Class **I** → CD**8**（殺手）。
- HLA class II 字母都靠後（**D**P/**D**Q/**D**R），class I 都靠前（**A**/**B**/**C**）。

### 1.4 兩訊號模型 (Two-signal model) ⚠️

T cell 活化**不只**需要辨識抗原：

- **Signal 1**：MHC–peptide–TCR 三者交互（核心辨識）。
- **Signal 2 (co-stimulation)**：APC 上 **CD80/CD86** 接合 T cell 上 **CD28** → 放大 signal 1 → **活化**。
- ⚠️ **陷阱**：若 T cell 只辨識抗原（signal 1）卻**缺乏正確的第二訊號** → 變成 **anergy（無反應/失能）**，而非活化。
- 結果取決於 context（細胞表面分子、cytokine、chemokine）→ 細胞可被 **activated / tolerized / anergic**。

> 📌 **重點 (High-Yield)**：
> - 抗體認**構型**，T cell 認 **8–12 aa 胜肽 + MHC**。
> - **CD4–Class II–細胞外**；**CD8–Class I–細胞內–殺傷**。
> - **CD80/86 ↔ CD28 = signal 2**；有 signal 1 無 signal 2 → **anergy**。

## 2. Central Tolerance（中央耐受）：胸腺裡的把關 🔬

耐受誘導是**分階段 (staged)** 的過程，**始於 thymus** 的 T 細胞成熟期。

- 關鍵：**peripheral antigens（周邊抗原）出現在 thymus 中**。
  - Peripheral antigens = 平常在免疫系統**外**的組織表現的自體抗原（例如 **insulin**），在 thymus 中**低量表現**。
- 對這些周邊分子（在 MHC 脈絡下）**強烈反應**的發育中 T cell → 被**刪除 (negative selection / 負選擇)** → 移出 T-cell repertoire。
- **AIRE 基因 knockout 小鼠**佐證：thymus 中周邊抗原表現量低 → 多器官淋巴球浸潤 → 自體免疫。

🧠 **記憶**：**AIRE = 胸腺裡的「自我廣告看板」**，把全身各器官的自體抗原拿到 thymus 展示，讓會攻擊自我的 T cell 在出場前被淘汰。AIRE 壞掉 → 看板撤掉 → 自反應 T cell 漏網 → APS-I。

## 3. Peripheral Tolerance（周邊耐受）：T 細胞出胸腺後 🔬

T cell 成熟離開 thymus 後，仍需周邊耐受機制把關。靠 **anergic** 與 **regulatory T cells**。

### 3.1 Regulatory T cell (Treg)

- 主要族群表面標記：**CD4 + CD25**，並表現 **FOXP3**。
- **CD4⁺/CD25^high** 細胞具**主動抑制活性**，依賴轉錄因子 **FOXP3**。
- **FOXP3 刪除 → 新生兒猛爆性自體免疫**（如 neonatal type 1 diabetes + enteropathy），常於**出生第一年內死亡**（= **IPEX syndrome**）。

### 3.2 共抑制分子：CTLA4 與 PD1（周邊耐受的剎車）

| 分子 | 機轉重點 |
|---|---|
| **CTLA4** | 表現於 T cell，為 T-cell signaling 的**負向調節者**；與活化分子 **CD28 競爭**結合 **CD80/CD86**，因**親和力更高**而**勝出 (outcompete)**；也廣泛表現於 **FOXP3⁺ Treg** 表面，阻斷 CD28–CD80 互動 |
| **PD1** | 在**慢性活化**的 T cell 上**上調**；經其胞內尾端的**抑制性訊號域**傳遞負向訊號 |

⚠️ **臨床橋接（極高 yield）**：
- **PD1 與 CTLA4 knockout 小鼠** → 自發**多器官自體免疫**。
- 接受**抗 CTLA4 / 抗 PD1（immune checkpoint blockade）**的癌症病人 → 許多人出現**自體免疫併發症**（後文：thyroiditis、type 1 diabetes、lymphocytic hypophysitis、adrenalitis）。

🧠 **記憶**：把 **CTLA4／PD1 想成 T 細胞的「剎車」**。剎車基因壞掉（KO）或被藥物（checkpoint inhibitor）「鬆開」→ 免疫車失控 → 自體免疫。

### 3.3 B 細胞耐受與 Cognate Help

- **Cognate help**：B cell 由**辨識相同抗原**的 CD4⁺ T cell 活化。
- 流程：CD4⁺ T cell 辨識 B cell 表面 MHC 上的抗原 → CD4⁺ T cell 分泌 **IL-4、IL-5、IL-6** → 促 B cell 成熟。
- 視 cytokine 環境，B cell 由 **IgM** 類別轉換為 **IgG、IgE 或 IgA**。
- **Linked recognition（連結辨識）**：找不到能結合同一抗原的 CD4⁺ T cell 的 autoreactive B-cell clone → **不會被活化**。
  - → 故產生 autoantibody 的 B cell，通常**連動一個針對同一自體抗原的 autoreactive T cell**。
- 新證據：autoreactive B cell 作為**重要 APC** 呈現給 autoreactive T cell → 形成**正回饋環**，擴大並維持自體免疫。

> 📌 **重點 (High-Yield)**：
> - **Treg = CD4⁺CD25^high FOXP3⁺**；FOXP3 失效 → **IPEX**，一歲內可致死。
> - **CTLA4 競爭性勝過 CD28**（親和力高），PD1 在慢性活化時剎車。
> - Checkpoint inhibitor（抗 CTLA4/PD1）會引發內分泌自體免疫。
> - B cell 出抗體幾乎都「**綁定**」一個同抗原的 autoreactive T cell（linked recognition）。

## 4. 自然病程：自體免疫的「四階段」模型 🩺

自體免疫病的自然史可分為一系列階段（**理論建構，已簡化，未反映可能的 relapsing-remitting 特性**）：

| 階段 | 名稱 | 內容 / 範例 |
|---|---|---|
| **① 遺傳易感** | Genetic susceptibility | 多基因（HLA 效應最強）+ 環境 |
| **② 環境誘發** | Triggering of autoimmunity | 例：celiac disease 的**膳食 gliadin (gluten) 暴露** |
| **③ 隱性/活躍自體免疫** | Active autoimmunity（臨床表現前） | 進行性腺體破壞；**autoantibody (+)** 但尚無症狀 |
| **④ 腺體衰竭 / 顯性疾病** | Overt disease | 腺體破壞達顯著程度 → 臨床發病 |

🧠 **口訣（四階段）**：**「基因 → 環境 → 抗體 → 衰竭」**（Genes → Trigger → Antibodies → Gland failure）。

> 📌 **重點 (High-Yield)**：
> - 四階段：**遺傳易感 → 環境誘發 → 抗體陽性的活躍自體免疫 → 腺體衰竭**。
> - 第③階段就有 autoantibody，**早於**症狀 → 可作篩檢窗口。
> - 此模型是**簡化的理論**，未涵蓋復發-緩解。

## 5. Genetic Associations：基因 → 疾病關聯 🧬

- APS-II 與其組成疾病有**家族聚集**，但**無單純遺傳模式**。
- 易感性由**多個遺傳位點**（**HLA 效應最強**）× 環境因素決定。
- 共同遺傳風險因子：**HLA、MICA、PTPN22、CTLA4、NLRP1 (NALP1)**。
- 也有**器官專一性**多型：例如 **insulin 基因上游的 VNTR**（variable number tandem repeat）與 type 1 diabetes 風險相關。

#### 📊 表格解讀（臨床意義與易混淆處）

- **HLA = 抗原呈現「桌子」**，效應最強。注意各病的特徵 haplotype：
  - **Type 1 DM / Graves**：DR3-DQ2 / DR4-DQ8（兩者相同組合！靠其他基因+器官抗原區分）。
  - **Celiac**：純 **DR3-DQ2**。
  - **Addison**：特殊的 **DRB1\*0404**-DQ8（DR4 的特定亞型）。
  - **Hypothyroidism**：DR3、DR5。
- **MICA 的 5.1** 是 Addison disease 的招牌（也見於 T1DM、celiac）。
- **PTPN22 R620W** 是「**廣譜**」自體免疫位點：T1DM、SLE、RA、Graves、hypothyroidism、vitiligo 全包（但與 Addison **僅弱相關**，見下文）。
- **CTLA4** 兩個重點多型：**CT60**（3′ UTR 的 SNP）與 **+49A/G**。
- ⚠️ **單基因 vs 多基因**：表中只有 **AIRE (AR)** 與 **FOXP3 (X-linked)** 是**單基因孟德爾遺傳**，其餘全是 **Multigenic**。別把 HLA/PTPN22/CTLA4 當成單基因病。

🧠 **基因記憶法（mnemonic）**：
- **「H-M-P-C-A-F」**順 = HLA、MICA、PTPN22、CTLA4、AIRE、FOXP3。
- **功能分群**：
  - **呈現抗原**：HLA（桌子）、MICA（priming naive CD8）、AIRE（胸腺展示自我抗原）。
  - **訊號剎車/調節**：PTPN22（TCR 訊號抑制）、CTLA4（CD28 競爭剎車）、FOXP3（Treg 司令）。

### 5.1 HLA 細節（DQ 為內分泌自體免疫最強關聯）

- **DQ 分子**是與內分泌自體免疫**最強相關**的 histocompatibility molecule。
- 命名：α 鏈 **DQA1\*0501**、β 鏈 **DQB1\*0201** = **DQ2**，常編碼在 **DR3 (DRB1\*0301)** haplotype 上。
- **Haplotype** = 同一染色體連續區段上一串一起遺傳的 allele；**Genotype** = 兩條染色體 haplotype 的組合。
- **HLA-A1/B8/DR3** haplotype 高度保守，約 <u>2.9 megabases</u> 不變 → 強 linkage disequilibrium → **難以鎖定致病基因**。

#### Type 1 DM 最高風險基因型（必背數字）⚠️

- 最高風險基因型：**DR3-DQ2, DR4-DQ8**（DQ8 = **DQA1\*0301-DQB1\*0302**）。
- 與 T1DM 手足**共享相同** DR3-DQ2, DR4-DQ8 的孩子：
  - 12 歲前發展出**自體免疫**風險 **> <u>75%</u>**；
  - 12 歲前發展出**糖尿病**風險 **> <u>50%</u>**。
- **DRB1\*0404**（DR4 的特定亞型）與 **Addison disease** 強相關。
- **DR3-DQ2** haplotype 與 **celiac disease** 相關（不論有無 T1DM）；也與 autoimmune thyroid disease 相關（但有矛盾報告）。

#### 保護性 HLA allele ✅

- **DQA1\*0102-DQB1\*0602**（通常伴 DR2）：
  - 以**顯性 (dominant)** 方式**強力保護**免於 **type 1A diabetes**；
  - 但**反而易感** **multiple sclerosis**；
  - ⚠️ 保護**非全面性**：對 **Addison disease 不提供保護**（DQB1\*0602 無保護）。
- **DP** 基因的 **0402** 多型：在最高風險基因型（DR3/DR4）者中，**降低** type 1 diabetes 風險。

🧠 **DQ 記憶法**：
- **DQ2 ≈ DR3**；**DQ8 ≈ DR4**。
- **「3-2、4-8」**配對：DR**3**-DQ**2** 與 DR**4**-DQ**8** 並存 = T1DM 最高風險。
- **0602 = 保護 (Protective)**，但只保護 **T1DM**，不保護 Addison，且把 **MS** 拉進來。

> 📌 **重點 (High-Yield)**：
> - **DQ** 是內分泌自體免疫最強的 HLA；DR3-DQ2, DR4-DQ8 = T1DM 最高風險。
> - 手足共享高風險基因型：12 歲前**自體免疫 >75%**、**糖尿病 >50%**。
> - **DRB1\*0404 → Addison**；**DQB1\*0602 → 保護 T1DM（但易感 MS、不保護 Addison）**。

### 5.2 非 HLA 基因細節

| 基因 | 蛋白 / 機轉 | 關鍵多型 | 關聯疾病 |
|---|---|---|---|
| **MICA** | 表現於 thymus 與 naive CD8⁺ T cell | **5.1**（插入單一鹼基 → premature stop codon → 截短蛋白） | T1DM、celiac、**Addison**（5.1 影響 Addison 風險） |
| **PTPN22** | 編碼 **LYP**；經與 **CSK** 等 kinase 互動，**抑制 TCR 下游訊號** | **R620W**（620 位 Arg→Trp）→ **阻斷 LYP–CSK 互動**；可能增加 autoreactive B cell | T1DM、RA、SLE、Graves、vitiligo；**與 Addison 弱相關** |
| **CTLA4** | 活化 CD4⁺/CD8⁺ T cell 上的負向調節者 | **AT repeats**（降低 Graves 中 CTLA4 抑制功能）；**CT60**（3′UTR SNP）→ Graves、hypothyroidism；**+49A/G** → celiac（荷蘭人）、AITD、Addison | Graves、hypothyroidism、celiac、Addison |
| **NALP1 (NLRP1)** | 調節**先天免疫 (innate immunity)** | — | 先發現與 **vitiligo** 相關，後關聯 **Addison disease、T1DM** |
| **Insulin VNTR** | insulin 基因上游 tandem repeat | **較多重複數** → thymus insulin 表現↑ → **改善 insulin-reactive T cell 的負選擇** → **保護** T1DM | T1DM |
| **Thyroglobulin** | 甲狀腺球蛋白基因多型 | — | autoimmune thyroid disease |

- **AIRE 與 sporadic Addison**：AIRE 多型**一般不參與** APS-II 或 sporadic Addison。
  - Addison（非 APS-I）患者：**1/90 (1.1%)** 有 AIRE mutation。
  - 對照組：**1/576 (0.2%)** 有 AIRE mutation。

🧠 **MICA 5.1 記憶**：**「5.1 = 截短 = Addison」**（單鹼基插入 → stop codon → truncated protein）。

🧠 **Insulin VNTR 邏輯記憶**：**重複越多 → 胸腺 insulin 越多 → 看板越清楚 → 反 insulin 的 T cell 被刪得越乾淨 → 越保護**。（與 AIRE「胸腺展示」概念同源。）

> 📌 **重點 (High-Yield)**：
> - **PTPN22 R620W** 阻斷 LYP–CSK，廣譜自體免疫；與 Addison **僅弱相關**。
> - **MICA 5.1**（截短蛋白）— Addison。
> - **Insulin VNTR 高重複 = 保護**（胸腺 insulin 表現↑ → 負選擇↑）。
> - AIRE 與 sporadic Addison **基本無關**（1.1% vs 0.2%）。

## 6. Environmental Triggers（環境誘發）🩺

遺傳**不能解釋全部**：T1DM 最高風險基因型 **DR3-DQ2, DR4-DQ8** 的糖尿病風險僅 **1/20**（高於一般族群 20 歲前 **1/200**，但遠非 100%）→ 必有其他（遺傳+環境）因素。

### 6.1 飲食 🍞

- **Celiac disease 的環境觸發已確定 = gluten。**
- 研究 **DAISY**、**BabyDiab**、**CEDAR**：**首次穀類 (cereal) 暴露時機**為風險因子 → **過早**接觸 cereal 的嬰兒，發展 diabetes 與 celiac 相關自體免疫的比率較高。
- **Cod liver oil（魚肝油）**：流行病學上與 **T1DM 風險降低**相關（含 omega-3 PUFA 與 vitamin D）；前瞻研究提示 **omega-3 PUFA 攝取較低 → T1DM 相關自體免疫風險↑**。
- **Microbiome**：飲食、食物組成、抗生素 → 改變腸道菌叢；動物研究示 gut microbiome 改變可影響疾病。

### 6.2 感染 🦠

- **病毒/感染**可致自體免疫：例如**子宮內 rubella 感染**可致 T1DM。
- prediabetes（抗體陽性）或 diabetes 病人的胰臟檢體有「**符合病毒感染**」的證據。
- ⚠️ **關聯 ≠ 因果**（Association is not causality），仍需進一步實驗。

### 6.3 醫源性 / 藥物與治療誘發 💊⚠️

| 治療 | 誘發的自體免疫 |
|---|---|
| **抗 CD52 monoclonal antibody**（治 MS） | <u>27 人中 1/3</u> 出現 **anti-TSH receptor 抗體 + hyperthyroidism** |
| **Interferon-α (IFNα)**（治 hepatitis） | thyroid autoimmunity、type 1 diabetes |
| **Methimazole**（治 Graves） | **Hirata disease**：insulin autoantibody 致**嚴重低血糖**（無外源 insulin 注射）；關聯 **HLA-Bw62/Cw4/DR4** 之 **DRB1\*0406** |
| **Immune checkpoint blockade**（抗 CTLA4 或抗 PD1） | thyroiditis、type 1 diabetes、lymphocytic hypophysitis、adrenalitis |

### 6.4 Paraneoplastic（腫瘤旁自體免疫）🔬

- 如 **paraneoplastic neurologic syndrome**：腫瘤**異位表現器官專一抗原** → 打破耐受 → 引發專一自體免疫。
- 近期描述：一種 paraneoplastic 病況，特徵為**孤立性 ACTH deficiency** 與 **anti-PIT-1 hypophysitis**。

> 📌 **重點 (High-Yield)**：
> - **Celiac 的觸發 = gluten（唯一已確定者）**；cereal 過早暴露 = T1DM/celiac 自體免疫風險↑。
> - **Hirata disease**：methimazole 引發 insulin autoantibody → 嚴重低血糖（**DRB1\*0406**）。
> - **Checkpoint inhibitor（抗 CTLA4/PD1）**可致 thyroiditis、T1DM、hypophysitis、adrenalitis。
> - **異位抗原表現 → paraneoplastic 自體免疫**（如 anti-PIT-1 hypophysitis + 孤立 ACTH 缺乏）。

## 7. Development of Organ-Specific Autoimmunity（器官專一自體免疫的發展）🔬

- **Autoantibody 在發病前即存在**，且**高度專一**；每個抗體只對應**單一 autoantigen**（雖該抗原可存在多組織）。

### 7.1 各疾病的標記抗體（必背）

| 疾病 | Autoantibody 標的 |
|---|---|
| **Type 1 diabetes** | **GAD**（glutamic acid decarboxylase）、**ICA512 / IA-2**（insulinoma antigen 2）、**insulin**、**ZnT8**（最新發現） |
| **Celiac disease** | **tTG**（tissue transglutaminase） |
| **Addison disease** | **21-hydroxylase** |

🧠 **T1DM 四抗體口訣**：**「GIIZ」= GAD、IA-2 (ICA512)、Insulin、ZnT8**。

### 7.2 臨床應用：篩檢與風險分層 🩺

- 抗體可在器官功能異常**之前**偵測 → 用於**篩檢高風險者**。
- 例：**TrialNet** 篩檢 T1DM 患者的**一等親**之 diabetes 相關抗體。
- **風險隨抗體數目↑ 與抗體持續性↑ 而增加。**
- 一般族群中 organ-specific autoantibody **罕見（約 <u>1/100</u>）**，標記出較高風險的次族群；抗體可在發病前**表現數年**，並隨時間增加。
- 發病速度差異大：T1DM 可早至 **1 歲**發病。
- 成人診斷的 type 2 diabetes 中，**<u>5%–10%</u>** 其實為自體免疫病因（部分因遺傳：年長發病者**保護性 allele DQB1\*0602 比例較高**；即便成人，DQB1\*0602 仍提供顯著保護）。

### 7.3 致病性 T cell 的特性

- 對 pathogenic T cell 的專一性所知**較少**。
- cross-reactive 辨識可能由 nonapeptide 中**僅 4 個正確間隔的胺基酸**決定；估計每個 TCR 可對**百萬種胜肽**反應 → 自體免疫模式有很大可能由 **cross-reactive T cells** 決定。
- thymus 與其他淋巴組織中發現 **peripheral antigen-expressing cells**（表現如 insulin 等自體抗原）→ 微量即可貢獻耐受。
- thymus 的 insulin mRNA 受 **insulin 基因多型**調控（呼應 VNTR）。
- 脾、淋巴結、循環中的 stromal 與 **CD11c⁺ lymphoid cells** 也表現多種類似抗原。

> 📌 **重點 (High-Yield)**：
> - T1DM 四抗體 = **GAD / IA-2(ICA512) / Insulin / ZnT8**；Celiac = **tTG**；Addison = **21-hydroxylase**。
> - **抗體數目越多、越持續 → 風險越高**（TrialNet）；一般族群陽性率約 **1/100**。
> - 成人 T2DM 中 **5%–10%** 其實是自體免疫（LADA 概念）；DQB1\*0602 在成人仍強力保護。

## 8. Failure of Gland（腺體衰竭）🩺

- 器官功能障礙**隨時間發展**，可經歷一段「**中間功能期**」：
  - **刺激性荷爾蒙升高**（如 **TSH**、**ACTH**）
  - 但**標的荷爾蒙仍正常**（**T3、T4、cortisol**）。
- 一旦腺體**被破壞達顯著程度** → **顯性疾病 (overt disease)** 出現。

🧠 **記憶**：中間期 = 「**上游荷爾蒙先喊救命 (TSH/ACTH↑)，下游荷爾蒙還撐得住 (T3/T4/cortisol 正常)**」→ 即 subclinical 階段；對應第四階段「腺體衰竭」的早期。這正是為何**抗體 + 上游荷爾蒙輕升**可早期抓到即將衰竭的腺體。

> 📌 **重點 (High-Yield)**：
> - 腺體衰竭前有「**中間功能期**」：**stimulatory hormone (TSH/ACTH)↑ + target hormone 正常**。
> - 此即 subclinical 階段，是早期偵測與監測的窗口。

## 9. 全章核心比較表（速記）📊

### 9.1 單基因孟德爾 vs 多基因

| | **APS-I** | **IPEX** | **APS-II（含 III/IV）** |
|---|---|---|---|
| 基因 | **AIRE** | **FOXP3** | **HLA + 多基因** |
| 遺傳 | **Autosomal recessive** | **X-linked** | **Multigenic** |
| 機轉核心 | 胸腺周邊抗原呈現失敗（central tolerance 崩） | Treg 失能（peripheral tolerance 崩） | 抗原呈現 + 多位點 |

### 9.2 Central vs Peripheral Tolerance

| | **Central tolerance** | **Peripheral tolerance** |
|---|---|---|
| 地點 | **Thymus / bone marrow** | 標的器官 / 淋巴組織 |
| 機制 | **Negative selection**（刪除高反應 T cell） | **Anergy、Treg (FOXP3)**、CTLA4/PD1 剎車 |
| 代表基因/疾病 | **AIRE → APS-I** | **FOXP3 → IPEX**；CTLA4/PD1 KO 或 checkpoint inhibitor |

---

> 🧠 **全章一句話總結**：自體免疫 = **耐受崩解**（中央 AIRE/胸腺、周邊 FOXP3/CTLA4/PD1）；走「**遺傳易感(HLA最強) → 環境誘發(gluten 等) → 抗體陽性 → 腺體衰竭**」四階段；抗體可早於症狀數年，是篩檢與早期診斷的關鍵。
