# Ch.44 APS-II 與其他多腺體自體免疫症候群

> 來源：Williams Textbook of Endocrinology, Ch.44 The Immunoendocrinopathy Syndromes — APS-II 及其他 polyendocrine deficiency autoimmune syndromes

## 🩺 一、APS-II 是什麼？臨床定義 (Clinical Features)

**APS-II (MIM <u>269200</u>)** 又稱 **Schmidt syndrome**（廣義）。診斷依據：**同一病人出現下列疾病中的 ≥ 2 種**。

🔑 **APS-II 組成清單（≥2 即診斷）**

- Primary adrenal insufficiency（**Addison disease**）
- **Graves disease**
- Autoimmune thyroiditis（自體免疫甲狀腺炎）
- **Type 1A diabetes**
- Primary hypogonadism（原發性性腺低下）
- Myasthenia gravis（重症肌無力）
- Celiac disease（乳糜瀉）

➕ **次要關聯**（病人與家屬發生率升高）：Vitiligo、alopecia、serositis、pernicious anemia。

### 流行病學特徵（與 APS-I 對比的關鍵）

- APS-II <u>比 APS-I 更常見</u>。
- **女性 > 男性**（female preponderance）。
- 常於 **成年期 (adulthood)** 發病，<u>peak onset 20–60 歲</u>。
- 呈 **familial aggregation（家族聚集）**，遺傳為 **polygenic（多基因）**，多代受影響。

> 🧠 **記憶法**：把 Schmidt 經典三角想成 **「腎上腺 + 甲狀腺 + 胰島」**＝ <u>Addison + 自體免疫甲狀腺病 + T1DM</u>。
> 口訣：**「A-T-D」→ Adrenal–Thyroid–Diabetes**，這是 fellow 必背的 Schmidt syndrome 三元素。

### 「一個病 → 預測下一個病」的風險（T1DM 病人為例）

在 **type 1 diabetes** 病人身上：

| 後續自體免疫風險 | 機率 |
|---|---|
| Hypothyroidism（甲狀腺低下） | <u>15%–20%</u> |
| Celiac-related autoimmunity（乳糜瀉相關自體免疫） | <u>5%–10%</u> |
| Adrenal autoimmunity（腎上腺自體免疫） | <u>1.5%</u> |

- **親屬風險更高**：在有 Addison disease 的 APS-II 家族中，<u>高達 15%</u> 的親屬帶有 21-hydroxylase 抗體、anti-islet 抗體或 tTG 抗體（即使尚無臨床疾病）。

📊 **表格解讀**：這組數字常被拿來考「T1DM 該篩什麼」。記住 **甲狀腺風險最高（兩成）、乳糜瀉次之、腎上腺最低（1.5%）但最致命**——所以腎上腺雖罕見卻最該警覺。

### 大型世代數據（要記方向，不必死背）

- **Neufeld 等人** 224 例 Addison + APS-II：最常合併的是 **autoimmune thyroid disease（<u>69%</u>）** 與 **type 1 diabetes（<u>52%</u>）**；vitiligo 5%、gonadal failure 4%。
- **義大利 >600 例** 世代（排除 APS-I 後）：<u>86.7%</u> 的 Addison 病人有額外自體免疫，最常見為 **甲狀腺病 56%**、**糖尿病 12%**；另有甲狀腺相關抗體陽性但未發病者 16%、糖尿病相關 8.8%（平均追蹤 10 年）。

> 📌 **重點 (High-Yield)**：
> - APS-II = **≥2 個** 自體免疫病；核心三角 **Addison + 自體免疫甲狀腺 + T1DM (Schmidt)**。
> - **女性多、成年發病 (20–60y)、多基因遺傳、多代受影響**。
> - Addison 合併最多的是 **甲狀腺病 (69%)** ＞ **T1DM (52%)**；celiac 是 APS-II 的特色（APS-I 不會）。

## 🔬 二、T1DM 病人的甲狀腺與乳糜瀉篩檢

### 甲狀腺自體免疫（要每年篩 TSH）

- **Thyroid peroxidase (TPO) 抗體** 出現在 <u>10%–20%</u> 的 T1DM 兒童；女性較高，且隨 **年齡** 與 **糖尿病病程** 增加。
- 追蹤 >15 年，**TPO 抗體陽性 + T1DM 的病人有 <u>80%</u> 變成 hypothyroid**。
- ⚠️ 但 **抗體陰性者仍有一部分** 會發生甲狀腺病 → 因此 **不能只靠抗體**。
- 👉 結論：**T1DM 病人每年篩 TSH**（cost-effective）。

### 乳糜瀉 (Celiac disease) 與 tTG

- **tTG（tissue transglutaminase）** 是乳糜瀉的主要 endomysial 自體抗原。
- <u>10%–12%</u> 的 T1DM 病人有 tTG 抗體。
- **HLA-DQ2** 相關性高：<u>1/3</u> 的 DQ2 同型合子（homozygous）表現 anti-tTG（同時影響 T1DM 與 celiac 發生）。
- 高效價抗體者做切片，**<u>70%</u> 確診** celiac。
- 病程 >15 年的 T1DM + celiac → **死亡率上升**；未治療的症狀性 celiac → **胃腸道惡性腫瘤（尤其 lymphoma）** 風險上升。
- 💊 處置：tTG 陽性且複檢確認 → **小腸切片** 確診 → 給 **gluten-free diet**。許多病人為 **無症狀 celiac**，但仍合併骨質缺乏 (osteopenia) 與生長受損。

> 📌 **重點 (High-Yield)**：
> - **T1DM 每年驗 TSH**；TPO+ 者 15 年後 <u>80%</u> 變甲低，但抗體陰性也可能發病。
> - tTG+ 約 <u>10%–12%</u>；**HLA-DQ2** 是 T1DM 與 celiac 的共同地基。
> - tTG+ → 複檢 → **小腸切片** → gluten-free diet；未治療 celiac 與 **lymphoma** 風險相關。

## 🧪 三、診斷 (Diagnosis)：自體抗體與腺體功能並用

核心邏輯：**單一自體免疫內分泌病人 → 評估其他自體免疫病風險**（病史 + 理學檢查 + 器官特異抗體 + 腺體功能）。

### 各疾病的關鍵自體抗原（必背配對）

| 疾病 | 自體抗原 (autoantigen) |
|---|---|
| Thyroiditis | **Thyroid peroxidase (TPO)** |
| Type 1 diabetes | **GAD（GAD65）、carboxypeptidase H、ICA512/IA2、ZnT8、insulin** |
| Addison disease | **21-hydroxylase、17α-hydroxylase** |
| Pernicious anemia | **壁細胞 H⁺/K⁺-ATPase** |

📊 **表格解讀**：
- T1DM 最具資訊量的 **四個抗體**：**insulin、GAD65、ICA512/IA2、ZnT8**。
- Addison 用 **21-hydroxylase 抗體**（disease specificity/sensitivity 佳）；**抗體常先於臨床 Addison 出現**。
- 重組抗原 radioimmunoassay **優於** 舊式組織切片免疫螢光（如 ICA testing）。

### 抗體的兩種角色：標記 vs 致病

- **多數抗體 = 標記**（疾病由 **T-cell 媒介的腺體破壞** 造成）。
- **少數抗體本身致病**：辨識標誌是出現 **新生兒型 (neonatal form)**，因抗體經 **胎盤傳遞**。
  - **Neonatal Graves disease** ← anti–TSH-receptor 抗體
  - **Neonatal myasthenia gravis** ← anti–acetylcholine-receptor 抗體

### 篩檢策略（家族與高風險者）

- 已紀錄症候群的家族 → 親屬應 **每 3–5 年** 做病史、理學檢查、篩檢：**anti-islet 抗體、敏感 TSH、血清 vitamin B₁₂**。
- 出現症狀/徵象，或 **21-hydroxylase 抗體陽性** → 每年驗 **basal ACTH + Cortrosyn (cosyntropin) 刺激試驗**。

> 🧠 **記憶法（致病性抗體）**：**「會傳給寶寶的才是真打手」** → 有 **neonatal 版** 的（Graves、MG）＝抗體本身致病；其餘多為「破壞後留下的足跡」。

> 📌 **重點 (High-Yield)**：
> - T1DM 四大抗體：**insulin / GAD65 / ICA512(IA2) / ZnT8**；Addison 看 **21-hydroxylase**（先於發病）。
> - **致病性抗體** = 有 neonatal form：**Graves（TSH-R）、MG（AChR）**。
> - 親屬 **每 3–5 年** 篩 anti-islet 抗體 + TSH + B₁₂；21-OH 抗體陽性 → 每年 ACTH + cosyntropin。

## 💊 四、治療 (Therapy)：APS-II 專屬重點

### ⚠️ 最致命陷阱：先補甲狀腺素會誘發 Addisonian crisis

- **Thyroxine therapy 可誘發致命的 addisonian crisis**（在未治療的腎上腺不足 + 甲狀腺低下病人）。
- 👉 **凡懷疑此症候群的 hypothyroid 病人，補甲狀腺素前必先評估腎上腺功能**。
- 🩺 早期線索：**胰島素需求量下降** 可能是腎上腺不足的 **最早期表現**，早於 hyperpigmentation 或電解質異常。

### 預防 (Prevention)：以 T1DM 為主戰場（三級概念）

| 層級 | 對象 | 可用藥物風險 |
|---|---|---|
| **Primary（初級）** | 有遺傳風險、**抗體陰性** | 介入須 **安全、易給予**（會暴露到永不發病者） |
| **Secondary（次級）** | **抗體陽性**、血糖尚正常 | 可用 **較高風險** 藥物 |
| **Tertiary（三級）** | 已發病 | 免疫調節以保 **β-cell mass**、延長 C-peptide |

- **Primary**：針對高風險嬰兒（家族史/特定 HLA、且抗體陰性）；試過 gluten-free diet、**DHA** 補充、缺牛胰島素配方 → **大型試驗皆未能預防** 抗體或糖尿病產生。
- **Secondary**：抗原特異（insulin、GAD65）與非抗原特異（nicotinamide）。
- **DPT-1 試驗**（NIH）：
  - 高風險組（5 年內風險 >50%）：**靜脈-皮下 insulin** → **無效**。
  - 中風險組（5 年風險 25%–50%）：**口服 insulin** → **無效**；但 **高 insulin 自體抗體** 亞群見到療效訊號。
- **Preclinical Addison**：短療程 **glucocorticoids** 可抑制腎上腺抗體、防止漸進性破壞。
- **正在研究中** 的免疫調節：**abatacept、teplizumab**。

### 免疫抑制/生物製劑（保 β-cell）

- **Cyclosporine**：可保留部分殘餘胰島素分泌，但 **腎毒性 + 潛在致癌** → 不宜廣泛使用。
- 較新者：**sirolimus**；生物製劑 **rituximab (anti-CD20)、abatacept、非促分裂 CD3 抗體** → 第一年延長 C-peptide、降低胰島素用量。

> 🧠 **記憶法（治療順序鐵律）**：**「先腎上腺，後甲狀腺」(Steroid before Thyroxine)**——順序顛倒會引爆 addisonian crisis。

> 📌 **重點 (High-Yield)**：
> - **補 thyroxine 前必先查/補腎上腺**，否則誘發 **addisonian crisis**。
> - **胰島素需求下降** = Addison 的最早期警訊（早於色素沉著與電解質異常）。
> - DPT-1：**口服與注射 insulin 皆未能預防 T1DM**；preclinical Addison 短期類固醇可抑制抗體。

## 🆚 五、APS-I vs APS-II 終極對照表（最高頻考點）

> 來源：Table 44.2（Contrasting Features of APS）＋ 內文。

| 項目 | **APS-I (APECED)** | **APS-II (Schmidt)** |
|---|---|---|
| MIM | <u>240300</u> | <u>269200</u> |
| 遺傳模式 (Inheritance) | **Autosomal recessive**（僅同代 siblings 受影響） | **Polygenic / 多基因**（多代受影響） |
| 關聯基因/HLA | **AIRE gene mutation** 🧬 | **HLA-DR3 與 DR4** 關聯 |
| 性別 (Gender) | **男女相等** (equal) | **女性偏多** (female preponderance) |
| 好發年齡 (Onset) | **嬰兒期 (infancy)** | **20–60 歲**（成年） |
| 念珠菌感染 (Candidiasis) | **有**（mucocutaneous candidiasis，核心三聯之一） | **幾乎沒有** ⚠️ |
| 副甲狀腺低下 (Hypoparathyroidism) | **有**（核心三聯之一） | **幾乎沒有** ⚠️ |
| Addison disease | 有 | 有（兩者皆有） |
| Celiac disease | **不出現** | **常出現**（APS-II 特色） |
| 核心臨床三聯 | Candidiasis + Hypoparathyroidism + Addison | T1DM + 自體免疫甲狀腺病 + Addison |
| 診斷性抗體 | **Anti-interferon（抗 IFN）** | （無單一特異診斷抗體） |
| 常見度 | 較罕見 | **較常見** |

📊 **表格解讀（易混淆點整理）**：

- **最快區分法**：看 **念珠菌 (Candidiasis) 與副甲狀腺低下 (Hypoparathyroidism)** → **只有 APS-I 有**（兩者在 APS-II「virtually absent」）。
- **Celiac disease 反向辨識**：**APS-II 常見、APS-I 不見** → 一旦看到乳糜瀉，想 APS-II。
- 遺傳：**APS-I = 單基因 AR（AIRE）、只同代手足；APS-II = 多基因（HLA-DR3/DR4）、跨代**。
- 性別/年齡：**APS-I 男女相等、嬰兒；APS-II 女多、成年 20–60 歲**。
- 共同點：**兩者都有 Addison disease**（這是交集，別當鑑別點）。

> 🧠 **記憶法（口訣）**：
> - **APS-I = 「CHA」三角**：**C**andidiasis + **H**ypoparathyroidism + **A**ddison；基因 **AIRE**，**小孩 (infant)**、男女一樣。
>   想成 **「I 是 1 號，從小（infant）就發病，靠 AIRE（空氣）守門人」**。
> - **APS-II = 「ATD（Schmidt）」三角**：**A**ddison + **T**hyroid + **D**iabetes；**HLA-DR3/DR4**、**女性、成年**、可有 **celiac**。
> - **辨識金句**：**「念珠菌＋副甲低 = APS-I；乳糜瀉 = APS-II」**。

> 📌 **重點 (High-Yield)**：
> - APS-I：**AR / AIRE / 嬰兒 / 男女相等 / 有 candidiasis + hypoparathyroidism / anti-IFN 抗體**。
> - APS-II：**多基因 / HLA-DR3,DR4 / 成年 20–60y / 女多 / 無 candidiasis 與 hypoparathyroidism / 有 celiac**。
> - **共同交集 = Addison disease**。

## 🧬 六、其他多腺體缺乏自體免疫症候群 (Other Syndromes, Table 44.4)

### Table 44.4 罕見多腺體症候群

| 疾病 (Disorder) | 臨床特徵 (Clinical Features) | 病因 (Cause) |
|---|---|---|
| **Hirata disease**（insulin autoimmune syndrome） | Hypoglycemia（低血糖） | **Insulin 自體抗體**；與 **methimazole** 相關 |
| **IPEX** 或 **CD25 deficiency** | T1DM、Enteropathy（腸病變） | **FOXP3 突變**（IPEX）；**IL2RA 突變**（CD25 缺乏） |
| **Autoimmune lymphoproliferative syndrome type V** | 自體免疫甲狀腺炎、自體免疫血球減少、Hypogammaglobulinemia | **CTLA4 突變 → haploinsufficiency** |
| **Infantile-onset autoimmune disease 1** | 自體免疫血球減少、甲狀腺炎、T1DM、矮小 (short stature) | **STAT3 顯性活化突變**（heterozygous） |
| **Kearns-Sayre syndrome** | Hypoparathyroidism、原發性性腺衰竭、**非自體免疫** 糖尿病、Hypopituitarism | **Mitochondrial DNA 缺失** |
| **POEMS** | Polyneuropathy、Organomegaly、Diabetes、原發性性腺衰竭 | **漿細胞 dyscrasia → M protein + cytokines** |
| **Thymic tumors（thymoma）** | Myasthenia gravis、紅血球低球蛋白、自體免疫甲狀腺病、腎上腺不足 | **Thymomas** |
| **Type B insulin resistance** | 嚴重胰島素阻抗 | **Insulin receptor 自體抗體** |
| **Wolfram syndrome** | 尿崩症 (DI)、**非自體免疫** 糖尿病、雙側視神經萎縮、感音神經性聽損 | **WFS1 突變**（編碼 wolframin） |

📊 **表格解讀（陷阱整理）**：

- **「非自體免疫」糖尿病** 的兩個：**Kearns-Sayre（粒線體）** 與 **Wolfram（WFS1）**——別被誤導成自體免疫。
- **Wolfram = DIDMOAD**：**D**iabetes **I**nsipidus、**D**iabetes **M**ellitus、**O**ptic **A**trophy、**D**eafness。
- 抗體致低血糖 vs 抗體致阻抗：**Hirata（insulin 抗體 → 低血糖，與 methimazole 相關）** vs **Type B（insulin receptor 抗體 → 嚴重阻抗）**。
- **CTLA4（haploinsufficiency）/ STAT3（活化）/ FOXP3 / IL2RA** 都指向 **Treg 功能** 的核心地位。

### 重點補充：IPEX（FOXP3）

- **IPEX (MIM 340790 / 300292)**，1982 首次描述，**X-linked recessive**，**FOXP3** 基因突變。
- 臨床三大主軸：**極早發 T1DM + 嚴重 enteropathy（failure to thrive）+ dermatitis**；通常 **出生頭一兩年內死亡**，除非積極治療。
- 其他：atopy、血小板減少、溶血性貧血、hypothyroidism、淋巴結病、腎病、alopecia；免疫檢查見 **IgE 升高 + 嗜酸性球增多**。
- 機制：**FOXP3** 為 forkhead/winged-helix 轉錄因子，表現於 **CD4⁺/CD25⁺ 調節性 T 細胞 (Treg)**；突變 → 無法生成 Treg → IPEX（**scurfy mouse** 為模型）。
- 💊 治療：支持（bowel rest、TPN、insulin）；穩定期用 **calcineurin inhibitors**；**根治 = 造血幹細胞移植 (HSCT)**（可逆轉 enteropathy；但已確立的 T1DM 與甲狀腺病通常不消失）。基因治療（植入功能性 FOXP3）研發中。

### 重點補充：CTLA4 / STAT3 / LRBA 突變

- **CTLA4 loss-of-function（heterozygous）**：多器官自體免疫，與 **Treg 功能缺陷** 相關；**自體免疫較 IPEX 輕微**。
- **LRBA 突變**：類似症候群；機制可能涉及 **CTLA4 在 T 細胞內的正常運輸**。
- **STAT3 顯性活化突變**：常見 **thyroiditis、cytopenias、T1DM**，加上 **short stature** 等特徵。

### 重點補充：Anti–Insulin Receptor 抗體（Type B / acanthosis nigricans）

- 罕見（**<100 例**），又稱 **type B insulin resistance**；由 **anti–insulin receptor 抗體 + anti-insulin 抗體** 造成。
- **約 1/3** 合併其他自體免疫病（**SLE、Sjögren**）；可見 ANA↑、ESR↑、hyperglobulinemia、leukopenia、hypocomplementemia。
- 臨床：**極度胰島素阻抗**——**每日靜脈給 175,000 U insulin 仍可能無效**；但 **ketoacidosis 罕見**，病程多變，部分自發緩解，部分反而 **嚴重低血糖**（抗體的 insulin-like 效應）。
- **Acanthosis nigricans** 與阻抗狀態相關。
- 💊 NIH 治療：**rituximab（針對 B 淋巴球）+ cyclophosphamide + 脈衝式 corticosteroids**。

> 🧠 **記憶法（兩個 insulin 抗體別搞混）**：
> | | 抗體標靶 | 主要表現 | 線索 |
> |---|---|---|---|
> | **Hirata disease** | **insulin 本身** | **低血糖** | 與 **methimazole** 相關 |
> | **Type B insulin resistance** | **insulin receptor** | **嚴重高血糖/阻抗** | acanthosis nigricans、SLE/Sjögren |

> 📌 **重點 (High-Yield)**：
> - **IPEX = X-linked、FOXP3、Treg(CD4⁺CD25⁺) 缺陷**；三聯 **早發 T1DM + 腸病變 + 皮膚炎**，根治靠 **HSCT**。
> - **Treg 軸基因群**：FOXP3 / IL2RA(CD25) / CTLA4(haploinsufficiency) / LRBA / STAT3(活化)。
> - **非自體免疫糖尿病**：Kearns-Sayre（mtDNA）、Wolfram（WFS1 = DIDMOAD）。
> - **Hirata（insulin 抗體 → 低血糖）** vs **Type B（receptor 抗體 → 阻抗）**。
