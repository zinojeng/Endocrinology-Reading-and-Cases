## ⭐ High-Yield 速記（跨節濃縮）

- **皮質三分帶與酵素分佈**：
  - **絲狀帶 (Zona Glomerulosa)**：Lack CYP17A1；含有 **CYP11B2 (Aldosterone Synthase)** → 生產 **Aldosterone**（受 Angiotensin II / K+ 調控）。
  - **束狀帶 (Zona Fasciculata)**：含有 **CYP17A1**, **CYP21A2**, **CYP11B1** → 生產 **Cortisol**（受 ACTH 調控）。
  - **網狀帶 (Zona Reticularis)**：高表達 **CYP17A1 17,20-lyase** & **SULT2A1** → 生產 **DHEA / DHEAS**。
- **11β-HSD1 vs 11β-HSD2**：
  - **11β-HSD1**（主要在肝臟/脂肪）：無活性 Cortisone → 活性 **Cortisol**（還原酶）。
  - **11β-HSD2**（主要在腎臟遠曲小管）：活性 Cortisol → 無活性 **Cortisone**（氧化酶），保護 Mineralocorticoid Receptor (MR)。若受甘草酸（Glycyrrhizic acid）抑制會導致 **Apparent Mineralocorticoid Excess (AME)**。
- **Addison Disease 診斷切點**：早晨 Cortisol `<u><3 μg/dL</u>` 直接確診；標準 250 μg Cosyntropin 測試峰值 Cortisol `<u><18 μg/dL</u>` 確認診斷。急救給予 `<u>Hydrocortisone 100 mg IV bolus</u>`。
- **Cushing 兩階段診斷金律**：
  - 第一步：確認高皮質醇血症（`<u>24h UFC > ULN</u>`、`<u>Midnight Salivary Cortisol >1.45 ng/mL</u>`、`<u>1-mg DST Cortisol >1.8 μg/dL</u>`；須至少兩項異常）。
  - 第二步：測 Plasma ACTH。`<u><10 pg/mL</u>` 為 ACTH-independent（腎上腺腫瘤）；`<u>>20 pg/mL</u>` 為 ACTH-dependent。
  - 第三步：HDDST (8mg) 或 `<u>IPSS (Central:Peripheral ≥2 基線 or ≥3 post-CRH)</u>` 鑑別 Cushing 病 vs 異位 ACTH。
- **PA 篩檢與分型切點**：`<u>ARR >20–30 (ng/dL per ng/mL/h)</u>` 且 `<u>PAC >15 ng/dL</u>` 為篩檢陽性；Saline Infusion 後 `<u>PAC >10 ng/dL</u>` 確診；`<u>AVS Lateralization Index >4:1</u>` 確認單側 APA。
- **Incidentaloma CT 密度的關鍵字**：非對比 CT `<u><10 HU</u>`（Hounsfield Units）代表富含脂質之良性腺瘤（Benign Adenoma）；`<u>>20 HU</u>` 或 Delayed Contrast Washout `<u><50%</u>` 提示惡性可能。
- **CAH 最常見型態**：`<u>21-Hydroxylase deficiency (CYP21A2)</u>` 佔 >90%，`<u>17-OHP 顯著升高</u>`。失鹽型（SW）與單純男性化型（SV）為典型；非典型型（NC-CAH）常於青春期後表現多毛與不孕。
