## 🧭 0. 命名學：NEN / NET / NEC 怎麼分？

NEN（Neuroendocrine **N**eoplasm）是**最上層的總稱**，往下再依分化程度分流：

- **NEN（Neuroendocrine Neoplasm，神經內分泌腫瘤總稱）**
  - 涵蓋從「分化良好、生長相對緩慢」到「分化極差、極具侵襲性」的整個光譜。
- **NET（Neuroendocrine **T**umor，神經內分泌「腫瘤」）**
  - = well-differentiated（**分化良好**），生長相對較慢。
- **NEC（Neuroendocrine **C**arcinoma，神經內分泌「癌」）**
  - = poorly differentiated（**分化極差**），非常 aggressive。

🔬 **NEN 的共同特徵（診斷依據）**：表現出
1. neuroendocrine differentiation（神經內分泌分化）標記
2. organ-specific bioactive substances（器官特異性活性物質）
3. tissue-specific transcription factors（組織特異性轉錄因子）

⚠️ **過時術語（Obsolete terminology）— 容易在老文獻/口試踩雷**：
- 「**APUDomas**」← Anthony Pearse（1916–2003），1960 年代末提出 **APUD**（amine precursor uptake and decarboxylation）概念。
- 「**carcinoid tumors / carcinoids**」← Siegfried Oberndorfer（1876–1944），<u>1907</u> 年提出。
- 「**islet cell tumors**」（胰島細胞瘤）。
- ⚠️ 但「**carcinoid syndrome（類癌症候群）**」這個詞**至今仍廣泛使用**。

> 🧠 **記憶法（命名光譜）**：
> **「NEN 是家族，NET 是乖小孩（well-diff），NEC 是壞小孩（poorly-diff）」**。
> 三個過時名字記人名：**A**PUD = **A**nthony Pearse；**carcinoid** = Oberndorfer（**O**ld 1907）；islet cell。

> 📌 **重點 (High-Yield)**：
> - **NEN** = umbrella term；**NET** = 分化良好（well-diff）；**NEC** = 分化極差（poorly-diff）、aggressive。
> - APUDomas / carcinoids / islet cell tumors 皆為 obsolete，但「carcinoid syndrome」仍沿用。
> - NEN 三大診斷標記：neuroendocrine differentiation + organ-specific bioactive substances + tissue-specific transcription factors。

## 🔬 1. WHO 分級概念（2022 分類）

原文 KEY POINTS 指出：**WHO 2022 分類系統**對 GEP（gastroenteropancreatic）與 lung NEN 的臨床處置是「informative and necessary（具資訊性且必要）」。

WHO 2022 分類把 NEN 分為：

| 類別 | 全名 | 分化程度 |
|---|---|---|
| **G1 NET** | Grade 1 Neuroendocrine Tumor | well-differentiated |
| **G2 NET** | Grade 2 Neuroendocrine Tumor | well-differentiated |
| **G3 NET** | Grade 3 Neuroendocrine Tumor | well-differentiated（仍屬 NET！）|
| **NEC** | Neuroendocrine Carcinoma | poorly-differentiated |

### 📊 表格解讀
- **最易混淆處 → G3 NET vs NEC**：兩者都是「高級別 / 高增殖」，但本質不同。
  - **G3 NET** 仍是 **well-differentiated（分化良好）**，只是增殖活性高 → 仍歸在 NET 大類。
  - **NEC** 是 **poorly-differentiated（分化極差）**的「癌」。
- 這個區分**直接影響治療策略**（後文章節）：例如 SSL（somatostatin receptor ligands）核准用於 **G1/G2/G3 NET**；而 cytotoxic chemotherapy 多保留給 **G2-3 panNET 與 NEC**。

⚠️ **本段（行 3–48）原文未列出 Ki-67 % 與 mitotic count（有絲分裂數）的具體 G1/G2/G3 cut-off 數字**，故此處**不杜撰**閾值；實際數字請以後續 Pathology 章節（NEN Pathology）為準。此處僅保留「WHO 以分化程度 + 增殖等級分 G1/G2/G3 NET 與 NEC」的概念架構。

> 📌 **重點 (High-Yield)**：
> - WHO **2022** 分類：G1 NET、G2 NET、**G3 NET（仍是 well-diff）**、**NEC（poorly-diff）**。
> - **G3 NET ≠ NEC**：差別在「分化程度」，不只是 grade 高低。
> - SSL 核准涵蓋 G1/G2/G3 **NET**；化療多保留給 G2-3 panNET 與 NEC。

## 🌍 2. NEN 的胚胎分區與部位流行病學

NEN 多源自胚胎腸道，傳統依胚胎來源分 **foregut / midgut / hindgut**。

| 胚胎分區 | 涵蓋的 NEN 原發部位 | 重點 |
|---|---|---|
| **Foregut（前腸）** | Bronchopulmonary (BP)、thymic、tracheal、esophageal、gastric、proximal duodenal（ampulla of Vater）、pancreatic (panNENs) | BP / thymic / gastric 可另外單獨分類 |
| **Midgut（中腸）** | 由 **superior mesenteric artery** 供血的腸段；好發 **ileocecal region**；**appendix NEN** 也屬此 | Appendix NEN 視為**獨立 entity**：好發於兒童/年輕成人，行為**相對良性** |
| **Hindgut（後腸）** | rectal NEN、colon NEN | **rectal NEN 盛行率 > colon NEN**；常由（篩檢）大腸鏡發現 |

### 🫁 Bronchopulmonary (BP) NEN 細分
- BP NEN 占肺癌的 <u>20%–25%</u>、占所有組織來源 NEN 的 <u>25%–30%</u>。
- **High-grade carcinomas**：SCLC（small cell lung cancer）、LCNEC（large cell neuroendocrine carcinoma）。
- **Low-grade tumors**：
  - **Atypical carcinoids (ACs)** → **intermediate grade**。
  - **Typical carcinoids (TCs)** → **low grade**。
- **SCLC 占所有 BP NEN 約 <u>80%</u>**，為 aggressive tumor。

### 🍔 Gastric NEN 四型（重要必背！）

| 型別 | 機轉 / 背景 | 胃泌素 (gastrin) | 行為 |
|---|---|---|---|
| **Type 1** | ECL cell 多發；**chronic hypergastrinemia** 來自**autoimmune atrophic gastritis** | ↑（高）| 相對良性 |
| **Type 2** | ECL cell；gastrin 來自 **gastrinoma**，發生於 **MEN1** 背景 | ↑（高）| — |
| **Type 3** | **Sporadic、solitary**；無 gastrin 升高 | **正常** | **aggressive**（雖 well-diff 形態）|
| **Type 4** | **Poorly-differentiated carcinoma** | — | **預後差** |

> 🧠 **記憶法（Gastric NEN 四型）**：
> **「1、2 號靠 gastrin（高胃泌素），3 號孤鳥（sporadic、gastrin 正常但很兇），4 號最差（poorly-diff）」**。
> - Type 1 = **自體免疫萎縮性胃炎**（autoimmune atrophic gastritis）。
> - Type 2 = **MEN1 + gastrinoma**。
> - 形態 well-diff ≠ 行為良性 → **Type 3** 就是反例（陷阱）。

### 📈 流行病學數字
- 過去 **4 個十年**，GEP NEN + BP NEN 整體 incidence 增加 <u>3.5–5 倍</u>。
- 西方世界增幅**最高 = gastric + rectal NEN**；**最低 = small intestinal + cecal NEN**。
- GEP NEN 年發生率約 <u>3.5–4 / 100,000</u> 人。
- **亞洲病人**：small intestinal NEN 罕見；**gastric 與 rectal NEN 較盛行**。
- 其他罕見原發部位：liver (bile ducts)、ovaries、testes、prostate、kidneys、breasts、inner ear、skin。
  - ⚠️ NEN 也可**轉移**到 ovaries/testes/kidneys/breasts/skin → 評估原發灶時要警覺。

> 📌 **重點 (High-Yield)**：
> - Foregut（含 BP、thymic、gastric、panNEN）/ Midgut（ileocecal、appendix）/ Hindgut（rectal > colon）。
> - **SCLC ≈ 80% 的 BP NEN**；BP NEN 占肺癌 20–25%、占所有 NEN 25–30%。
> - Gastric NEN：Type 1=萎縮性胃炎、Type 2=MEN1/gastrinoma、Type 3=sporadic 且 aggressive、Type 4=poorly-diff。
> - GEP NEN 年發生率 **3.5–4/10 萬**；40 年來整體增 **3.5–5 倍**。

## 🧬 3. Genetic Syndromes With NENs（遺傳症候群 → 相關 NEN）

NEN 可發生於多種遺傳症候群。原文（KEY POINT + 內文）明確列出的症候群包括：
**MEN1、MEN4、VHL、NF1、Pacak-Zhuang syndrome、Mahvash disease、insulinomatosis**。

### 📊 症候群 → 基因 → 相關 NEN 對照表

| 症候群 | MIM number | 致病基因 / 機轉 | 相關 NEN（原文敘述） |
|---|---|---|---|
| **MEN1**（multiple endocrine neoplasia type 1） | **131100** | （MEN1）— 原文未在本段標基因符號 | 原發 **pancreatic、gastric、duodenal、thymic、BP NEN** |
| **MEN4**（multiple endocrine neoplasia type 4） | **610755** | **inactivating mutation in *CDKN1B***（cyclin-dependent kinase inhibitor 1B） | **GEP 與 BP NEN** |
| **VHL**（von Hippel-Lindau disease） | **193300** | （VHL）— 本段未標基因符號 | **panNEN** |
| **NF1**（neurofibromatosis 1） | **162200** | （NF1）— 本段未標基因符號 | **Ampullary-type duodenal somatostatinoma** 與 **panNEN** |
| **Pacak-Zhuang syndrome** | **603349** | ***EPAS1* gain-of-function** → HIF-2α 降解↓、穩定化↑ | **Somatostatinoma** |
| **Mahvash disease** | **619290** | ***GCGR*（glucagon receptor）inactivating mutation** | 多發 **glucagon-producing panNEN** + diffuse **α cell hyperplasia**；但**無**臨床 glucagonoma syndrome |
| **Insulinomatosis**（多發胰島素瘤；autosomal dominant） | **147630** | ***MAFA*（MAF BZIP transcription factor A）mutation** | **Multiple insulinomas of the pancreas** |

### 📊 表格解讀（臨床意義與易混淆處）
- **MEN1 是 NEN 遺傳症候群的「主角」**：可同時涵蓋 pancreatic / gastric / duodenal / thymic / BP 多部位 NEN，也是 **Type 2 gastric NEN** 與 **gastrinoma** 的背景（連結到上方胃 NEN 表）。
- **MEN4 的記憶鉤子**：基因是 ***CDKN1B***（一個 CDK inhibitor）→ inactivating mutation；表型近似 MEN1 但基因不同 → 兩者常被一起考。
- **Pacak-Zhuang vs Mahvash（兩個都會出 somatostatinoma / α-cell 相關，但機轉相反）**：
  - **Pacak-Zhuang**：*EPAS1* **gain-of-function** → HIF-2α 穩定 → **somatostatinoma**（屬假性缺氧 pseudohypoxia 路徑）。
  - **Mahvash**：*GCGR* **loss-of-function（inactivating）** → α 細胞增生 + glucagon-producing panNEN，**但臨床上沒有 glucagonoma 症候群**（陷阱：有 glucagon 分泌卻無症候群）。
- **NF1 的招牌 NEN** = **duodenal（ampullary-type）somatostatinoma** → 「NF1 + 十二指腸壺腹 somatostatinoma」是經典配對。
- **Insulinomatosis**：autosomal dominant、**多發**胰島素瘤，基因為 ***MAFA*** → 與「單一散發 insulinoma」鑑別。

> 🧠 **記憶法（症候群 → 基因/NEN）**：
> 用一句串聯易混的「基因 vs 表型」——
> - **MEN4 → CDKN1B**（「MEN**4**ree the CDK inhibitor」= 抑制子壞掉）。
> - **Pacak-Zhuang → EPAS1（=HIF2A 基因）gain → somatostatinoma**（「**P**acak = **P**seudohypoxia」）。
> - **Mahvash → GCGR loss → α 細胞增生但無 glucagonoma syndrome**（「Mahvash = **M**uted glucagon syndrome」）。
> - **Insulinomatosis → MAFA**（「**MA**ny insulinomas → **MAFA**」）。
> - **NF1 → duodenal somatostatinoma**（「NF1 愛十二指腸壺腹 SST 瘤」）。

> 🧠 **記憶法（哪些症候群會有 NEN，一次背齊）**：
> **「M-M-V-N-P-M-I」**：**M**EN1、**M**EN4、**V**HL、**N**F1、**P**acak-Zhuang、**M**ahvash、**I**nsulinomatosis。

> 📌 **重點 (High-Yield)**：
> - **MEN4 = *CDKN1B*** inactivating mutation（MIM 610755）；表型似 MEN1。
> - **Pacak-Zhuang = *EPAS1* gain-of-function → HIF-2α↑ → somatostatinoma**。
> - **Mahvash = *GCGR* inactivating → 多發 glucagon-producing panNEN + α-cell hyperplasia，但無 glucagonoma syndrome**。
> - **Insulinomatosis = *MAFA* mutation（AD）→ multiple insulinomas**。
> - **NF1 → ampullary duodenal somatostatinoma + panNEN**；**VHL → panNEN**。

## 🩺 4. Endocrine-Related Symptoms and Syndromes（引言概念）

- NEN（孤立或轉移）可因**過度分泌一種或多種 peptide / amine** 而表現出荷爾蒙相關症狀與症候群。
- 分泌型態分兩類：
  - **Eutopic secretion（同位分泌）**：產物為「該組織原本就會產生」者 → 形成 secretory syndrome。
  - **Ectopic secretion（異位分泌，較罕見）**：bioactive peptide/hormone 來自「通常不會由該部位分泌」的來源。
- ⚠️ **大多數 NEN 是 nonfunctioning（無功能）**：不分泌具生物意義的活性 peptide/hormone。
  - 例：nonfunctioning pancreatic NET — ICD-O **8150/3**；ICD-11 **2C10.1 / XH3709**。

> 📌 **重點 (High-Yield)**：
> - 多數 NEN 為 **nonfunctioning**；functioning 者才出現荷爾蒙症候群。
> - 分泌分 **eutopic（同位）** 與 **ectopic（異位，罕見）** 兩型。
