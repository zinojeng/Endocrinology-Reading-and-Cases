# Ch.43 Neuroendocrine Tumors and Disorders — NEN 病理與分期系統

> 來源：Williams Textbook of Endocrinology, Ch.43 Neuroendocrine Tumors and Disorders — NEN Pathology、分級 (WHO 2022)、分期 (ENETS/AJCC panNEN TNM)、支氣管肺 (BP) NEN 分類

---

## 🔬 一、NEN 的組織學診斷與免疫標記

所有懷疑 NEN 的病人，都應以 **biopsy 或 resection** 取得組織做 histology。

### 1. 神經內分泌分化的「通用標記」(general markers)
- **Chromogranin** (嗜鉻顆粒蛋白)
- **Synaptophysin** (突觸素)

### 2. 功能性 panNEN 的荷爾蒙免疫染色 (IHC)
- **Insulin、Gastrin、Glucagon** → 用來辨識功能性 panNEN 的分泌型別。

### 3. 預後 / 治療預測標記 (prognostic & theranostic)
- **somatostatin receptor subtype 2 (SST₂)** → 與 <u>SSTR 影像 (如 Ga-DOTA PET)</u> 及 <u>SSA / PRRT 治療反應</u> 相關，因此是 theranostic（療效預測）標記。

> 📌 **重點 (High-Yield)**：
> - 診斷 NEN 必背雙通用標記：**Chromogranin + Synaptophysin**。
> - 功能性 panNEN 加做 insulin / gastrin / glucagon IHC。
> - **SST₂** 是 theranostic 標記（影像 + PRRT/SSA 療效預測）。

## 🧬 二、辨識「不明原發部位」NEN 的轉錄因子 (transcription factors)

當原發位置不明時，可**依序施用**轉錄因子 IHC 來定位來源。

| 部位 / 來源 | 表現 (positive) | 不表現 (negative) |
|---|---|---|
| **GEP NEN（一般）** | **CDX2** (caudal-type homeobox 2) | **TTF1** (thyroid transcription factor 1) |
| **Duodenum / Pancreas** | **ISL1**、**PDX1**、**ARX** | — |
| **Hindgut（後腸）NEN** | **PSAP** (prostate-specific acid phosphatase) | — |
| **BP NEN（支氣管肺）/ Thymic NEN** | **TTF1**（可表現） | — |

📊 **表格解讀**：
- **GEP NEN 的金句**：<u>CDX2 (+) / TTF1 (−)</u>；反之 **BP / thymic NEN 可 TTF1 (+)** → 故 TTF1 是區分「肺/胸腺來源 vs 消化道來源」的關鍵分水嶺。
- Duodenum 與 pancreas 共享 **ISL1 / PDX1 / ARX**（前腸—胰十二指腸同源）。
- **後腸 (hindgut)** 的特異印記是 **PSAP**（注意：PSAP 是攝護腺酸性磷酸酶，但在此用於 hindgut NEN，不要誤判為攝護腺癌）。

🧠 **記憶法（轉錄因子定位）**：
- **「C 腸 D 肺」** → **C**DX2 看腸道、**TTF1** 看肺/胸腺（D = 對立負染色）。
- **十二指腸+胰臟 = 「IPA」三字訣**：**I**SL1、**P**DX1、**A**RX。
- **後腸找 PSAP**（後腸 = Posterior → **P**SAP）。

> 📌 **重點 (High-Yield)**：
> - GEP NEN：**CDX2 (+), TTF1 (−)**；BP/thymic NEN：**TTF1 可 (+)**。
> - Duodenum/Pancreas：**ISL1 / PDX1 / ARX**。
> - Hindgut：**PSAP**。

## 📊 三、WHO 2022 分級系統（GI 與胰膽道上皮性 NEN）— TABLE 43.2

GEP NEN 的分級依 **proliferative indices（Ki-67 與 mitotic index）** 與**分化程度 (degree of differentiation)** 決定。

### TABLE 43.2 — 2022 WHO Grading of Epithelial NEN of the GI & Pancreaticobiliary Tract（完整重排）

| 定義 / Grade | Ki-67 Index (%) | Mitotic Count (/2 mm²) |
|---|---|---|
| **Well-Differentiated NETs（分化良好）** | | |
| G1 NET | <u>&lt;3</u> | <u>&lt;2</u> |
| G2 NET | <u>3–20</u> | <u>2–20</u> |
| G3 NET | <u>&gt;20</u> | <u>&gt;20</u> |
| **Poorly Differentiated NECs（分化不良，預設即 G3）** | | |
| Small cell NEC | Small cell cytomorphology（小細胞型態） | &gt;20 |
| Large cell NEC | Large cell cytomorphology（大細胞型態） | &gt;20 |

> 註：原文內文以 **10 HPF** 描述 mitotic cutoff（G1 &lt;2 mitoses/10 HPF；G2 2–20/10 HPF；G3 &gt;20/10 HPF），而 WHO 2022 表格採 **/2 mm²** 為單位；數值門檻相同（&lt;2、2–20、&gt;20）。

📊 **表格解讀（最重要的觀念）**：
- **三級分法**：G1（low）／G2（intermediate）／G3（high）依 **Ki-67** 與 **mitotic count** 切分。
- **G3 再依分化程度二分**：
  - **G3 NET**：腫瘤細胞**分化良好 (well-differentiated)**。
  - **G3 NEC**：腫瘤細胞**分化不良 (poorly-differentiated)** → 預設即為 G3，再分 small cell 與 large cell 型態。
- ⚠️ **G3 NET ≠ G3 NEC**：兩者在 **病人存活、分子致病機轉、治療策略** 都「差異極大」。雖然 Ki-67 都 &gt;20%，但 NET 是「分化好但增殖快」，NEC 是「真正去分化的癌」。

🧠 **記憶法（Ki-67 cutoff）**：
- **「3 與 20」是兩道牆**：**&lt;3 = G1**、**3–20 = G2**、**&gt;20 = G3**。
- Mitotic count 同樣是 **「2 與 20」**：**&lt;2 / 2–20 / &gt;20**。
- 口訣：**「低三、高廿」**（G1 看 3%、G3 看 20%）。

### 🧠 G3 NET vs G3 NEC 比較表（易混淆核心）

| 項目 | **G3 NET** | **G3 NEC** |
|---|---|---|
| 分化程度 | Well-differentiated（分化良好） | Poorly-differentiated（分化不良） |
| Ki-67 | &gt;20% | &gt;20% |
| 細胞型態 | — | Small cell / Large cell |
| 支持診斷的分子/IHC（panNEN） | 常見 **ATRX/DAXX 缺失**、**MEN1** 突變、或 **p27**、且 **保留 SST₂&₅ 染色** | 通常無上述 well-diff 特徵 |
| 預後 / 機轉 / 治療 | 與 NEC「差異極大」 | 與 NET「差異極大」 |

> 📌 **重點 (High-Yield)**：
> - Ki-67：**&lt;3 / 3–20 / &gt;20 = G1 / G2 / G3**；Mitoses：**&lt;2 / 2–20 / &gt;20**。
> - **G3 NET = 分化好**、**NEC = 分化不良（預設 G3，分 small/large cell）**。
> - panNEN 中支持 **G3 NET** 的線索：**ATRX/DAXX loss、MEN1 mut、p27、保留 SST₂&₅**。

## 🧬 四、panNEN 與小腸 NEN 的分子特徵（高分子背景）

### 1. 非功能性 panNEN 的三種 epigenomic/transcriptomic 亞型

| 亞型 | 細胞特徵 | 分子特徵 | 預後 |
|---|---|---|---|
| **第一型** | 類 **B cell** | 無 copy number alteration、無 MEN1、無 ATRX/DAXX 突變（故**無 ALT**） | **預後佳 (favorable)** |
| **第二型** | **ARX (+)**、A-cell 特徵 | copy number variation 有限、復發性 **MEN1** 突變 | 中間 |
| **第三型** | **ARX (+)**、A-cell 特徵較不明顯 | **顯著 copy number alteration**、**MEN1** 突變、復發性 **ATRX/DAXX** 突變 | 較差 |

- ⚠️ **ARX 表現的非功能性 panNEN 轉移潛能最高**，可能因其易**啟動 ALT (alternative lengthening of telomeres)**。

### 2. Insulinoma：indolent vs aggressive（用 A/B cell 特徵區分）

| 類型 | 細胞特徵 | 表現 | 行為 |
|---|---|---|---|
| **Indolent（非轉移）insulinoma** | 強烈類 **B cell** epigenetic | **PDX1 (+)、ARX (−)** | 良性、非轉移 |
| **Aggressive（轉移）insulinoma**（較罕見） | 類 nonfunctioning NET | **ARX (+)** | **腫瘤大 (&gt;3 cm)**、轉移行為 |

- 推論兩型 **起源不同**；aggressive insulinoma 與 nonfunctioning NET 較相關。

### 3. 小腸 (small intestinal) NEN
- **Chromosome 18 缺失**：見於 <u>60%–90%</u> 的小腸 NEN，但目前**未發現該染色體上的突變基因**。
- **CDKN1B**：是小腸 NEN 唯一被證實的突變基因，但**頻率僅 <u>8%</u>** → 作為 driver 的角色有限。

🧠 **記憶法**：
- **ARX = Aggressive / metastatic**（A 開頭→侵襲）；**PDX1 (+)/ARX (−) = 乖巧 B-cell-like = indolent**。
- 小腸 NEN：**「18 號染色體丟失 (60–90%) 但無基因；CDKN1B 只 8%」**。

> 📌 **重點 (High-Yield)**：
> - 非功能性 panNEN：類 B-cell 型**預後最佳**；**ARX (+) 型轉移潛能最高（ALT）**。
> - Insulinoma：**ARX (+) + &gt;3 cm → aggressive/轉移**；**PDX1(+)/ARX(−) → indolent**。
> - 小腸 NEN：Chr18 loss **60–90%**；CDKN1B 突變僅 **8%**。

## 📊 五、panNEN 分期：ENETS vs AJCC/UICC 8th — TABLE 43.3

GEP NEN 採 **TNM 分期**（腫瘤大小與侵犯、淋巴結、遠端轉移）。

### TABLE 43.3 — ENETS 與 AJCC TNM Staging（panNEN，完整重排）

| T-N-M | **ENETS** | **AJCC / UICC 8th Edition** |
|---|---|---|
| **T1** | 局限於胰臟ᵃ，&lt;2 cm | 局限於胰臟ᵃ，&lt;2 cm |
| **T2** | 局限於胰臟ᵃ，2–4 cm | 局限於胰臟ᵃ，2–4 cm |
| **T3** | 局限於胰臟ᵃ，&gt;4 cm，或侵犯 duodenum 或 common bile duct | 局限於胰臟ᵃ，&gt;4 cm，或侵犯 duodenum 或 common bile duct |
| **T4** | 侵犯鄰近結構（stomach、spleen、colon，或大血管壁含 celiac axis 或 SMA） | 侵犯鄰近器官（stomach、spleen、colon、**adrenal gland**）或大血管壁（celiac axis 或 SMA） |
| **N0** | 無區域淋巴結轉移 | 無區域淋巴結轉移 |
| **N1** | 區域淋巴結轉移 | 區域淋巴結轉移 |
| **M0** | 無遠端轉移 | 無遠端轉移 |
| **M1** | 遠端轉移 | 遠端轉移 |
| **M1a** | — | 僅 **肝臟** 轉移 |
| **M1b** | — | 僅 **肝外** 轉移 |
| **M1c** | — | **肝 + 肝外** 皆有 |

### Stage grouping 對照

| **AJCC / UICC 8th** | TNM | | **ENETS** | TNM |
|---|---|---|---|---|
| **Ia** | T1 N0 M0 | | **I** | T1 N0 M0 |
| **Ib** | T2 N0 M0 | | **IIA** | T2 N0 M0 |
| **IIa** | T3 N0 M0 | | **IIB** | T3 N0 M0 |
| **IIb** | T1–3 N1 M0 | | **IIIA** | **T4 N0 M0** |
| **III** | **T4 N0–1 M0** | | **IIIB** | **Any T N1 M0** |
| **IV** | Any T M1 | | **IV** | Any T Any N M1 |

> ᵃ「Limited to the pancreas」= 未侵犯鄰近器官或大血管壁；**延伸至胰周脂肪 (peripancreatic adipose tissue) 仍算「局限於胰臟」**。

📊 **表格解讀 — ENETS vs AJCC 的關鍵差異**：
1. **T1–T3 兩者完全相同**（大小 2 cm / 4 cm 為界）。
2. **T4 的器官清單不同**：
   - **ENETS T4** 列：stomach、spleen、colon、大血管壁。
   - **AJCC T4** 多列 **adrenal gland（腎上腺）**。
3. **M1 細分只有 AJCC 有**：AJCC 把 M1 分成 **M1a（僅肝）/ M1b（僅肝外）/ M1c（肝+肝外）**；ENETS 不細分。
4. **Stage grouping 的最大歧異在 T4 與 N1 如何歸 stage III**：
   - **AJCC**：**T4 N0–1 M0 → Stage III**；而 **T1–3 N1 M0 → 只到 IIb**（即淋巴結陽性但 T 較小者仍偏早期）。
   - **ENETS**：**T4 N0 M0 → IIIA**、**Any T N1 M0 → IIIB**（即**只要 N1 就升到 III 期**）。
   - 👉 **核心差異**：同樣是 **N1（淋巴結陽性、無 T4、無 M1）**，**ENETS 歸 Stage III (IIIB)**，但 **AJCC 仍歸 Stage II (IIb)** → ENETS 對淋巴結轉移「更重視、分期更高」。

🧠 **記憶法（ENETS vs AJCC）**：
- **「A 多腎、A 細肝」**：**A**JCC 的 T4 多 **adrenal**；**A**JCC 才把 **M1 細分成 a/b/c（肝相關）**。
- **「N1 之爭」**：**ENETS 看到 N1 就跳 III**（嚴格）；**AJCC 的 N1（T1–3）還停在 II**（寬鬆）。

> 📌 **重點 (High-Yield)**：
> - **T1/T2/T3 = &lt;2 / 2–4 / &gt;4 cm**（兩系統一致）。
> - **AJCC T4 含 adrenal gland；AJCC 獨有 M1a/b/c（肝 / 肝外 / 兩者）**。
> - **N1 分期之別**：ENETS → **Stage III (IIIB)**；AJCC（T1–3 N1）→ **Stage IIb**。
> - 胰周脂肪侵犯仍算「limited to pancreas」。

## 📊 六、支氣管肺 NEN (BP NEN)：WHO 2022 分類 — TABLE 43.4

BP NEN 分為：
- **高惡性度 (high-grade)**：**SCLC**（small cell lung carcinoma）與 **LCNEC**（large cell NEC）。
- **低惡性度 (low-grade)**：**AC**（atypical carcinoid）與 **TC**（typical carcinoid）。

定義性組織學標準：**每 2 mm² 的 mitotic count、necrosis 有無、細胞大小與形狀、核特徵、整體架構**。

### TABLE 43.4 — BP NEN（2022 WHO Lung Tumors，完整重排）

| 定義 / Grade | 特異特徵 (Specific Features) | Mitotic Count (/2 mm²) |
|---|---|---|
| **Well-Differentiated NETs（分化良好）** | | |
| **Typical carcinoid (TC)** / G1 NET | **無 necrosis** | <u>&lt;2</u> |
| **Atypical carcinoid (AC)** / G2 NET | **有 necrosis** | <u>2–10</u> |
| Carcinoids/NETs with elevated mitotic counts and/or Ki67 | Atypical carcinoid 型態 | **&gt;10 且/或 Ki67 &gt;30%** |
| **Poorly Differentiated NECs（分化不良）** | | |
| **Small cell (lung) carcinoma (SCLC)** | 常有 necrosis + 小細胞型態 | &gt;10 |
| **Large cell NEC (LCNEC)** | 幾乎總是 necrosis + 大細胞型態 | &gt;10 |

📊 **表格解讀**：
- **TC vs AC 的兩大鑑別軸**：
  - **Mitotic count**：TC **&lt;2** vs AC **2–10**。
  - **Necrosis**：TC **無** vs AC **有**。
- **新增的灰區類別**：「**carcinoids/NETs with elevated mitotic counts and/or Ki67**」→ 型態像 atypical carcinoid，但 **mitoses &gt;10 且/或 Ki67 &gt;30%**，是介於 carcinoid 與 NEC 間的高增殖 NET（對應前述 GEP 的「G3 NET」概念）。
- **NEC 兩型（SCLC / LCNEC）**：mitoses 皆 **&gt;10**，靠 **細胞型態（小 vs 大）** 與 necrosis 程度區分（LCNEC「幾乎總是」壞死）。
- ⚠️ **TTF1 陷阱**：**BP NEN 與 thymic NEN 可表現 TTF1**（與 GEP NEN 的 CDX2(+)/TTF1(−) 相反）。

🧠 **記憶法（BP carcinoid）**：
- **「TC 乖、AC 壞」**：**T**ypical **C**arcinoid → 無壞死、mitoses &lt;2；**A**typical → **A** for **A**bnormal（有 necrosis、mitoses 2–10）。
- **數字牆**：**「2 與 10」** → carcinoid 用 **2 (TC/AC 界)** 與 **10 (carcinoid 與 NEC 界)**；對比 GEP 用「3 與 20」。

### 🧠 GEP NEN vs BP NEN 分級對照（避免數字混淆）

| | **GEP NEN (WHO 2022)** | **BP NEN (WHO 2022)** |
|---|---|---|
| 低度界線 (mitoses) | **&lt;2 / 2–20 / &gt;20** | **&lt;2 (TC) / 2–10 (AC) / &gt;10 (NEC)** |
| 主要 Ki-67 切點 | **3% 與 20%** | 灰區 NET 用 **Ki67 &gt;30%** |
| Well-diff 命名 | G1/G2/G3 NET | Typical / Atypical carcinoid |
| Poorly-diff (NEC) | Small cell / Large cell NEC | **SCLC / LCNEC** |

> 📌 **重點 (High-Yield)**：
> - **TC：無 necrosis、mitoses &lt;2**；**AC：有 necrosis、mitoses 2–10**。
> - **SCLC / LCNEC：mitoses &gt;10**，靠細胞型態 + necrosis 區分。
> - BP NEN 用 **「2 與 10」**；新增灰區 NET 用 **Ki67 &gt;30%**。
> - **BP/thymic NEN 可 TTF1 (+)**。

## 🔬 七、特殊實體：MiNEN、NEHI、DIPNECH

### 1. MiNEN (Mixed Neuroendocrine–Nonneuroendocrine Neoplasms)
- 定義：**混合性上皮腫瘤**，含一個 **neuroendocrine 成分** + 一個 **nonneuroendocrine 成分**，兩者在**型態與免疫組化上各自獨立可辨**。
- ⚠️ 須與「**腺癌中散在偶見神經內分泌細胞**」或「異常表現神經內分泌分化的惡性腫瘤」**區分**（後者不是 MiNEN）。
- **預後取決於生物學上最惡性的成分** → 因此**兩個成分都要各自描述、定量、分級**。
- 舊稱：**MANEC (mixed adeno-neuroendocrine carcinoma)**。

### 2. NEHI (Neuroendocrine cell hyperplasia of infancy)
- **罕見的兒科** interstitial / diffuse lung disease。
- 在「其他正常肺組織」的**遠端肺上皮**內神經內分泌細胞增多。

### 3. DIPNECH (Diffuse Idiopathic Pulmonary Neuroendocrine Cell Hyperplasia)
- **罕見成人發病**的肺症候群：散在小結節含 BP NEN，或 BP NEN 的線性增生。
- 舊稱這些特徵為 **tumorlets**。
- 流行病學：**多為女性**；**與抽菸或其他肺病無關**。
- 臨床表現：**咳嗽、用力時呼吸困難 (exertional dyspnea)**；肺功能檢查常見 **阻塞性或混合性阻塞/限制性** 型態。

🧠 **記憶法**：
- **MiNEN = 兩成分各自獨立**，**預後看最壞的那個**（舊名 MANEC）。
- **NEHI = iN**fancy（嬰幼兒）；**DIPNECH = aDult**（成人、女性、不抽菸、tumorlets）。

> 📌 **重點 (High-Yield)**：
> - **MiNEN**：兩成分形態/IHC 各自獨立，**預後依最惡性成分**，分別分級；舊稱 **MANEC**。
> - **NEHI = 嬰幼兒**；**DIPNECH = 成人、女性、非吸菸**，舊稱 **tumorlets**，肺功能呈阻塞/混合型。
