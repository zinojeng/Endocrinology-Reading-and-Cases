## 🧠 一頁總複習（高頻記憶點）

- **NET vs NEC**：NET = well-differentiated（G1/G2/G3，依 Ki-67 與 mitotic count）；NEC = poorly-differentiated（small/large cell，化療為主）。
- **影像 flip-flop**：分化好 → <u>SSTR PET (68Ga-DOTATATE) 陽性</u>；分化差 / 高 Ki-67 → <u>18F-FDG PET 陽性</u>。
- **功能性 panNEN 三本柱**：Insulinoma（低血糖，Whipple triad）、Gastrinoma（ZES，secretin test）、VIPoma（WDHA / Verner-Morrison）。
- **Carcinoid syndrome**：flushing + diarrhea + 右心瓣膜病 (Hedinger)；診斷靠 <u>尿 5-HIAA</u>，治療首選 <u>somatostatin analog</u>。
- **全身治療軸**：SSA → PRRT (177Lu-DOTATATE) / everolimus / sunitinib（NET）；platinum-etoposide / CAPTEM（NEC / 高分級）。
