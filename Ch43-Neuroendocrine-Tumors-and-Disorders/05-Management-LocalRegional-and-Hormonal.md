# Ch.43 Neuroendocrine Tumors and Disorders — NEN 整體處置與荷爾蒙症候群治療

> 來源：Williams Textbook of Endocrinology, Ch.43 Neuroendocrine Tumors and Disorders — Management（Watchful waiting、Local-regional、Extensive disease、Liver-directed、Hormonal syndromes）

---

## 🩺 1. Management 引言：腫瘤量決定一切

NEN 的處置必須「**個別化 (tailored)**」。核心概念：

- 荷爾蒙症候群的處置不只看「給什麼藥」，更要先想**能不能開刀**。
- ⭐ **腫瘤量 (tumor bulk) 常與荷爾蒙濃度相關** → 因此即使無法根治，**radical resection 或 debulking（減積手術）** 也能降低荷爾蒙、減輕內分泌症候群嚴重度。
- 對症狀未控制 (uncontrolled) 的病人，應於**術前 (preoperative) 與術中 (perioperative)** 先給抗荷爾蒙或控制荷爾蒙過量負面效應的治療。

> 📌 **重點 (High-Yield)**：
> - NEN 處置 = 個別化。
> - <u>腫瘤量 ↔ 荷爾蒙濃度</u>相關，所以「**減積手術**」本身就是控制症候群的手段。
> - 症狀未控制者，**先在術前/術中壓制荷爾蒙**再手術。

## 🩺 2. 治療階梯總覽：依疾病範圍 (Treatment Ladder)

把整個 Management 章節整理成一條「由輕到重」的階梯。

### 📊 依疾病範圍的治療階梯表

| 階梯 | 疾病範圍 | 首選策略 | 具體做法（原文） |
|---|---|---|---|
| ① **Watchful waiting** 觀察追蹤 | G1 NET、limited tumor sites；非轉移無功能 panNET 直徑 <u>≤2 cm</u> | 不一定要第一線抗增生治療；密切追蹤 | G1-2 NET 生長**惰性 (indolent)**；PANDORA/ASPEN 支持 ≤2 cm 無功能 panNEN 可追蹤（尤其 MEN1、VHL） |
| ② **Local-regional 局部治療** | Stage I–III（局部）可切除 | **手術根治**或內視鏡切除 | EMR / ESMR-L / ESD / TEMS（消化道）；segmentectomy / wedge / (bi)lobectomy / pulmonectomy（BP·thymic NEN）；EUS-guided RFA（小功能 panNEN ≤2 cm） |
| ③ **Extensive disease 廣泛/轉移** | Stage IV、midgut 轉移為主 | **姑息手術 (palliative surgery)** 防併發症 | 解決 mesenteric desmoplastic reaction 造成的 ischemia / venous congestion / ileus / 腸穿孔；colon NEN 單一病灶仍應切除（行為較 small bowel 惡性） |
| ④ **Liver-directed 肝臟導向** | 肝轉移（liver-dominant / oligometastatic） | 切除 / 消融 / 經動脈栓塞 | metastasectomy、hemihepatectomy、segment resection；RFA / MWA（小且寡轉移）；TAE / TACE / TARE；極選病例 → 肝移植 |
| ⑤ **Systemic 全身治療** | 不可切除/廣泛轉移 + 荷爾蒙症候群 | **SRL、PRRT、everolimus、telotristat** 等（見下方症候群表） | 用於控制荷爾蒙與抗增生 |

### 📊 表格解讀

- **Watchful waiting 的證據基礎**：PROMID trial 中 octreotide 與 placebo 的**中位 OS 相近**（<u>84.7 vs 83.7 個月</u>）→ 對惰性 G1 NET 觀察是合理的。
- ⚠️ **易混淆**：「≤2 cm 無功能 panNET 可追蹤」只適用於**非轉移、無功能**者；有功能或有淋巴結轉移就不在此列。
- **局部治療的「直徑門檻」是高頻考點**（見下方專表）。
- **姑息手術的精髓**：midgut NEN 多以轉移呈現，開刀重點不是根治，而是**防止 mesenteric fibrosis 造成的腸缺血、阻塞、穿孔**。

### 📊 局部治療「直徑門檻」速查表（高頻必背）

| 部位 | 門檻 | 處置 |
|---|---|---|
| Gastroduodenal NEN（無淋巴結轉移） | 直徑 <u>1–2 cm</u> | 內視鏡黏膜切除 **EMR** |
| Rectal NEN（無淋巴結轉移） | 直徑 <u><1.5 cm</u> | EMR 或 **ESMR-L**（其他：ESD / TEMS） |
| Gastric type 1 NEN | 直徑 <u><1 cm</u> | 預後極佳、無疾病相關死亡 → **每 1–2 年內視鏡監測** |
| Duodenal NEN | 直徑 <u>≤1.0 cm</u> | 內視鏡切除；>1.0 cm → 外科 duodenectomy |
| 功能性 panNEN | 直徑 <u>≤2 cm</u> | 切除 / enucleation；選擇性病例可 **EUS-guided RFA** |

> 🧠 **記憶法（直徑門檻）**：「**胃十二指腸 2、直腸 1.5、第一型胃與十二指腸 1**」
> → Gastroduodenal ≤**2**、Rectal <**1.5**、Gastric type1 / Duodenal ~**1**。數字越往下游（rectal→gastric type1）門檻越小。

> 📌 **重點 (High-Yield)**：
> - 階梯：<u>Watchful waiting → 手術/局部 → 肝臟導向 → 全身</u>。
> - **PROMID**：octreotide vs placebo OS <u>84.7 vs 83.7 mo</u>（觀察的依據）。
> - midgut 手術多為**姑息**，目的防 mesenteric fibrosis 併發症。

## 🩺 3. Gastric NEN 分型處置（type 1–4）

胃 NEN 依型別差很多，務必分開記。

| 型別 | 處置（原文） | 備註 |
|---|---|---|
| **Type 1** | 直徑 <1 cm 預後極佳 → 每 1–2 年內視鏡監測 | 無疾病相關死亡 |
| **Type 2** | sporadic 個案考慮**內視鏡或外科切除** | 中度惡性潛能；**須一併考量並存的 duodenal/pancreatic gastrinoma 及其轉移** |
| **Type 3** | **外科 partial/complete gastrectomy + 淋巴結廓清** | — |
| **Type 4** | 同 type 3：外科 gastrectomy + lymphadenectomy | — |

> 🧠 **記憶法**：「**1 看、2 想 gastrinoma、3/4 開刀切胃**」（type1 surveillance；type2 要連想 gastrin 來源；type3/4 gastrectomy）。

> 📌 **重點 (High-Yield)**：
> - **Gastric type 2** 是「gastrinoma 相關」型 → 評估 duodenal/pancreatic gastrinoma。
> - **Type 3/4** → gastrectomy + lymphadenectomy。

## 🩺 4. Liver-Directed Therapies（肝臟導向治療詳解）

肝轉移是 NEN 病程與症候群的關鍵。手段分三大類：

### 🔪 (A) 手術 / 消融
- **Resection**：原發切除 + 局部淋巴結廓清 + 肝轉移 metastasectomy / hemihepatectomy / segment resection → 與**長期 PFS** 相關。
- **Ablation**：RFA 或 **MWA（microwave ablation）** → 用於**體積有限的寡轉移 (oligometastatic)**。

### 🩸 (B) 經肝動脈栓塞 (Transarterial embolization)

| 技術 | 機轉 |
|---|---|
| **TAE**（bland particle embolization 純顆粒栓塞） | 缺血 (ischemia) |
| **TACE**（chemoembolization 化療栓塞） | 缺血 + 化療 |
| **TARE**（radioembolization 放射栓塞，radiospheres） | **放射誘發 DNA damage**（非缺血） |

- ⚠️ **毒性比較**：原文指出 **TACE 觀察到比 TAE 更高的毒性 (higher rates of toxicity)**。
- **TARE 的腫瘤反應來自輻射 DNA 損傷**，而非 TAE 的缺血機轉。

### 🫁 (C) 肝移植 (Liver transplantation)
- **極選病例 (very select cases)**：**無肝外疾病**、多葉肝轉移、且**原發 NEN 及其局部轉移已完全切除**後，由**專科中心 (specialized centers)** 考慮。

### 📊 表格解讀
- 三類栓塞差別在「**靠什麼殺腫瘤**」：TAE/TACE 靠**缺血**，TARE 靠**輻射**。
- 🧠 **記憶法**：「**E = Embolization 缺血；R(TARE) = Radiation**」；毒性「**TACE > TAE**」（多加化療藥多了毒性）。

> 📌 **重點 (High-Yield)**：
> - 栓塞三式：**TAE / TACE / TARE**；TARE = 輻射 DNA damage，其餘 = 缺血。
> - **TACE 毒性 > TAE**。
> - **肝移植**僅限：無肝外病灶 + 原發已完全切除的多葉肝轉移。

## 🩺 5. 荷爾蒙症候群治療：症候群 → 首選藥對照（核心）

### 📊 症候群 → 首選藥對照表

| 症候群 | 主要病生理 | 🥇 首選藥物 | 補充/二線（原文） |
|---|---|---|---|
| **Carcinoid syndrome** | serotonin (5-HT) 過量 | **長效第一代 SRL（octreotide / lanreotide）** | 短效 octreotide（refractory/危象/圍術期）；**telotristat ethyl**（SRL-refractory diarrhea）；IFN-α（難治）；liver-directed；PRRT |
| **Insulinoma** | insulin 過量 → 低血糖 | **Diazoxide**（+ 飲食、glucose） | SRL（住院試用，慎防加重低血糖）；pasireotide（理論優勢）；glucocorticoid；**PRRT + everolimus**（最成功） |
| **Gastrinoma (ZES)** | gastrin 過量 → 酸過度分泌 | **高劑量 PPI** | H2-blocker；SRL（PPI/H2 難治時）；手術（唯一根治）；PRRT |
| **VIPoma** | VIP 過量 → 水瀉 | **先矯正體液電解質**，再 **SRL** | cytoreductive / 抗增生；PRRT |
| **Glucagonoma** | glucagon 過量 → NME、高血糖 | **SRL**（治 necrolytic migratory erythema） | 飲食/胺基酸支持；口服降糖；PRRT |
| **Somatostatinoma** | somatostatin 過量 | 支持治療為主 | glucose 管理、**cholecystectomy**、胰酶；SRL 偶可改善；PRRT |

### 🧠 記憶法：「**每個症候群的『擋路藥』**」

> **C-I-G-V**（carcinoid / insulinoma / gastrinoma / VIPoma）四大功能症候群首選：
> - **Carcinoid → SRL**（octreotide/lanreotide）＋ **Telotristat** 救瀉不救 flush。
> - **Insulinoma → Diazoxide**（開 K_ATP channel，關 insulin）。
> - **Gastrinoma → PPI**（高劑量擋酸）。
> - **VIPoma → 先補液電解質**再 SRL。
>
> 口訣：「**洗（C）手（SRL）、胰（I）島糖（Diazoxide）、胃（G）泌（PPI）、V 先補水**」。
>
> ⭐ **共通主幹**：幾乎所有功能性 NEN，**SRL + PRRT + cytoreductive** 都能降該荷爾蒙（gastrin/VIP/glucagon/insulin），差別在「**第一線專屬藥**」。

> 📌 **重點 (High-Yield)**：
> - Carcinoid 首選 **SRL**；難治瀉加 **telotristat**（對 flushing 無顯著效果）。
> - Insulinoma 首選 **diazoxide**（機轉：活化 **ATP-sensitive K⁺ channel** 抑制 insulin 分泌）。
> - Gastrinoma 首選 **高劑量 PPI**。
> - VIPoma **先矯正體液電解質**。

## 🩺 6. Carcinoid Syndrome 與 Carcinoid Heart Disease (CHD) 治療詳解

### 💊 慢性腹瀉是「多因性」— 先排除其他原因
| 病因 | 對應治療 |
|---|---|
| 外分泌胰功能不足 (exocrine pancreatic insufficiency) | 胰酶 (pancreatic enzymes) |
| 膽酸吸收障礙 (bile acid sequestration) | **cholestyramine** |
| 細菌過度生長 (bacterial overgrowth) | 抗生素 |
| 腹部術後短腸 (short bowel) | 飲食調整 |

- 嚴重分泌性腹瀉 → 支持措施：避免誘發食物/壓力；止瀉劑 **loperamide、codeine phosphate**，甚至 **opium / morphine**。

### 💊 主軸藥物
- **SRL（最有效）**：carcinoid syndrome **症狀反應率 <u>65%–70%</u>**；常伴隨生化反應（血漿/尿 5-HIAA 下降）。確診後即開始長效第一代 SRL（octreotide / lanreotide）。
- **短效 (immediate-release) octreotide** 適應症：
  - 難治 carcinoid syndrome 的**輔助 (adjunct)**；
  - **carcinoid crisis** 治療；
  - 圍術期 (peri-interventional) **預防 carcinoid crisis**；
  - 第一次長效 octreotide 注射後**至少前 2 週**併用。
  - ⚠️ 已用長效 SRL 的症狀病人，圍術期是否仍需短效 octreotide infusion 仍有爭議，但**歷史研究與指引仍建議施行**。
- **Telotristat ethyl**：serotonin **合成抑制劑**；緩解 **SRL-refractory 腹瀉**，但**對 flushing 無顯著效果**。
- **Interferon-α**：歷史上可緩解症狀，但**療效有限、副作用多** → 僅限難治病例。
- **Liver-directed therapy**：liver-dominant disease 可能非常有效。
- **PRRT（radiolabeled SRL）**：可改善 carcinoid syndrome 症狀。

### ⚠️ 營養併發症（易被忽略的陷阱）
- 嚴重 carcinoid syndrome → tryptophan 代謝轉向 serotonin → **niacin (vitamin B₃) 缺乏 / pellagra（糙皮病）** → 補 **niacin 或 nicotinamide**。
- **SRL 本身**可致**脂溶性維生素（如 vitamin D）與 vitamin B₁₂ 缺乏** → 有 steatorrhea 者須監測並補充。

### 🫀 Carcinoid Heart Disease (CHD)
- CHD 與右心衰竭 → 先以**藥物**治療；必要時行 **tricuspid 與 pulmonary 瓣膜置換**。

> 🧠 **記憶法（carcinoid 腹瀉階梯）**：「**先排雜因 → SRL → telotristat → IFN-α / liver / PRRT**」。
> Telotristat 口訣：「**止瀉不止紅 (diarrhea yes, flush no)**」。

> 🧠 **記憶法（營養）**：「**Carcinoid 偷走色胺酸 → 缺 B₃（pellagra 3D：Dermatitis/Diarrhea/Dementia）**」；「**SRL 偷走脂溶性維生素 + B₁₂**」。

> 📌 **重點 (High-Yield)**：
> - SRL 症狀反應率 <u>65–70%</u>；確診即開長效 SRL。
> - 短效 octreotide → **carcinoid crisis** 與圍術期預防；長效首針後**至少 2 週**併用。
> - **Telotristat** 緩解難治瀉、**對 flush 無效**。
> - 警覺 **niacin (B₃) 缺乏 / pellagra**；SRL 致脂溶性維生素 + B₁₂ 缺乏。
> - CHD 重點瓣膜：**tricuspid + pulmonary**（右心）。

## 🩺 7. Insulinoma 治療詳解

**治療目標：預防嚴重低血糖。**

### 💊 階梯
1. **血糖監測**：給 glucose monitor，規律監測直到日夜血糖正常。
2. **手術**：根治性切除為**首選 (curative for locoregional)**，偶可用於有限且可切除的遠端轉移。
3. **飲食**：少量多餐、長效碳水、必要時夜間進食；嚴重者 NG/ND 管灌；不耐受 → **中央靜脈 parenteral glucose infusion**。
4. **Diazoxide**：常用；抑制 insulin、改善血糖。**機轉：活化胰島細胞 ATP-sensitive K⁺ channel → 抑制 insulin 分泌**。
5. **SRL**：可試用，但 ⚠️ **可能降低反調節荷爾蒙 → 反而加重低血糖** → **務必住院、用短效 SRL 起始**。
   - **pasireotide** 理論上優於 octreotide/lanreotide（其**高血糖副作用**反而有利），有零星臨床支持。
6. **Glucocorticoid**：有效，但所需**高劑量**伴隨顯著不良事件。
7. **Cytoreductive / 抗增生**：降 insulin。**PRRT + everolimus 是最成功的**。
   - ⭐ **everolimus（mTOR 抑制劑）的『高血糖』副作用反而是治療優勢**。

> 🧠 **記憶法（insulinoma 用藥的『副作用即療效』）**：
> - **Diazoxide**：開 K channel = 關 insulin。
> - **Pasireotide / Everolimus**：本來會「升血糖」的副作用，在 insulinoma 變成**好處**。
> 口訣：「**會讓你高血糖的藥，正好治低血糖瘤**」。

> 📌 **重點 (High-Yield)**：
> - 首選 **diazoxide**（機轉 = 活化 **ATP-sensitive K⁺ channel**）。
> - **SRL 可能惡化低血糖** → 住院、短效起始。
> - **PRRT + everolimus** 最成功；everolimus 高血糖副作用是**優勢**。

## 🩺 8. Gastrinoma (ZES) 治療詳解

- gastrinoma **惡性比例 <u>60%–90%</u>**。
- 兩大目標：**控制酸過度分泌** + **腫瘤控制**。
- ⭐ **首要目標 = 降低過量胃酸**（因酸過度分泌導致 morbidity）。

### 💊 治療
1. **高劑量 PPI**：通常可控酸，ZES 相關症狀緩解。
2. **手術**：duodenal/pancreatic gastrinoma 切除是**唯一根治選項** → 應評估 radical resection。
   - ⚠️ **MEN1 病人** duodenal gastrinoma **多發 (multiplicity)** → 是否手術仍有爭議。
3. **SRL**：降 tumoral gastrin 與酸分泌；用於 **PPI/H2-blocker 難治**者，且對**不可切除/轉移**者有抗增生加成。
4. **Cytoreductive / 抗增生（含 PRRT）**：進一步降 gastrin。

> 📌 **重點 (High-Yield)**：
> - gastrinoma 惡性 <u>60–90%</u>；首要 = **高劑量 PPI 控酸**。
> - **手術是唯一根治**；MEN1 因多發病灶手術有爭議。

## 🩺 9. VIPoma 治療詳解

- **第一步 = 矯正體液與電解質缺失 (fluid & electrolyte deficits)**。
- **SRL**：減少腹瀉次數與量 → 協助恢復體液電解質平衡。
- **Cytoreductive / 抗增生（含 PRRT）**：降 VIP 產生。

> 🧠 **記憶法**：「**VIPoma = WDHA（Watery Diarrhea, Hypokalemia, Achlorhydria）→ 先補水補鉀，再 SRL**」。

> 📌 **重點 (High-Yield)**：VIPoma **先補液電解質 → 再 SRL → 再抗增生/PRRT**。

---

## 🩺 10. Glucagonoma 與 Somatostatinoma 治療

### 💊 Glucagonoma
- **Necrolytic migratory erythema (NME)** → 對 **SRL** 與 cytoreductive/抗增生（含 PRRT）有反應。
- 注意**飲食與循環胺基酸**（dietary support）。
- 必要時優化**口服降糖**。
- Cytoreductive/抗增生（含 PRRT）→ 降 glucagon。

### 💊 Somatostatinoma
- 多為**支持治療**：glucose 管理、**cholecystectomy（膽囊切除）**、pancreatic enzymes。
- ⭐ **耐人尋味**：SRL **偶可改善** somatostatinoma 症狀（intriguingly）。
- 證據多為 anecdotal series/reports。

> 🧠 **記憶法**：「**Glucagonoma 看皮膚 (NME) → SRL；Somatostatinoma 三支持：糖、膽囊、胰酶**」。

> 📌 **重點 (High-Yield)**：
> - Glucagonoma：**NME → SRL**；顧胺基酸與血糖。
> - Somatostatinoma：支持為主（glucose / **cholecystectomy** / pancreatic enzymes）；SRL 偶有效。

### 📊 表格解讀
- **Liver embolization** 在「symptoms（0.85）」與整體症狀控制上反應率最高 → 呼應「liver-dominant disease 可非常有效」。
- **Telotristat** 的特色：**5-HIAA 大降 (0.75)** 但 **diarrhea 僅 0.4、flush 無數據** → 與內文「緩解難治瀉、對 flush 無顯著效果」一致（生化降很多，臨床瀉中等）。
- **PRRT**：5-HIAA 反應率看似低 (0.15)，但**症狀/腹瀉/潮紅全達 0.75** → 臨床症狀改善顯著。
- **SRL**：各項 0.45–0.7，與內文「症狀反應 65–70%」吻合（diarrhea 0.65、flush 0.7）。
- ⚠️ **易混淆**：「**生化反應 ≠ 臨床反應**」。Telotristat 生化(5-HIAA)很漂亮但 flush 沒救；PRRT 生化(5-HIAA)平平但症狀全面改善。看 carcinoid 治療要**分開讀『生化欄』與『症狀欄』**。

> 🧠 **記憶法（Fig. 43.9）**：
> - 「**Telo 降數值不退紅**」（5-HIAA↓↓、flush 無效）。
> - 「**PRRT 數值小贏、症狀全贏**」。
> - 「**肝栓塞 症狀王 (0.85)**」。

> 📌 **重點 (High-Yield)**：
> - 控制 carcinoid **症狀**最佳：**liver embolization (0.85)** 與 **PRRT**。
> - **Telotristat**：5-HIAA↓ 明顯、腹瀉中等、**flush 無效**。
> - SRL 全面但中等（呼應症狀反應 65–70%）。
