> 來源：Williams Textbook of Endocrinology, Ch.43 Neuroendocrine Tumors and Disorders — Carcinoid Syndrome 與 Flushing / Diarrhea 鑑別診斷

## 🩺 一、Carcinoid Syndrome 總論

- **NEN 的內分泌相關症狀**來自一種或多種 peptide／amine 的過度分泌：
  - **Eutopic secretion**：分泌物與原發組織相符（典型 secretory syndrome）。
  - **Ectopic secretion（罕見）**：分泌物來自其他解剖部位的 bioactive peptides/hormones。
  - ⚠️ **多數 NEN 其實是 nonfunctioning**（不分泌生物學上有意義量的 active peptide/hormone）。

- **Carcinoid syndrome 三大臨床特徵**：
  - 頻繁的 **watery（secretory）diarrhea**（水樣／分泌性腹瀉）
  - **Flushing**（潮紅）
  - 偶爾合併 **wheezing**（喘鳴）

- **流行病學**：發生率約 <u>2 cases / 100,000 persons</u>。

- **主要分泌產物（可能致病）**：
  - **Serotonin（5-hydroxytryptamine, 5-HT）**
  - **Histamine**
  - **Bradykinins 與 tachykinins**
  - **Kallikrein**
  - **Prostaglandin**

> 🧠 **記憶法（5 大分泌物）**：「**Sero-His-Bra-Ka-Pro**」
> → **Sero**tonin、**His**tamine、**Bra**dykinin/tachykinin、**Ka**llikrein、**Pro**staglandin。

### ⚠️ 為什麼要「繞過肝臟」才會出現症狀？（最重要觀念）

- 這些 hormone/peptide **可被肝臟有效代謝（first-pass metabolism）**。
- 因此症狀通常只在**腫瘤位置在 portal vein 引流系統之外、或繞過 portal drainage** 時才出現。
- **繞過 portal drainage 的例子**：
  - 原發 **BP NEN**（bronchopulmonary，支氣管肺）
  - **Thymic**（胸腺）
  - **Ovarian**（卵巢）
  - 廣泛 **retroperitoneal**（腹膜後）部位
- **最常見的 carcinoid syndrome 原發來源**：**Midgut NEN** ＞ **BP NEN**。
- 在這些腫瘤合併 **liver 和／或 bone metastases** 的病人中，carcinoid syndrome 出現率約 <u>20%–30%</u>。

> 🧠 **記憶法（症狀出現條件）**：要嘛**肝轉移把肝功能吃掉**，要嘛腫瘤在**「不經過肝門靜脈」的位置**（肺、胸腺、卵巢、腹膜後）→ vasoactive 物質才能直接進入體循環。

> 📌 **重點 (High-Yield)**：
> - Carcinoid syndrome = **diarrhea + flushing ± wheezing**，發生率 **2/10萬**。
> - 症狀出現前提：**肝轉移**或**腫瘤繞過 portal vein**（midgut NEN 最常見原發，其次 BP NEN）。
> - 有 liver/bone 轉移者約 **20%–30%** 出現 carcinoid syndrome。

## 🧬 二、Serotonin / 5-HIAA 機制

### Serotonin 的生理作用

- **調節腸道蠕動（motility）與腸道內液體分泌**，並**抑制吸收（absorption）** → 造成 **secretory diarrhea**。
- 參與 **fibrosis（纖維化）**：與許多 carcinoid syndrome 病人的纖維化現象有關。
- **Diarrhea 與 carcinoid heart disease (CHD)** 都被歸因於**全身性 serotonin 過量（systemic serotonin excess）**。

### 🔬 Serotonin 生合成與代謝路徑（Fig. 43.3）

| 步驟 | 受質 → 產物 | 酵素 |
|---|---|---|
| 1 | Tryptophan → 5-Hydroxytryptophan | **Tryptophan hydroxylase** |
| 2 | 5-Hydroxytryptophan → **5-Hydroxytryptamine (serotonin)** | **Aromatic-L-amino acid decarboxylase** |
| 3 | Serotonin → 5-Hydroxyindoleacetaldehyde | **Monoamine oxidase (MAO)** |
| 4 | 5-Hydroxyindoleacetaldehyde → **5-Hydroxyindoleacetic acid (5-HIAA)** | **Aldehyde dehydrogenase** |

**📊 表格解讀**：
- 起點是 **Tryptophan**，終點代謝產物是 **5-HIAA**，**由尿液排出** → 臨床上量 **urine 5-HIAA** 來追蹤 serotonin 分泌。
- 限速第一步由 **Tryptophan hydroxylase** 催化（這也是 telotristat 類藥物的作用點概念基礎）。
- 易混淆：**5-Hydroxytryptophan（中間產物）** vs **5-Hydroxytryptamine（= serotonin 本尊）**，差一個字母但完全不同；**5-HIAA 才是尿液檢驗物**。

> 🧠 **記憶法（路徑）**：「**色胺酸 → 羥色胺酸 → 血清素 → 醛 → 5-HIAA**」
> 口訣：**Trp 先羥化（hydroxylase），再去羧（decarboxylase）變血清素，MAO + 醛去氫酶最後排成 5-HIAA**。

### ⚠️ Tryptophan deficiency 的下游後果

- Tryptophan **大量轉成 serotonin** → 可能造成 **tryptophan deficiency**。
- 後續連鎖：
  - **蛋白質合成下降** → **hypoalbuminemia**
  - **Nicotinic acid（vitamin B₃）deficiency** → **偶爾**出現 **pellagra** 臨床表現。

### ⚠️ Flushing 與 serotonin 的關係（陷阱題）

- **臉部與上半身的 flushing 並非直接由 serotonin 引起！**
- Flushing 很可能由 NEN 及其轉移所**產生、釋放的 vasoactive substances** 媒介。

> 📌 **重點 (High-Yield)**：
> - **Serotonin → diarrhea + 纖維化（CHD）**；**Flushing 不是 serotonin 直接造成**，而是其他 vasoactive 物質。
> - **5-HIAA = serotonin 的尿液代謝產物**（追蹤指標）。
> - Tryptophan 被消耗 → **hypoalbuminemia + B₃ 缺乏（pellagra）**。

## ❤️ 三、Carcinoid Heart Disease (CHD) — Hedinger Syndrome（右心）

- CHD 是 carcinoid syndrome 的**嚴重併發症**，盛行率 <u>20%–50%</u>（在 carcinoid syndrome 病人中）。
- **病理特徵（最常見）**：
  - **Tricuspid valve (TV)** 與 **pulmonary valve (PV)** **纖維化** → 造成 **regurgitation（逆流）+ valve stenosis（狹窄）**。
  - **Endocardial fibrosis**（心內膜纖維化）。
- **為什麼是右心？** Serotonin 經過**肺循環時被代謝清除**，故纖維化主要打擊**右心瓣膜（TV/PV）**。
- **左心受侵犯（約 1/3 病例）的情況**：
  - 合併 **patent foramen ovale (PFO)**（讓 serotonin 繞過肺部代謝）
  - **serotonin-producing BP NEN**（serotonin 直接進入左心系統）
  - 影響 **aortic 與 mitral valves**。

- **臨床表現**：
  - 可**無症狀**。
  - 進展後：**漸進性 exertional dyspnea、fatigue**，以及**右心衰竭**徵象 → **頸靜脈壓升高、hepatomegaly、peripheral edema**。

- **篩檢與預後**：
  - ⚠️ **所有 carcinoid syndrome 病人都應篩檢 CHD**。
  - 為**主要預後因子**：若未治療，**3 年存活率降低 <u>30%</u>**。
  - 處理需要在專責中心採**多專科團隊（multidisciplinary）**。

| 比較項目 | 右心 CHD（典型 Hedinger） | 左心 CHD（約 1/3 病例） |
|---|---|---|
| 受累瓣膜 | **Tricuspid + Pulmonary** | **Aortic + Mitral** |
| 機轉 | serotonin 直接作用右心 | **PFO** 或 **BP NEN**（繞過肺代謝） |
| 病灶 | 瓣膜纖維化、逆流＋狹窄、心內膜纖維化 | 同類纖維化但在左側瓣膜 |

**📊 表格解讀**：CHD 預設打**右心**，因為 serotonin 在肺被清掉；只有當 serotonin **繞過肺**（PFO 右→左分流，或肺本身就是分泌源 BP NEN）才波及左心。考試常考「為何 carcinoid 主要影響右心瓣膜」。

> 🧠 **記憶法**：「**TRiPs to the Right heart**」→ **TR**icuspid + **P**ulmonary 在右側（**R**ight）。左心被波及一定要想到 **PFO 或 BP NEN**。

### ⚡ Carcinoid Crisis（急性致命表現）

- **碳水危機 = uncontrolled carcinoid syndrome 的急性致命併發症**。
- **臨床**：**突發 hemodynamic instability**，常伴隨嚴重 flushing 與 **cardiovascular collapse**；未治療**可致命**。
- **誘發機轉**：**生理壓力（physiologic stress）或直接腫瘤操作（tumor manipulation）** → 誘發 **vasoactive peptides/hormones 大量釋放**，可能由 **catecholamines** 媒介。
- ⚠️ 目前**仍不確定** IV somatostatin receptor ligands (SRLs，如 **octreotide**) 是否能**預防或逆轉**這些危機。

> 🧠 **記憶法（碳水危機誘發/觸發因子）**：「**手術操作、麻醉、生理壓力 → 大量釋放**」
> → **觸發 = STRESS**：**S**urgery/**S**tress、**T**umor manipulation/biopsy、anesthesia 等 → catecholamine 驅動 vasoactive 物質爆發。

### 🕸️ Mesenteric Fibrosis（腸繫膜纖維化）

- 纖維化也可發生在**原發 NEN 或腸繫膜轉移灶**周圍，被廣泛纖維化反應包繞。
- 可導致 **intestinal obstruction、edema、ischemia** → **腹痛、cachexia**，常需**手術**。
- 在**轉移性 midgut NEN** 病人中觀察到 **>60%** 有 mesenteric fibrosis。

> 📌 **重點 (High-Yield)**：
> - CHD 盛行率 **20%–50%**；典型侵犯 **TV + PV（右心）**，未治療使 **3 年存活率↓30%** → **所有病人都要篩 CHD**。
> - 左心受累（約 **1/3**）→ 想 **PFO** 或 **serotonin-producing BP NEN**。
> - **Carcinoid crisis** 由 **stress / tumor manipulation** 觸發（catecholamine 媒介）；octreotide 能否逆轉**尚不明確**。
> - **Mesenteric fibrosis >60%**（轉移性 midgut NEN）。

## 🔥 四、BOX 43.2 — Flushing 鑑別診斷（Wet vs Dry）

> **定義**：Flushing = 皮膚發紅的陣發性發作（episodic redness），伴隨臉、頸、較少見的上軀幹與腹部的**溫熱或灼熱感**。
> **Wet / Dry** 的差別在於**有無增加出汗（perspiration）**。

| 類型 | 是否出汗 | 鑑別診斷 |
|---|---|---|
| **Wet Flushing**（有汗 💦） | 增加出汗 | • Hypogonadism (menopause)<br>• Pheochromocytoma / paraganglioma<br>• Neurologic disorder<br>• Medication |
| **Dry Flushing**（無汗 🌵） | 無增加出汗 | • **Emotions**（Medication、Food、Capsaicin、Toxins、Food additives）<br>• Alcoholism<br>• **Carcinoid syndrome**<br>• Pheochromocytoma / paraganglioma<br>• VIPoma<br>• Medullary thyroid carcinoma<br>• Mastocytosis<br>• Serotonin syndrome<br>• Anaphylaxis |

*Modified from Huguet I, Grossman A. Eur J Endocrinol. 2017;177(5):R219–R229.*

**📊 表格解讀**：
- **Carcinoid syndrome 屬於 Dry flushing**（這是高頻考點！）。
- ⚠️ **Pheochromocytoma / paraganglioma 同時出現在 Wet 與 Dry 兩欄** → 是 flushing 鑑別的「兩棲動物」，不能用乾濕單獨排除。
- **Menopause（hypogonadism）是 Wet flushing** 的代表（停經潮熱常伴盜汗）。

> 🧠 **記憶法**：
> - **Wet = 出汗 →「更年期＋pheo＋神經＋藥物」**（記「**汗淋淋的更年期太太被嚇到流汗**」）。
> - **Dry = 不出汗 →「CARVM-MSA」**：**C**arcinoid、**A**lcohol、**R**(pheo)、**V**IPoma、**M**TC、**M**astocytosis、**S**erotonin syndrome、**A**naphylaxis（+ emotions/food）。
> - 一句話：**carcinoid 的 flushing 是「乾」的，更年期是「濕」的**。

> 📌 **重點 (High-Yield)**：
> - **Carcinoid = Dry flushing**；**Menopause = Wet flushing**。
> - **Pheo/PGL 橫跨 Wet 與 Dry 兩類**。
> - Wet/Dry 的關鍵分界 = **有無出汗**。

## 💧 五、BOX 43.1 — 轉移性 NEN 病人的 Diarrhea 鑑別診斷（四型）

| 腹瀉類型 | NEN-Related 病因 | Not NEN-Related / 其他病因 |
|---|---|---|
| **① Secretory（分泌性）** | • **Carcinoid syndrome**<br>• Gastrinoma<br>• VIPoma<br>• CCKoma<br>• Medullary thyroid carcinoma | • Bile acid diarrhea（膽酸性腹瀉） |
| **② Osmotic（滲透性）** | —（原文未列） | —（原文未列） |
| **③ Inflammatory（發炎性）** | —（原文未列） | —（原文未列） |
| **④ Maldigestive（消化不良性）** | • **Somatostatinoma**<br>• Pancreatic exocrine insufficiency：<br>　– Postsurgery (postpancreatectomy)<br>　– Medication（**somatostatin receptor ligands, SRL**） | **Maldigestive/Malabsorptive**：<br>• Short bowel<br>• Bacterial overgrowth<br>• Other（irritable bowel、lactose intolerance、celiac disease、laxatives、hyperthyroidism、enteropathies、common variable immunodeficiency） |

*Modified from Eads JR, et al. Pancreas. 2020;49(9):1123–1130.*

**📊 表格解讀**：
- **Carcinoid syndrome 的腹瀉屬 Secretory（分泌性）** → 與 serotonin 促分泌、抑吸收一致（呼應上文機制）。
- **Secretory NEN 群（必背）**：Carcinoid、**Gastrinoma**、**VIPoma**、**CCKoma**、**MTC**。
- ⚠️ **治療陷阱**：**SRL（如 octreotide）本身可導致 maldigestive diarrhea**（透過 pancreatic exocrine insufficiency）→ 用來治療 carcinoid 的藥，反而可能造成另一型腹瀉。
- ⚠️ **Somatostatinoma 造成的是 maldigestive 型**（抑制胰酶分泌），不是 secretory，常被混淆。
- 原文中 Osmotic 與 Inflammatory 兩型**僅列標題、未列具體病因**（如實保留，不外推）。

> 🧠 **記憶法（Secretory NEN）**：「**Carcinoid 帶著 G-V-C-M 一起拼命分泌**」
> → **G**astrinoma、**V**IPoma、**C**CKoma、**M**TC。
> 對照記憶：**「分泌型 → 腫瘤分泌激素逼腸子排水」**；**「消化不良型 → 胰酶不夠（somatostatinoma / 術後 / SRL 藥物）」**。

> 📌 **重點 (High-Yield)**：
> - NEN 腹瀉四型：**Secretory / Osmotic / Inflammatory / Maldigestive**。
> - **Carcinoid = Secretory**；同型還有 **Gastrinoma、VIPoma、CCKoma、MTC**（＋非NEN的 bile acid diarrhea）。
> - **Somatostatinoma 與 SRL 藥物 → Maldigestive 型**（胰外分泌不足）。

## 🧠 六、整章記憶總表

| 主題 | 一句話必記 |
|---|---|
| 三聯徵 | Diarrhea（secretory）+ Flushing（dry）± Wheezing |
| 症狀前提 | 肝轉移 or 腫瘤繞過 portal vein（肺/胸腺/卵巢/腹膜後） |
| 最常原發 | Midgut NEN ＞ BP NEN |
| Serotonin 作用 | Diarrhea + 纖維化（CHD）；**flushing 不是它直接造成** |
| 尿液指標 | **5-HIAA** |
| CHD | 右心 TV+PV（Hedinger）；20–50%；3年存活↓30%；全部要篩檢 |
| 左心 CHD | 想 **PFO** 或 **BP NEN** |
| Crisis | stress / tumor manipulation 觸發（catecholamine 媒介） |
| Flushing | Carcinoid = **Dry**；Menopause = **Wet**；pheo 兩者皆有 |
| Diarrhea | Carcinoid = **Secretory**；Somatostatinoma/SRL = **Maldigestive** |

> 📌 **重點 (High-Yield)**：
> - 把握**三條主軸**：①症狀需繞過/吃掉肝臟；②serotonin → 腹瀉＋右心纖維化（flushing 另有元凶）；③鑑別診斷記住「**carcinoid = dry flushing + secretory diarrhea**」。
