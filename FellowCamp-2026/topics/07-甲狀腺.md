# 甲狀腺（Thyroid）— 孕期甲狀腺功能異常 高頻考點摘要

> 來源：FellowCamp 2026 · Thyroid（孕期甲狀腺 dysfunction 之診斷與治療）· 黃君睿 醫師

> ℹ️ **本講題範圍說明**：本場逐字稿內容**聚焦於「孕期（pregnancy）甲狀腺功能異常」**——涵蓋孕期生理變化、妊娠孕吐 vs Graves、孕期 Graves 治療與停藥、孕期甲低與 LT4 調整、subclinical hypothyroidism、euthyroid TPOAb positive。一般甲亢/甲低、thyroiditis、結節與癌、甲狀腺風暴等主題**不在本場錄音內**，故本摘要不杜撰補充，僅忠實重組講者內容。

## 一、孕期甲狀腺生理變化 🧪（⚠️ 老師明示：幾乎每年必考，「最重要的一張圖」）

講者強調這張生理變化圖**「過去好像每一年都有考」**，一定要會。

### 兩個階段的兩個機轉

| 時期 | 主要荷爾蒙驅動 | 機轉 | 抽血表現 |
|---|---|---|---|
| **First trimester（早期）** | **HCG ↑** | HCG 與 TSH 的 **α-subunit 為 identical**，結合並刺激 **TSH receptor** | **free T4 ↑ → 回饋 suppress TSH（TSH ↓）**；此為**正常生理**，非疾病 |
| **2nd/3rd trimester（中後期）** | **Estrogen ↑ → TBG ↑** | Thyroxine-binding globulin 上升 | **total T4、total T3 ↑**（free 趨近 non-pregnant） |

🧠 **記憶法**：「**早 HCG 壓 TSH（free 升）；晚 estrogen 升 TBG（total 升）**」。

### 必背 cutoff（Reference interval）
- 需用 **trimester-specific（且 assay/population-specific）reference interval** 判讀。
- 若無 local data，ATA 建議：non-pregnant TSH range <u>0.5–4.5</u>，early gestation 會壓低至約 <u>0.1–4</u>，仍屬正常生理。
- 講者荣总 data：first trimester free T4 的 <u>97.5 percentile 可達約 2.4</u>，故 <u>≤2.5 內</u>都蠻常見。
- Total T4：自 7–16 週逐漸上升，16 週後可達原上限的 <u>約 1.5 倍</u>。⚠️（逐字稿口誤「增加 5%/1.5%」，依生理應為「上限約 1.5 倍」，回查 ATA。）

> 📌 **High-Yield（必背）**：早期 **free T4↑ + TSH↓ = 正常生理（HCG 刺激）**，勿誤判為甲亢；中後期 **total T4/T3↑ = TBG 上升**。判讀一定要用 **trimester-specific reference**。

## 二、Gestational（妊娠）甲亢 vs Graves 鑑別 🩺

**Case 1**：27 歲，GA 9+1 週，惡心嘔吐 3 週、體重↓4 kg，無甲狀腺病史；free T4↑、TSH↓（subclinical/overt thyrotoxicosis）。

### ⚠️ 秒殺題：第一步「不是」開 anti-thyroid drug！
- 正確處置：**supportive care**（止吐、矯正 electrolyte、dehydration）+ **測 TSH receptor antibody / TSI** 做鑑別。
- 講者酸點：新陳代謝科**不該像家醫/一般科看到抗體就開藥**。

| 項目 | Gestational thyrotoxicosis（妊娠孕吐） | Graves' disease |
|---|---|---|
| 機轉 | **HCG 刺激**（暫時性） | autoimmune |
| **TSI / TRAb** | **Negative** | **Positive**（極少數 TSI-neg Graves，罕見） |
| 處置 | supportive care，**不需 ATD** | 需評估治療 |
| 病程 | **約孕 14–18 週自然緩解** | 持續、可影響胎兒至產後 |

🧠 **Hyperemesis gravidarum（妊娠劇吐）**：HCG 高 → 暫時性甲亢，**給 supportive care 即可**。

> 📌 **High-Yield（必背）**：早孕 thyrotoxicosis 第一步是 **supportive care + 測 TSI/TRAb**，不是馬上 ATD；妊娠孕吐**約 14–18 週自然緩解**。

## 三、孕期 Graves' disease 的治療與停藥 💊（高頻考點）

### 核心生理 🧪
- **Fetal thyroid gland** 在 first trimester 才開始 develop；早期完全靠媽媽供應 T4，中後期仍部分靠媽媽，**出生後才完全自給**。
- 故孕期 thyroid hormone demand↑：母體甲狀腺可**大 10–15%**，hormone 製造可**多約 50%**。
- ⚠️ **過胎盤考點**：**唯一「不」過胎盘的是 TSH**。其餘（thyroid hormone、thyroid antibody、碘、PTU、MMI/CMZ、甚至 beta-blocker）**全部會過胎盤** → 用越少藥越好。

🧠 **口訣**：「**只有 TSH 不過胎盤，其他通通過**」。

### 用藥選擇 ⚠️
- **First trimester：PTU is preferred over MMI/carbimazole**。
- 原因：**PTU 與 MMI/CMZ 都有致畸胎性**，但 **MMI/CMZ 致畸胎的 severity 與 frequency 較高**（非 PTU 無致畸胎）。
- **ATD monotherapy + lowest possible dose**。
- ⚠️ **孕期禁忌：不可 block-and-replace**（不可一邊 ATD 一邊 LT4），因 **ATD 過胎盤 > LT4**，會造成胎兒甲低。
- 治療 **target：free T4 控制在 reference 上限、甚至 slightly above**（對胎兒較好；胎兒對 ATD 很 sensitive，媽媽壓太「正常」= 胎兒相對甲低）。講者：free T4 控在 **1.7–2.0 以下都可接受**，**不需壓太低**。

### 停藥時機（新 ATA criteria）✅
治療 Graves 孕婦，符合以下可**直接停藥**以避開 ATD 致畸胎：
- 已治療 **>6 個月**
- 現 **TSH 已正常**
- ATD 劑量低：**MMI <10 mg/day** 或 **PTU <200 mg/day**
- **無眼病**（不需特別壓抗體）
- **antibody titer < 3 倍 upper limit**

🧠 免疫 tolerance：胎兒一半外來抗原 → 孕期免疫反應較弱，抗體後期常自然轉好；**故多數此類病人可選擇停藥**。
⚠️ **Postpartum rebound**：產後抗體可能突然惡化 → 衛教「坐完月子馬上回診測 thyroid function，可能需重啟 ATD」。

### 胎兒/新生兒監測 🩺（易混陷阱）
- Graves（即使已 use I-131/手術）**first trimester 仍要 check TRAb/TSI**，因 TSI 過胎盤可致胎兒甲亢。
- 高 titer 時需 **fetal monitor**：看 **fetal goiter、fetal tachycardia**。

#### ⚠️ 易混淆秒殺題：新生兒 free T4↓ + TSH 破百 ≠ 先天性甲低
- 若媽媽孕期持續用 ATD：ATD 過胎盤 → **新生兒暫時性甲低**（data 長得跟 congenital hypothyroidism 一模一樣）。
- **鑑別關鍵**：ATD 約**產後 3–5 天代謝掉**，過幾天 retest 會自行恢復正常 → 非真正 congenital hypothyroidism。
- 反向（新生兒 free T4↑、TSH↓ = neonatal thyrotoxicosis）判斷：
  - **孕 1–3 個月內**亢進 → 多為**媽媽 TSI 過胎盤**所致。
  - **3 個月後**仍亢進（母抗體應已代謝）→ 才考慮**胎兒自身**先天性亢進。

> 📌 **High-Yield（必背）**：① 只有 TSH 不過胎盤；② 1st trimester **PTU 優先**；③ **禁 block-and-replace**；④ free T4 壓在上限即可；⑤ 停藥四要件（>6 月、TSH 正常、MMI<10/PTU<200、抗體<3×ULN、無眼病）；⑥ 產後 rebound；⑦ 新生兒 TSH 破百先想「媽媽 ATD」，3–5 天後 retest。

## 四、孕期甲狀腺功能低下與 LT4 調整 💊

**Case 3**：nodular goiter 行 total thyroidectomy，長期 LT4 125 mcg，TSH 偏高（如 4.0），first trimester。

### LT4 劑量調整原則
- **>50% 病人孕期需增加 LT4 劑量**。
- **無殘存組織者（total thyroidectomy、RAI 後）增加幅度更大**；有殘存（如 autoimmune thyroiditis）較少。
- 增加幅度約 **20–50%**（講者另舉「一週多 2 顆 ≈ +29%」=平日 1 顆、週六日 2 顆，一週共 9 顆）。
- **產後立即回到孕前 maintenance dose**。

### TSH target 與監測
- 孕期 **TSH target：lower half of trimester-specific reference range**（因 LT4 過胎盤非 100%，怕胎兒相對不足）。
- 監測：孕期 **每 4–6 週**抽血。
  - 每 **4 週**抽 → 可抓到 <u>92%</u> 需調量者。
  - 每 **6 週**抽 → 可抓到 <u>73%</u> 需調量者。

> 📌 **High-Yield（必背）**：已知甲低孕婦 LT4 **增 20–50%**（無組織者更多），target = trimester reference **下半段**；**4 週抽抓 92% / 6 週抓 73%**；產後回孕前劑量。

## 五、Subclinical Hypothyroidism（孕期）— Guideline 大改變 📊（⚠️ 最新考點）

**Case 4**：GA 15+4 週（2nd trimester），free T4 正常、TSH 偏高 = subclinical hypothyroidism（SCH）。

### Guideline 演變（10 年大轉向）
| 年份/學會 | 立場 |
|---|---|
| **2015 ASRM**（生殖醫學會） | SCH 增 miscarriage、影響胎兒 IQ → **TSH >4 即給 LT4** |
| **2017 ATA** | SCH + **TPOAb positive** → 及早給 LT4；認為 LT4 安全 |
| **2024 ASRM**（大改） | SCH 與 miscarriage、obstetric outcome **無顯著相關**；3 個 RCT 顯示 LT4 **對胎兒 IQ 無正面幫助**；開始警示**過度治療有害** |
| **2026 ATA**（即將發表，已 accept） | 強調 **retesting**；以 **trimester** 分層治療 |

### 三大 LT4 trials（SCH → neonatal outcome）📊
- **NIH trial、CATCH trial → negative**（無 benefit）。
- 唯一 **positive** 者 = **較早給藥**（median 約 **11 週、first trimester** 即開始）→ 暗示「太晚給，胎兒腦神經已發育，看不到差別」。

### 2026 ATA 新治療邏輯（以 trimester 分層）⚠️
- **First trimester（關鍵期）**：
  - **TSH ≥10 → 給 LT4**（同 non-pregnant：SCH 也是 TSH >10 才治療）。
  - **TSH 4–10 → 可 consider**（非絕對）。
- **2nd/3rd trimester**：已過胎兒甲狀腺神經發育 critical period →
  - **TSH 4–10 不建議治療**，比照一般人（**TSH >10 才治療**）。
- ⚠️ **過度治療害處**：LT4 過量可致心悸，並**增加小朋友 ADHD 風險**。

### Thyroid abnormality 的「暫時性」(retest 證據) 📊
- **Chinese cohort**（9–14 週 vs 32–36 週）：SCH concordance 低，**只約 1/4** 後期仍為 SCH。
- **Danish cohort**（7–8 週 vs 11–12 週）：even overt hypo **僅 45% persistent**；SCH 約 40 餘 %。
- 結論：**孕期 thyroid abnormality 常為暫時性，應 retest**。

🩺 **Case 4 處置（2026）**：2nd trimester SCH → **reassure + repeat thyroid function**（再測可能已非 SCH）。

> 📌 **High-Yield（必背）**：2026 新觀念——**SCH 沒以前認為的可怕，先 retest**。治療看 trimester：**1st trimester TSH≥10 給藥、4–10 consider**；**2nd/3rd trimester 比照非孕（>10 才治）**。過度 LT4 → 兒童 ADHD 風險↑。

## 六、Euthyroid + Thyroid Autoimmunity（TPOAb positive）🩺

**Case 5**：GA 10+2 週，無症狀，prenatal screening，**甲功正常、TPOAb positive**。

### ⚠️ 強 evidence 秒殺題：euthyroid TPOAb+ **不給 LT4**
- **三個大型 RCT（2017、2019、2022）一致：euthyroid + TPOAb+ 孕婦給 LT4 無任何 benefit**。
- 正確答案：**repeat thyroid function**（追蹤是否進展為 SCH / overt hypo，即 Hashimoto 病人）。
- 若進展為 **overt hypothyroidism → 才建議治療**。
- ⚠️ 注意：有風免科醫師用 **plaquenil (hydroxychloroquine)** for thyroid autoimmunity，但**無 RCT 支持作為標準**處置。

> 📌 **High-Yield（必背）**：**Euthyroid + TPOAb positive 孕婦「不給」LT4**（3 RCT 證實無益），只 repeat TFT；惡化為 overt hypo 才治。

## 七、全章秒殺整理（考前快掃）🧠

| 情境 | 一句話處置 |
|---|---|
| 早孕 free T4↑ TSH↓ + 嘔吐體重↓ | supportive care + 測 TSI（鑑別妊娠孕吐 vs Graves），14–18 週緩解 |
| 孕期 Graves，低量 ATD、治療>6 月、抗體<3×ULN | **可停藥** |
| 必須用 ATD，1st trimester | **PTU 優先**，free T4 控上限，**禁 block-and-replace** |
| 已知甲低（術後/RAI）懷孕 | LT4 **增 20–50%**，4–6 週監測，產後回孕前量 |
| 1st trimester SCH，TSH≥10 | 給 LT4（4–10 consider） |
| 2nd/3rd trimester SCH，TSH 4–10 | **不治療**，比照非孕（>10 才治） |
| Euthyroid + TPOAb+ | **不給 LT4**，repeat TFT |
| 新生兒 free T4↓ TSH 破百 + 母用 ATD | 暫時性，**3–5 天後 retest**，非 congenital hypo |

🧠 **跨章記憶**：
- **過胎盤口訣**：「TSH 不過，其他全過」。
- **生理口訣**：「早 HCG 壓 TSH、晚 estrogen 升 TBG」。
- **「不要急著開藥」貫穿全場**：妊娠孕吐、euthyroid TPOAb+、2nd/3rd trimester SCH **都先 supportive / retest**。
