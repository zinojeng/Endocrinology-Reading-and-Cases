# 性腺（Gonad）— 女性 HPG 軸與生殖荷爾蒙考試摘要
> 來源：FellowCamp 2026 · 生殖荷爾蒙 / 性腺 · 王聖和 醫師

> ⚠️ 本講以**女性 HPG 軸生理機轉**為主軸，用 4 個 case「逆推月經週期分期」。核心考點是：**用 FSH / LH 的相對高低 + Estradiol 數值，反推病人卡在週期哪一期、被什麼機制壓抑**。男性低下、性早熟/延遲、不孕、SDD 等本講未深入，請另回 Williams 補齊。

---

## 一、女性 HPG 軸與月經週期生理 🧪

### 1. 基本軸線
- 下視丘分泌 **GnRH** → 刺激腦下垂體製造 **FSH + LH** → 刺激卵巢製造 **estradiol（E2）+ progesterone（P4）** → E2/P4 回饋抑制。
- ⚠️ **一個 GnRH 要控制兩個荷爾蒙（FSH、LH），且各有節律 → 必非單一機制**。

### 2. 🧠 核心記憶法：GnRH pulse 頻率決定 FSH vs LH（恆河猴實驗）
> **「快 LH、慢 FSH」**（pulse frequency 快 → 偏 LH；慢 → 偏 FSH）

| GnRH pulse 頻率 | 腦下垂體偏向 | 結果 |
|---|---|---|
| **快**（每小時一次）| 偏分泌 **LH** | LH↑、FSH↓ |
| **慢** | 偏分泌 **FSH** | FSH↑、LH↓ |

- 臨床逆推：**抽血 FSH > LH → GnRH pulse 偏慢**（被某機制壓抑）；**LH > FSH → pulse 偏快**。

### 3. Inhibin（抑制素）— 今日主角之一
- Inhibin / Activin 同一 family，結構相似，**共用 α subunit**。
- **Inhibin 回饋抑制腦下垂體 FSH**。

| | 來源細胞 | 刺激 | 出現時機 |
|---|---|---|---|
| **Inhibin A** | dominant follicle + 黃體 | **LH** dominant | 排卵前後、**黃體期** |
| **Inhibin B** | granulosa cell | **FSH** | 排卵前、**濾泡期** |

- ⚠️ Inhibin A+B「手牽手」一路把 FSH 壓低，**只在 late luteal phase + early follicular phase 開一個小空窗**讓 FSH 彈起來 → 推測為**節省 follicle**（避免一直 recruit 新 follicle 被浪費）。

### 4. 卵泡與細胞 🧪
- **Tertiary follicle = antral follicle**（有腔/有「倉庫」）→ **生育期 E2 主力來源**。
- **Granulosa cell**（密集，內層）：受 **FSH** → 製造 **Inhibin B、AMH**。
- **Theca cell**（外層鬆散，來自 ovarian stroma）：受 **LH** → 製造 **androstenedione**（弱效雄性素）。

### 5. 月經週期逐期（從黃體中期講起）
1. **黃體中期**：E2、P4 達 plateau（尤其 P4 高）→ 回饋抑制下視丘 → **GnRH pulse 變慢**。
2. **黃體晚期 / 濾泡早期**：pulse 慢 → 偏 FSH → **FSH > LH**（FSH 升、LH 降）；FSH 升用來 **recruit 一群 secondary follicle → 變 antral follicle**。
3. **濾泡中/晚期**：失去 P4/E2 抑制 → pulse 變快 → 偏 LH（LH 升、FSH 降）；antral follicle 累積 E2。
4. **LH surge**：當 <u>E2 升到 300–500 pg/mL 並維持 > 36 小時</u> → **E2 對下視丘由負回饋轉為正回饋** → pulse 更快、amplitude 更高 → **FSH/LH surge → 排卵**。
5. **排卵後**：→ 黃體；高 LH 維持黃體 → 黃體中期再次 E2/P4 plateau。
6. LH 不足 → 黃體崩潰；E2/P4 不足維持子宮內膜 → **月經**。

> 🧠 ⚠️ **張清忠教授名言（老師強調必考印象）**：
> 「**正常育齡期女性抽血，LH 幾乎都 > FSH**」——<u>唯二例外：early follicular phase 與 late luteal phase（此時 FSH > LH，所謂 FSH/LH 倒掛）</u>。
> ⚠️ 講者明示：網路上多數 FSH/LH 週期圖（含 AI 生成圖）畫成「FSH 全程 > LH」**都是錯的**。

> 📌 **High-Yield（必背）**：
> - **快 LH、慢 FSH**（pulse 頻率決定 FSH/LH）。
> - **LH surge 觸發條件**：<u>E2 300–500 pg/mL 維持 >36hr → 負轉正回饋</u>。
> - **正常育齡女性 LH > FSH**，例外 = early follicular + late luteal（FSH>LH）。
> - Inhibin B = granulosa（FSH 刺激，濾泡期）；Inhibin A = follicle+黃體（LH 刺激，黃體期）。

## 二、逆推分期邏輯（秒殺題核心）🩺

⚠️ **單看 FSH/LH/E2 絕對值無法分期**（reference range 大量 overlap，例：FSH=5 可橫跨濾泡/排卵/黃體期）。要靠「相對關係」逆推：

- **FSH > LH** → pulse 慢 → 只可能在 **early follicular** 或 **late luteal**（或被病理壓抑）。
- **LH > FSH** → pulse 快 → 排除 early follicular / late luteal。
- **E2 很高（300–500 並升）** → 接近排卵（late follicular）。
- **E2 中等（如 ~44 pmol/L）且 LH>FSH** → 多半 **mid-follicular**。

## 三、四個 Case（機轉逆推）🩺

### Case 1：36 歲，2 年無月經，厭食/暴食催吐，5 年內 42→28 kg
- Lab：FSH **7.32** > LH **1.43**，E2 22.9（偏低但仍足以維持性徵）。
- 邏輯：**FSH > LH → pulse 慢 → 有東西壓抑 GnRH neuron**。
- 診斷：**Anorexia nervosa → 下視丘性不排卵**。
- 🧪 機轉：能量/脂肪不足 → **leptin↓ → 回饋抑制 GnRH neuron → pulse 變慢 → HPG 軸關閉**。
- 文獻名稱：**functional / hypogonadotropic hypogonadism（下視丘性）**。
  - ⚠️ 講者口誤把名稱說成「hypergonadotropic」，**正確為 hypogonadotropic hypogonadism**（中樞性、FSH/LH 低或不適當正常）。
- DDx：anorexia nervosa、女性減重、運動員（芭蕾、田徑）。
- ⚠️ **易混陷阱**：女性健美/舉重選手自打 **testosterone（可達 1500 ng/dL，男性高值）** → 是**外源雄性素直接壓抑 HPG 軸**，**不走 leptin 路徑**。

### Case 2：33 歲，G1P1，停止哺乳 >1 年仍無月經
- Lab：Prolactin **221 ng/mL**↑↑；MRI 0.8 cm nodule → **Prolactinoma**。
- FSH **3.8** > LH **0.82**，E2 **< 5**（測不到）。
- 邏輯：FSH > LH → pulse 慢（frequency 被壓）；**LH 被壓到 0.8 → amplitude（振幅）也被壓** → 絕對值全低。
- 🧪 機轉：<u>Prolactin 達 **50–70 ng/mL** 即開始壓抑 GnRH neuron → pulse 變慢 → 偏 FSH</u>。治療後 PRL↓ → pulse 回快 → LH > FSH。

> 📌 **High-Yield**：高泌乳素 → 壓 GnRH pulse；**PRL ~50–70 ng/mL 為開始抑制門檻**；表現為 FSH>LH 且絕對值偏低。

### Case 3：31 歲，10 年月經不規則、曾診斷 PCOS、半年無月經、肥胖、echo multiple ovarian cysts ⭐重點
- Lab：FSH **4.3** < LH **6.48**，E2 **44.2 pmol/L**。
- 逆推分期：
  - **LH > FSH** → 排除 early follicular、late luteal。
  - E2 44.2 → 只在濾泡早/中期出現（晚期會升到 300–500；剛排卵 E2 仍高排除 luteal early；黃體中期有 part 2 排除）；黃體晚期雖也會落到此值，但黃體晚期 FSH/LH 已倒掛而她沒倒掛。
  - ⟹ **卡在 mid-follicular phase**（上去下不來、無法排卵、卡半年）。

#### PCOS 機轉鏈 🧪
- 卡在濾泡中期 → 已 recruit 一群 antral follicle 但無法排卵 → **超音波密密麻麻小水泡 = 一堆 antral follicle**。
  - ⚠️ 影像：<u>水泡數量可 **> 12 顆**，每顆 **< 1 cm**</u>；正常卵巢為大小不一、dominant follicle 可達 3 cm。
  - ⚠️ 特徵：**ovarian stroma 較緻密、粗壯**（hyperthecosis）。
- 一堆 granulosa cell（FSH 刺激）→ **Inhibin B↑、AMH↑**（比同齡高）。
- **高 insulin + 高 LH** → insulin 的 **mitogenesis pathway**（非 PI3K）維持 theca cell 不散掉 → 聚集 = **hyperthecosis** → 粗壯 stroma。
- 大量 theca cell（LH 刺激）→ **androstenedione↑ → testosterone↑** → 多毛、痤瘡。
- 惡性循環：androgen↑ → central obesity → insulin resistance↑ → hyperinsulinemia↑ → 再推 PCOS。

#### PCOS 診斷（Rotterdam，3 取 2）📊
| Criteria | 對應本 case |
|---|---|
| ① 寡排卵 / 無排卵 | 卡 mid-follicular，無法進展排卵 ✔ |
| ② 高雄性素（臨床或抽血 testosterone↑） | testosterone 偏高 ✔ |
| ③ 卵巢超音波多囊（多 cysts）**或** AMH 偏高 | echo multiple cysts / AMH↑ ✔ |

- ⚠️ **AMH 可替代超音波**作為「多囊」證據（granulosa 多 → AMH 高）。
- 🧪 進階：三個值得研究的 receptor 與藥物開發相關；其中 **AMH 過高會直接刺激 GnRH neuron → pulse↑**，理論上抑制之或可打破 PCOS 惡性循環。

> 📌 **High-Yield**：PCOS 超音波 <u>>12 顆、每顆 <1 cm</u>；高 testosterone 來自 **theca cell androstenedione**；高 insulin 經 **mitogenesis pathway** 維持 theca → **hyperthecosis**；AMH↑、Inhibin B↑；診斷 **3 取 2**，AMH 可代替超音波。

### Case 4：61 歲，停經 8–10 年（更年期）
- Lab：E2 **< 5**、P4 **0.31**（皆極低）。
- 預期：E2/P4 低 → 回饋抑制少 → pulse 快、amplitude 高 → 應 **LH > FSH 且絕對值高**。
- ⚠️ **實際卻 FSH > LH（約 2 倍）→ 看似矛盾**！補 E2+P4 後 amplitude 被壓低（絕對值降），**但 FSH:LH 比例仍 ~2:1 不變**。
- 🧪 解答：**follicle 存量耗竭 → Inhibin A/B↓ → 失去對 FSH 的壓抑 → FSH 彈高 → FSH/LH 倒掛**。
- ⟹ **停經期 FSH 升高的根本原因 = Inhibin 下降**（不只是 E2 低）。

> 📌 **High-Yield**：更年期 **FSH↑↑（FSH>LH 倒掛）的關鍵 = Inhibin（B 為主）下降**，補 E2/P4 改不了 FSH/LH 比例。

## 四、總整理表（必背速記）📊

| Case | 病因 | FSH vs LH | 關鍵機轉 | 必背數字 |
|---|---|---|---|---|
| 厭食 | functional hypogonadotropic hypogonadism | FSH>LH（pulse 慢） | **leptin↓** 壓 GnRH | 體重 42→28 kg |
| Prolactinoma | 高泌乳素 | FSH>LH，絕對值極低 | PRL 壓 GnRH（freq+amp） | PRL **50–70** 起壓 |
| PCOS | hyperinsulinemia + 高 LH | **LH>FSH** | hyperthecosis、androgen↑ | 囊 **>12、<1cm**；3取2 |
| 更年期 | follicle 耗竭 | FSH>LH（倒掛） | **Inhibin↓** | E2<5、P4 0.31 |

> 🧠 ⚠️ **總口訣**：
> 「**快 LH、慢 FSH**；正常育齡 **LH>FSH**（例外 early-follicular／late-luteal）。」
> 三種把 GnRH pulse 壓慢（→FSH>LH）的中樞病因：**Leptin↓（厭食）、Prolactin↑、（生理性）黃體期高 P4**；停經 FSH 高則是周邊 **Inhibin↓**。

## 五、易混陷阱總表 ⚠️

- ⚠️ **絕對值不能分期**：reference range overlap 大，必看「FSH vs LH 相對 + E2」。
- ⚠️ **anorexia 為中樞性 hypo*gonadotropic* hypogonadism**（講者口誤 hyper-，勿記錯）。
- ⚠️ **女性自打 testosterone**（健美）→ 外源雄性素壓軸，**非** leptin 路徑。
- ⚠️ **更年期 FSH>LH 倒掛** 是 Inhibin↓ 造成，補 E2/P4 無法翻轉比例。
- ⚠️ 網路 / AI 週期圖把「FSH 全程 > LH」皆為錯誤。
- ⚠️ PCOS 高 testosterone 來源是 **theca cell**（LH+insulin 維持），非 granulosa。
