# 內分泌動態試驗（Endocrine Dynamic Tests）考試重點摘要
> 來源：FellowCamp 2026 投影片 · Endocrine Dynamic Test · 臺北市立聯合醫院仁愛院區 內分泌新陳代謝科 陳思綺醫師

---

## 0. 總綱：動態試驗的核心邏輯

🧪 **基本原則（背這個就抓到一半分數）**
- **刺激試驗（Stimulation test）→ 找「不足（deficiency）」**：給刺激看激素能不能升上去。
- **抑制試驗（Suppression test）→ 找「過量（excess）」**：給抑制看激素能不能壓下去。
- 動態試驗的時機：**基礎值（basal level）borderline / 模稜兩可**、或臨床與抽血不符時才做。

🧠 **記憶法「刺缺抑多」**：刺激試驗查缺乏、抑制試驗查過量。

⚠️ 做試驗前先排除 **iatrogenic 因素**：抽血時間錯、藥物干擾、實驗室誤差。

📊 **下視丘-腦垂體-標的器官軸對應的動態試驗速查**

| 軸 / 激素 | 過量（→抑制試驗） | 不足（→刺激試驗） |
|---|---|---|
| GH | Acromegaly → **75g OGTT**、Octreotide test | GHD → **ITT(gold standard)**、Glucagon test、Macimorelin、GHRH/Arginine |
| ACTH/Cortisol | Cushing → **DST**、Desmopressin、CRH、IPSS | AI → **Cosyntropin (ACTH) test**、ITT |
| TSH | TSHoma → TRH test | 中樞甲低 → TRH test |
| LH/FSH | — | Hypogonadism → GnRH test（2016 已不建議）、Clomiphene |
| PRL | — | TRH test |
| ADH | — | DI → **Water deprivation test** |
| Aldosterone | PA → **SIT、CCT**、AVS | — |
| Insulin | Insulinoma/NIPHS → **72h fasting**、Mixed-meal、SACST | DM → OGTT、Glucagon test |

> 📌 **High-Yield（必背）**：刺激試驗找不足、抑制試驗找過量。GHD 與 secondary AI 的 gold standard 都是 **ITT**；Cushing 的 subtype gold standard 是 **IPSS**；PA 的 lateralization gold standard 是 **AVS**。

## 1. 生長激素 GH — Acromegaly（過量）

📊 **75g OGTT for Acromegaly（抑制試驗）**
- **目的**：Acromegaly 在 IGF-1 結果模稜兩可時的確認試驗。
- **2024 第14版 Acromegaly consensus**：典型症狀 + <u>IGF-1 > 1.3 倍 ULN</u>（依年齡）即可診斷；equivocal 才加做 OGTT。
- **準備**：OGTT 前 <u>停口服 estrogen 4 週</u>（estrogen 影響 GH axis）。
- **操作**：空腹，75g glucose / 300mL 水 5 分鐘內喝完；**測 GH 於 0、30、60、90、120 分鐘**。
- **判讀（Acromegaly = GH 無法被壓下去）**：
  - 超敏感 assay：GH nadir **<u>未能抑制至 < 0.4 μg/L</u>**
  - 標準 assay：**<u>< 1.0 ng/mL</u>**，且 IGF-1 上升
  - BMI 校正：<u>BMI < 25 → cutoff < 0.4 μg/L；BMI ≥ 25 → < 0.2 μg/L</u>
- ⚠️ **False positive**：starvation、protein-caloric malnutrition、anorexia nervosa。

🧠 **記憶法**：診斷 acromegaly 看「IGF-1 高 + GH 壓不下去」；正常人糖水會壓 GH，肢端肥大症壓不動。

> 📌 **High-Yield（必背）**：Acromegaly OGTT → GH nadir <u>< 0.4 μg/L（超敏）/ < 1.0 ng/mL（標準）</u> + IGF-1 升高。OGTT 前停口服 estrogen 4 週。

## 2. 生長激素 GH — GH deficiency（不足）

📊 **GHD 刺激試驗**：ITT（**gold standard**）、Glucagon stimulation test、Macimorelin、GHRH/Arginine。

🩺 **重點觀念**
- AGHD（成人 GHD）臨床表現**非特異**。
- ⚠️ **只有打算治療時才做確診試驗**（intention to treat）。

💊 **Glucagon Stimulation Test (GST) for GHD**
- **操作**：禁食 8–10 hr；**IM glucagon**（<u>BW < 90kg 給 1mg；≥ 90kg 給 1.5mg</u>）；測 GH + glucose 於 0/30/60/90/120/150/180/210/240 分。
- **判讀（依 BMI）**：

| BMI | GHD cutoff（GH peak） |
|---|---|
| 正常（< 25） | **<u>≤ 3 ng/mL</u>** |
| 過重（25–30）高機率 | ≤ 3 ng/mL |
| 過重（25–30）低機率 | ≤ 1 ng/mL |
| 肥胖（> 30） | **<u>≤ 1 ng/mL</u>** |

> 📌 **High-Yield（必背）**：ITT 是 GHD 與 secondary AI 的 gold standard。GST GHD：正常體位 GH peak <u>≤ 3 ng/mL</u>、肥胖 <u>≤ 1 ng/mL</u>。GHD 確診前提是「有意願治療」。

---

## 3. Glucagon Stimulation Test for β-cell reserve（注意：與 GHD 版本不同）

🧪 **目的**：評估 **β-cell reserve**（區分 T1DM）。
- **準備**：測前 3 天足量碳水（≥ 5g/kg）避免 ketosis；勿劇烈運動/抽菸；停干擾 OHA、測前 12hr 停/減 insulin。
- **操作**：**Glucagon 30 μg/kg（max 1mg）slow IV**；測 **C-peptide 於 0 與 6 分**。
- **判讀（符合任一 → likely T1DM）**：
  - <u>0 min C-peptide < 0.5 ng/mL</u>，或
  - <u>6 min C-peptide < 1.8 ng/mL</u>，或
  - <u>Δ C-peptide < 0.7 ng/mL</u>

⚠️ **陷阱**：同名 Glucagon test 有兩種用途——β-cell reserve 測 **C-peptide（IV）**；GHD 測 **GH（IM）**，別搞混給藥途徑與檢測項目。

> 📌 **High-Yield（必背）**：β-cell reserve glucagon test → 0min C-pep <u>< 0.5</u>、6min <u>< 1.8</u>、Δ <u>< 0.7</u> ng/mL 任一 = likely T1DM。

## 4. HPA 軸 — Cushing 診斷（是否有 Cushing？）

🩺 **第一步：確認「有沒有」Cushing（門診篩檢首選）**
- Late-night salivary cortisol / circadian rhythm
- 24h UFC（≥ 2 次）
- **Low-dose / 1mg DST**

📊 **1mg Overnight DST（篩檢）**
- **操作**：<u>11 pm 給 dexamethasone 1mg po</u> → 隔天 <u>8–9 am 測 cortisol</u>（進食前）。
- **判讀**：正常 **<u>cortisol 抑制到 < 1.8 μg/dL</u>**。
- ⚠️ **False positive**：憂鬱、失眠、生理/情緒壓力、近期酗酒、肥胖、甲亢、acromegaly、懷孕、藥物（**oral estrogen、phenytoin、rifampin、barbiturates**——加速 dex 代謝）。

📊 **48h Low-dose DST（2mg/天，確診）**
- **操作**：D1 baseline cortisol/ACTH（8am）→ **dexamethasone 0.5mg q6h × 2 天（8 doses = 4mg）** → D3 8am 測 cortisol（末次劑量後 6hr）。
- **判讀**：正常 **<u>D3 cortisol < 1.8 μg/dL</u>**。

🧠 **記憶法**：低劑量 DST（1mg 或 2mg/天）= 問「有沒有 Cushing」；高劑量 = 問「是哪一種」。

> 📌 **High-Yield（必背）**：DST 抑制 cutoff 一律 <u>cortisol < 1.8 μg/dL</u>。1mg overnight 為篩檢、48h 2mg/天為確診。estrogen/phenytoin/rifampin/barbiturates 造成偽陽性。

## 5. HPA 軸 — Cushing 鑑別診斷（是哪一種？）

📊 **第一步看 plasma ACTH**

| 類別 | Plasma ACTH (pg/mL) | 機轉 |
|---|---|---|
| Cushing disease（垂體） | ~10–200 | ACTH-dependent |
| Adrenal tumor（腎上腺） | **< 10（suppressed）** | ACTH-independent |
| Ectopic ACTH | ~100–12,000 | ACTH-dependent |

- ACTH **suppressed → CT adrenals**（考慮 macronodular hyperplasia）。
- ACTH **detectable → ACTH-dependent disease**（鑑別 Cushing disease vs ectopic）。

📊 **ACTH-dependent 的鑑別工具**
1. **IPSS（gold standard）**
2. High-dose DST（8mg/天）
3. CRH stimulation test
4. Desmopressin stimulation test
5. Pituitary MRI

🩺 **指向 Cushing disease 的綜合判讀**（皆滿足才直接 TSH 手術）：
- 高/低劑量 DST cortisol 抑制 > 50% **AND**
- CRH 後 cortisol 上升 > 50% **AND**
- Pituitary MRI 陽性
- 若任一不符 → 做 **IPSS with CRH**。

---

### 5a. Inferior Petrosal Sinus Sampling (IPSS)

📊 **目的**：ACTH-dependent CS 中，最 robust 區分「垂體 vs 非垂體」並 lateralize 垂體瘤。
- **操作**：3 個 port（Peripheral、Left IPS、Right IPS）同時抽 baseline ACTH → 給 **DDAVP 10μg IV 或 CRH 100μg IV** → 在 **2、5、15 分**再同時抽三 port ACTH。
- **判讀（IPS:P = 中心對周邊比值）**：
  - **Ectopic ACTH**：<u>baseline IPS:P < 1.4</u>
  - **Pituitary-origin（Cushing disease）**：<u>baseline IPS:P ≥ 2</u> 且 **<u>刺激後 ≥ 3</u>**
  - **Lateralization（左右定位）**：<u>inter-sinus ratio (IPS:IPS) ≥ 1.4</u>
- ⚠️ **併發症**：groin hematoma、thromboembolic event。

🧠 **記憶法「2/3」**：Pituitary baseline **≥ 2**、刺激後 **≥ 3**；ectopic < 1.4；定位用 1.4。

> 📌 **High-Yield（必背）**：IPSS 是 Cushing subtype gold standard。Pituitary：baseline IPS:P <u>≥ 2</u>、刺激後 <u>≥ 3</u>；Ectopic：<u>< 1.4</u>；lateralization IPS:IPS <u>≥ 1.4</u>。

---

### 5b. High-Dose DST（8mg/天）

📊 **目的**：Cushing disease vs Ectopic（**僅在無 IPSS 時用**）。
- **操作**：D1 baseline → **dexamethasone 2mg q6h × 2 天（8 doses = 16mg）** → D3 8am cortisol。
- **判讀**：
  - **Cushing disease**：<u>D3 cortisol < 50% baseline（可被抑制）</u>
  - **Ectopic ACTH**：non-suppressible（壓不動）

🧠 **記憶法**：垂體瘤還保留部分負回饋 → 高劑量壓得下去；異位/腎上腺源 → 壓不動。

---

### 5c. CRH Stimulation Test 與 Desmopressin Test（鑑別 origin）

📊 兩者判讀 cutoff 相同：

| 反應 | Pituitary ACTH adenoma | Adrenal tumor / Ectopic / Pseudo-Cushing |
|---|---|---|
| **Δ Cortisol** | **> 20%** | No response |
| **Δ ACTH** | **> 50%** | No response |

- **CRH test**：IV ovine/human CRH **1 μg/kg 或 100μg**；測 ACTH/cortisol q15min × 1–2hr。正常人 CRH 升 ACTH/cortisol 15–20%；Cushing disease **誇張反應**（10% 不反應）；adrenal/ectopic 多無反應。
- **Desmopressin test**：IV DDAVP 10μg；測 ACTH/cortisol -15/0/15/30/45/60/90/120 min。也可鑑別 **Cushing disease vs Pseudo-Cushing**。

> 📌 **High-Yield（必背）**：CRH/DDAVP 刺激下 **Cushing disease：Δcortisol > 20%、ΔACTH > 50%**；ectopic/腎上腺/pseudo 無反應。High-dose DST：Cushing disease D3 cortisol < 50% baseline。

## 6. HPA 軸 — Adrenal Insufficiency（不足）

🩺 **基礎評估**
- 第一線：**8–9 am serum cortisol**。
- 合成 glucocorticoid 須停 ≥ 18–24hr 後再測 HPA。
- **<u>Basal cortisol > 14.5 μg/dL → HPA 完整</u>**。
- Endocrine Society：morning cortisol <u>3–15 μg/dL</u> 才做 ACTH stimulation test。
  - <u>cortisol < 3 μg/dL → AI</u>
  - <u>cortisol > 15 μg/dL → 排除 AI</u>

📊 **Rapid ACTH (Cosyntropin/Synacthen) Stimulation Test**
- **操作**：任何時間、不需 NPO；**250 μg（標準）或 1 μg（低劑量）cosyntropin slow IV**；抽血 0min（ACTH+cortisol）、30/60min（cortisol）。
- **判讀（正常）**：**<u>peak cortisol ≥ 18.1–20 μg/dL（標準劑量，依 assay）；低劑量 30min cortisol ≥ 18.1 μg/dL</u>**。
- ⚠️ **陷阱**：
  - CBG/albumin 異常時不可靠。
  - **垂體損傷 < 6 週時敏感度低**（腎上腺尚未萎縮，可能假陰性）。
  - **若 ACTH test 正常，多數情況不需做 ITT**。

📊 **ITT for secondary AI（gold standard）**
- **禁忌**：缺血性心臟病、懷孕、腦血管疾病、癲癇、嚴重 hypopituitarism（<u>9am cortisol < 6.5 μg/dL</u>）；老人/重症為相對禁忌。
- **操作**：測前 EKG、baseline cortisol；NPO；**regular insulin IV <u>0.1–0.15 U/kg</u>**（肥胖/DM/acromegaly/Cushing 用 <u>0.2–0.3 U/kg</u>）。
- **達標低血糖**：<u>plasma glucose < 40 mg/dL</u>（多在 20–35 分、伴 neuroglycopenia）；45 分仍未達標 → 重打半量 RI。低血糖時測 cortisol/GH，並於 **30 分後再測**（補糖後）。
- **判讀**：
  - 正常 cortisol：**<u>peak > 18 μg/dL</u>**
  - 正常 GH：**<u>peak > 5 ng/mL</u>**；**<u>GHD: peak < 3 ng/mL</u>**

🧠 **記憶法「18 大關」**：ACTH test 與 ITT 的正常 cortisol peak 都記 **≥ 18 μg/dL**。

> 📌 **High-Yield（必背）**：Cosyntropin test 正常 peak cortisol <u>≥ 18.1–20 μg/dL</u>。ITT 達標需 glucose <u>< 40 mg/dL</u>，cortisol peak <u>> 18</u>、GH peak <u>> 5</u>（GHD < 3）。垂體損傷 < 6 週做 ACTH test 易假陰性。

## 7. 垂體-性腺軸（LH/FSH）

📊 **Hypogonadism 刺激試驗**：GnRH/LHRH test、Clomiphene test。
- ⚠️ **Endocrine Society 2016 不建議**用 GnRH 動態試驗（無有用診斷資訊）。

> 📌 **High-Yield（必背）**：性腺軸動態試驗（GnRH test）2016 指引已**不建議**，考題若問 hypogonadism 評估首選，答案偏向基礎 LH/FSH/testosterone，而非 GnRH 刺激。

## 8. 垂體-甲狀腺軸 — TRH Test

📊 **目的**
- 區分 **TSHoma vs Thyroid hormone resistance (RTH)**。
- 偵測中樞甲低 / PRL 缺乏（目前少用）。
- **禁忌**：懷孕早期。**Caution**：氣喘、缺血性心臟病。

🧪 **操作**：synthetic TRH (Protirelin) **400 μg（200–500μg）IV** 15–30 秒；測 TSH 於 0/30/60/(180) 分。

📊 **判讀（TSH）**
- 正常：**<u>ΔTSH ≥ 5–6 mU/L</u>** at 20–30 min / **<u>peak TSH ≥ 2.5 倍</u>**，60 分回落。
- **<u>TSHoma：blunted（鈍化）反應</u>** ；**<u>RTH：preserved 或 exuberant 反應</u>**。
- Blunted/delayed 也見於：中樞甲低、甲亢、euthyroid Graves、抑制性甲狀腺治療、isolated TSH deficiency、腎衰、憂鬱、藥物（glucocorticoid、L-dopa、bromocriptine、OCP、aspirin）。

📊 **判讀（PRL）**
- 正常：**<u>PRL ≥ 2.5 倍</u>**（或 > 2 μg/L 且升 > 200%）。
- 反應**降低/缺失 + 低 baseline PRL → 泌乳素儲備不足**（垂體破壞/切除等）。
- ⚠️ **TRH test 對 hyperprolactinemia 鑑別無用**。

🧠 **記憶法「瘤鈍抗旺」**：TSH**oma** 反應**鈍**、**R**TH（抗性）反應**旺**。

> 📌 **High-Yield（必背）**：TRH test 鑑別 TSHoma（blunted）vs RTH（exaggerated）。正常 ΔTSH <u>≥ 5–6 mU/L</u> / peak <u>≥ 2.5×</u>。

## 9. 後葉 — ADH / Diabetes Insipidus

🩺 **多尿-多飲（polyuria-polydipsia）流程**
- 確認 polyuria：<u>> 50 mL/kg/24h</u>，且 urine osm < 800。
- 看 serum Na：
  - <u>Na < 135 → primary polydipsia</u>
  - <u>Na > 147 → central / nephrogenic DI</u>
  - Na 136–146 → **water deprivation test** 或 baseline copeptin
- ⚠️ 傳統試驗準確率僅 ~70%（primary polydipsia 僅 41%）。

📊 **Water Deprivation Test（操作）**
- 8am 起禁食禁水；每 1hr 記尿量/尿滲；每 2hr 記體重/BP/Na/血滲。
- **立即停試驗**：體重↓ 5%、低血壓、意識變化。
- **到「n 時間 (0')」給 DDAVP（定義達任一）**：體重↓ 3–5%；連續兩次 Δurine osm ≤ 10% 且 BW↓ 2%；或連續兩次 Δurine osm < 30 mOsm/kg；或 plasma osm 達 295–300；或 Na ≥ 150 mmol/L。
- **(0') 給 DDAVP 2μg SC**，於 60/120 分再測。

📊 **判讀表**

| 診斷 | 脫水後 Urine Osm | 給 DDAVP 後 |
|---|---|---|
| **Normal** | **> 800** | — |
| **Central DI** | **< 300** | **↑ ≥ 50%** |
| **Nephrogenic DI** | **< 300** | **↑ < 50%（甚至 < 15%）** |
| **Partial central DI** | 300–800 | ↑ > 9%（另版 < 50%） |
| **Primary polydipsia** | 300–800（另版 > 800） | ↑ < 9%（< 50%） |

🧠 **記憶法**：脫水後 < 300 = 「真 DI」；給 DDAVP 後**升 ≥ 50% = central**（缺 ADH 補就好）、**升不動 = nephrogenic**（腎臟對 ADH 無反應）。

> 📌 **High-Yield（必背）**：脫水後 urine osm 正常 <u>> 800</u>、DI <u>< 300</u>。DDAVP 後 central DI <u>↑ ≥ 50%</u>、nephrogenic <u>↑ < 50%</u>。

## 10. 腎上腺 — Primary Aldosteronism (PA)

🩺 **流程**：ARR 篩檢 → 確認試驗 → subtype（影像/AVS）→ MRA 或腹腔鏡腎上腺切除。
- ⚠️ Endocrine Society 2025：**所有高血壓都考慮篩 PA**；同測 renin + aldosterone + **potassium**。初篩可先不停干擾藥，但判讀要考量偽陰/偽陽。

📊 **ARR 篩檢 cutoff（依指引）**

| 指引 | ARR (PAC/PRA) | 備註 |
|---|---|---|
| Endocrine Society / ESH / ESC 2016/2020 | **30**（最常用） | PAC/DRC 用 3.7 |
| **Taiwan Society of Aldosteronism 2017** | **35**（提高 specificity） | — |
| ACC 2017 | 30 + PAC > 10 | — |
| Hypertension Canada 2020 | 20（高敏）/ 27（高特異） | — |

- 💊 **Endocrine Society 2025 PA 診斷標準**：renin 低/抑制 + aldosterone 相對不當升高：
  - PRA ≤ 1 ng/mL/h 或 DRC ≤ 8.2 mU/L
  - Aldosterone（免疫法）≥ 10 ng/dL（≥ 277 pmol/L）或 LC-MS/MS ≥ 7.5 ng/dL
  - **AND** ARR 升高：免疫法/PRA **> 20**、LC-MS/MS/PRA **> 15**
- ⚠️ 自發性低血鉀 + PAC > 20 ng/dL + renin 測不到 → 可**免確認試驗**。

📊 **干擾 ARR 的藥物（必背偽陰/偽陽）**

| 藥物 | ARR 效應 | 停藥時間 |
|---|---|---|
| K⁺-sparing diuretics（MRA/螺內酯） | ↓ 偽陰 | **4–6 週** |
| K⁺-wasting diuretics | ↓ 偽陰 | 4–6 週 |
| ACEi / ARB | ↓ 偽陰 | 2 週 |
| DHP-CCB | ↓ 偽陰 | 2 週 |
| **β-blocker、clonidine、α-methyldopa、NSAID** | **↑ 偽陽** | 2 週 |
| Renin inhibitor | PRA↓/DRC↑ → 偽陽 | 2 週 |
| Hydralazine | ↓ 偽陰 | 2 天 |

- 🧠 **記憶法**：**MRA/利尿劑停最久（4–6 週）**；β-blocker/clonidine 是少數造成**偽陽**的藥（降 renin）。對 ARR 影響最小可續用：**non-DHP CCB、α1-blocker、中樞 α-agonist**。

> 📌 **High-Yield（必背）**：ARR 篩檢 cutoff 國際 <u>30</u>、台灣 <u>35</u>。Endocrine Society 2025 診斷 ARR（免疫法/PRA）<u>> 20</u>。MRA/利尿劑停 <u>4–6 週</u>；β-blocker 造成偽陽。

---

### 10a. PA 確認試驗總表（SIT / CCT / OSLT / FST / Losartan）

📊

| 試驗 | 操作要點 | 判讀（PA 可能性高） | 陷阱 |
|---|---|---|---|
| **Seated SIT**（首選 1C） | 0.9% NaCl 2L over 4h，座姿前/中 60min；測前後 PAC、PRA、K | **<u>Post-PAC < 7.8 ng/dL（免疫法）/ 5.8（LC-MS/MS）→ PA unlikely</u>**；Australia > 6、**<u>TAIPAI > 16 → very probable PA</u>** | 禁：未控 HTN、腎功能不全、心律不整、心衰、嚴重低血鉀 |
| **CCT（Captopril）**（首選 1B） | 座姿 60min；**captopril 50mg po**；測 0/(1)/2h PAC、PRA、cortisol | 正常 PAC↓ > 30%；**<u>2h PAC ≥ 10 ng/dL（免疫法）/ 7.5（LC-MS/MS）+ renin 抑制 → PA</u>**；或 2h PAC > 11 + ARR > 20；TAIPAI 90min ARR > 35 | Sens 70–100%、Spec 68–95%；可用於有容積過載風險者；**血管性水腫風險** |
| **OSLT（口服鈉負荷）** | NaCl > 6g/d × 3 天；測 D3–D4 24h 尿 aldosterone | **<u>> 12–14 μg/24h → 高度可能</u>**；< 10 → unlikely（須尿 Na > 200 mmol/d 確認鈉攝足） | 同 SIT 禁忌 |
| **Fludrocortisone suppression** | Fludro 0.1mg q6h + KCl × 4 天 + NaCl | D4 **<u>PAC > 6 ng/dL + PRA < 1 + 10am cortisol < 7am</u>** | 需住院頻抽血；QT/心律不整風險 |
| **Losartan suppression（TAIPAI）** | losartan 50mg po；2h 測 PAC、PRA | **<u>ARR > 35 ng/dL per ng/mL/h</u>** | 診斷準確度優於 CCT |

🧠 **記憶法**：SIT 看「壓不下去的 aldosterone」；正常人鹽水會壓 aldosterone，PA 壓不動。臨床最常用 **SIT + CCT**。

⚠️ Endocrine Society 2025：**確認試驗僅在「intermediate probability of lateralizing PA + 有手術意願」時才做**；severe PA 高機率者可直接走 CT/AVS。

> 📌 **High-Yield（必背）**：Seated SIT post-PAC <u>< 7.8（免疫法）</u> 排除 PA、TAIPAI <u>> 16</u> 確診；CCT 2h PAC <u>≥ 10 ng/dL</u> + renin 抑制。最常用 SIT 與 CCT。

### 10b. Adrenal Venous Sampling (AVS) — lateralization gold standard

🧪 **目的**：可靠區分 **lateralizing（單側）vs bilateral** PA。
- ⚠️ **可免 AVS**：< 35 y/o + 自發性低血鉀 + PAC > 30 ng/dL + 抑制 PRA + 單側 > 1cm 腺瘤。
- **操作**：矯正 HTN/低血鉀；隔夜禁食、平躺 1hr；可 unstimulated 或 cosyntropin（**250μg bolus 或 50μg/h infusion**）刺激。

📊 **三大指標（必背 cutoff）**

| 指標 | 公式 | Cutoff | 意義 |
|---|---|---|---|
| **Selectivity index (SI)** | cortisol[AV] / cortisol[IVC] | **<u>Unstimulated ≥ 2（>1.4–3）；ACTH-stim > 5（≥3–5）</u>** | 確認 AV 插管成功 |
| **Lateralization index (LI)** | (Aldo/Cort)[high AV] / (Aldo/Cort)[low AV] | **<u>≥ 4（unstim 或 stim；另版 2–4 / 2.6–4）</u>** | 區分 lateralizing vs bilateral |
| **Contralateral suppression index (CSI)** | (Aldo/Cort)[low AV] / (Aldo/Cort)[IVC] | **<u>< 1（刺激前後皆）</u>** | 對側 aldosterone 被抑制 |

- 🧠 **記憶法「選2側4抑1」**：SI ≥ 2（插管成功）、LI ≥ 4（定側）、CSI < 1（對側被壓）。ACTH 刺激後 SI 門檻升到 5。
- 📌 ACR（aldosterone-cortisol ratio）= 用 cortisol 校正 aldosterone，避免稀釋誤差。

> 📌 **High-Yield（必背）**：AVS 三指標 **SI ≥ 2（unstim）/ > 5（stim）、LI ≥ 4、CSI < 1**。SI 確認插管、LI 定側、CSI 看對側抑制。

## 11. 胰臟 — 低血糖 / Insulinoma（過量 insulin）

🧪 **原則**：低血糖時 insulin 應被抑制；**低血糖下 C-peptide/insulin 仍高 = endogenous hyperinsulinism**。

📊 **72-hour Fasting Test（操作）**
- baseline 測 glucose/insulin/C-peptide；只給無熱量無咖啡因液體。
- one-touch **Q4H till < 60 → 改 Q1H till < 55**；< 55 時測 plasma glucose/insulin/C-peptide + 看 neuroglycopenia。
- **結束條件**：神經低血糖症狀；或症狀 + glucose < 55；或**無症狀 + glucose < 45**；或滿 72hr。

📊 **Mixed Meal Test**（懷疑**餐後**低血糖）
- 給已知會誘發的特定餐（**不用 75g glucose**）；Q30min till < 55（至少 3hr、可延 5hr）。

📊 **判讀（Whipple's triad 達標時的內分泌診斷 cutoff）**
- 當 **plasma glucose < 55 mg/dL** 或嚴重低血糖症狀：
  - **<u>C-peptide > 0.6 ng/mL（0.2 nmol/L）</u>**
  - **<u>Insulin > 3.0 μU/mL（18 pmol/L）</u>**
  - **<u>Pro-insulin > 5.0 pmol/L</u>**
  - β-hydroxybutyrate **≤ 2.7**、glucagon 後 glucose **↑ > 25 mg/dL**

📊 **低血糖鑑別表（必考）**

| 診斷 | Insulin | C-peptide | Proinsulin | OHA | 抗體 |
|---|---|---|---|---|---|
| **Insulinoma / NIPHS / PGBH** | ≥ 3 | **≥ 0.2** | ≥ 5 | No | Neg |
| **Exogenous insulin（外源）** | **»3（很高）** | **< 0.2（低）** | < 5 | No | Neg/(Pos) |
| **OHA（磺脲類）** | ≥ 3 | ≥ 0.2 | ≥ 5 | **Yes** | Neg |
| **Insulin autoimmune** | »3 | »0.2 | »5 | No | **Pos** |
| **IGF 介導** | < 3 | < 0.2 | < 5 | No | Neg |

🧠 **記憶法**：**內源 insulinoma → insulin↑ + C-peptide↑**（一起做出來）；**外源 insulin → insulin↑ 但 C-peptide↓**（外打的沒 C-peptide）；OHA → insulin↑ C-pep↑ 但**血裡驗得到藥**。

> 📌 **High-Yield（必背）**：低血糖（glucose < 55）時 **insulin > 3、C-peptide > 0.6 ng/mL、proinsulin > 5** = 內源性高胰島素。外源 insulin 的招牌是 **C-peptide 低**。

### 11a. SACST（Selective Arterial Calcium Stimulation Test）

🧪 **適應症**：endogenous hyperinsulinism 中 **insulinoma 定位 / 鑑別 nesidioblastosis**。
- **操作**：肝靜脈取樣；於 **GDA、SPA、SMA** 注入 **calcium gluconate（0.025 mEq Ca²⁺/kg）**；右肝靜脈於 baseline 及 20/40/60 秒測 insulin；每次刺激間隔 5 分。
- **判讀**：**<u>insulin 上升 2 倍（peak > 100 μIU/mL）於 30 或 60 秒</u>** = 該動脈供應區陽性。

> 📌 **High-Yield（必背）**：SACST 陽性 = 注鈣後肝靜脈 insulin **升 2 倍 / peak > 100 μIU/mL**，可區域定位 insulinoma。

## 12. 試驗總表（試驗 → 目的 → 操作 → 判讀 cutoff → 陷阱）

📊

| 試驗 | 目的 | 操作重點 | 判讀 cutoff | 陷阱 |
|---|---|---|---|---|
| **1mg overnight DST** | 篩檢 Cushing | 11pm dex 1mg → 8–9am cortisol | 正常壓至 **< 1.8 μg/dL** | estrogen/phenytoin/rifampin/barbiturate、憂鬱/肥胖偽陽 |
| **48h low-dose DST（2mg/d）** | 確認 Cushing | dex 0.5mg q6h×2d → D3 cortisol | **< 1.8 μg/dL** | 同上 |
| **High-dose DST（8mg/d）** | Cushing dz vs ectopic | dex 2mg q6h×2d | Cushing dz **D3 < 50% baseline**；ectopic 壓不動 | 僅 IPSS 不可用時 |
| **CRH / DDAVP test** | Cushing origin | CRH 1μg/kg(100μg) / DDAVP 10μg IV | Cushing dz **Δcortisol > 20%、ΔACTH > 50%** | 10% Cushing dz 不反應 |
| **IPSS** | Cushing subtype（gold std） | 三 port + CRH/DDAVP | Pituitary baseline **≥ 2**、stim **≥ 3**；ectopic < 1.4 | 侵入性、血栓/血腫 |
| **Cosyntropin (ACTH) test** | 評 AI | 250μg/1μg IV，測 0/30/60min cortisol | peak **≥ 18.1–20 μg/dL** | 垂體損傷 < 6 週假陰性；CBG 異常 |
| **ITT** | secondary AI + GHD（gold std） | RI 0.1–0.15 U/kg，glucose < 40 | cortisol peak **> 18**、GH peak **> 5**（GHD < 3） | 禁 CAD/癲癇/孕/cortisol < 6.5 |
| **75g OGTT（acromegaly）** | Acromegaly 確認 | 75g glucose，測 GH 0–120min | GH nadir **< 0.4 μg/L**（標準 < 1.0）+ IGF-1↑ | 停 estrogen 4 週；饑餓/AN 偽陽 |
| **Glucagon test（GHD）** | GHD | IM glucagon 1–1.5mg | GH peak **≤ 3（瘦）/ ≤ 1（胖）** | 與 C-peptide 版混淆 |
| **Glucagon test（β-cell）** | β-cell reserve | IV 30μg/kg，C-pep 0/6min | T1DM：6min C-pep **< 1.8** | 須避 ketosis |
| **TRH test** | TSHoma vs RTH | TRH 400μg IV，TSH 0/30/60 | 正常 ΔTSH **≥ 5–6**；TSHoma 鈍、RTH 旺 | 孕婦禁；多種藥/病鈍化 |
| **Water deprivation** | DI 鑑別 | 禁水→DDAVP 2μg SC | DI < 300；DDAVP 後 central **↑≥50%**、nephro **<50%** | 體重↓5%/低血壓停止 |
| **Seated SIT** | PA 確認 | NaCl 2L/4h 座姿 | post-PAC **< 7.8** 排除；TAIPAI **> 16** | 心衰/腎不全/低鉀禁 |
| **CCT** | PA 確認 | captopril 50mg | 2h PAC **≥ 10** + renin 抑制 | 血管性水腫 |
| **AVS** | PA 定側（gold std） | ±cosyntropin | SI **≥2/>5**、LI **≥4**、CSI **< 1** | 插管失敗看 SI |
| **72h fasting / mixed meal** | insulinoma/低血糖 | 禁食 till glucose < 55 | insulin **> 3**、C-pep **> 0.6**、proinsulin **> 5** | 外源 insulin C-pep 低 |
| **SACST** | insulinoma 定位 | 動脈注鈣，肝靜脈測 insulin | insulin **升 2×/> 100 μIU/mL** | 區域定位非精準定位 |

> 📌 **High-Yield（必背）總結金句**：
> - **三大 gold standard**：ITT（secondary AI + GHD）、IPSS（Cushing subtype）、AVS（PA 定側）。
> - **DST cortisol 一律 < 1.8 μg/dL**；**ACTH test/ITT cortisol peak ≥ 18**；**GHD GH peak < 3、正常 > 5**。
> - **CRH/DDAVP**：Cushing disease Δcortisol > 20% / ΔACTH > 50%。
> - **ARR**：國際 30、台灣 35。
> - **AVS**：SI 2、LI 4、CSI 1（「選2側4抑1」）。
> - **低血糖三寶**：insulin > 3、C-peptide > 0.6、proinsulin > 5；外源 insulin 招牌是 C-peptide 低。
