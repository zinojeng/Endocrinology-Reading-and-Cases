# 糖尿病的骨骼脆弱與抗骨鬆治療（Denosumab）考試精華摘要
> 來源：FellowCamp 2026 · Managing Skeletal Fragility in Diabetes（Denosumab）· 黃兆山醫師（骨鬆學會前理事長）／Amgen Symposium

---

## 🧪 一、糖尿病骨鬆與一般骨鬆「不一樣」在哪？

糖尿病骨鬆的核心觀念：**不能只看 BMD（density alone），要看 bone quality（骨品質）**。

### Type 1 vs Type 2 糖尿病的骨頭差異（比較表）

| 項目 | **Type 1 DM** | **Type 2 DM** |
|---|---|---|
| 病生理 | 自小 **insulin/IGF-1 不足** | **insulin resistance** |
| BMD | 相對**偏低**（平均↓<u>20–40%</u>） | **正常或偏高** ⚠️ |
| Bone turnover | **低**（low turnover） | **低**（low turnover） |
| Microarchitecture / 骨品質 | 異常、變差 | 異常、變差 |
| FRAX | 偏高，但**易低估** | 偏高，但**易低估** |

- ⚠️ **秒殺易混點**：Type 2 DM 病人 BMD 看起來「正常甚至偏高」，**會誤判為沒有骨鬆**，但其實骨品質差、骨折風險高。這是最常考的陷阱。
- 兩型 DM 都是 **low bone turnover** → bone turnover marker（BTM）偏低。
- 🧪 機轉總結：DM 病人 **osteoclast activity ↑（resorption ↑）、osteoblast 被抑制（formation ↓）** → 動態平衡偏向骨流失 → 骨折風險↑。涉及 AGEs/RAGE system、膠原蛋白醣化（影響骨膠原品質）、GLP-1/GIP↓、leptin/脂肪組織、腎臟、副甲狀腺等多重路徑。

### Bone Turnover Marker（BTM）抽血原則
- **抽血比尿液穩定**；建議同時驗一個 **formation marker + 一個 resorption marker**，觀察 coupling effect。

> 📌 **High-Yield（必背）**：DM 骨鬆＝**低骨轉換 + 骨品質差**。<u>Type 1 BMD↓20–40%</u>；**Type 2 BMD 正常/偏高但骨折風險仍高（測 BMD 會低估）**。

---

## 📊 二、BMD cutoff 校正：DM 的「-2.0 等於一般人 -2.5」

- 一般人骨鬆診斷：<u>T-score ≤ -2.5</u>。
- 糖尿病（尤其用 insulin）病人：BMD 落在 <u>約 -1.9 ~ -2.0</u> 時，骨折風險已與一般人 **-2.5** 相當。
- ⚠️ **老師強調考點**：DM 病人 **T-score ≤ -2.0 就應視為骨鬆/高風險**，要積極處理，不要等到 -2.5。

### FRAX 在 DM 要「校正」（三選一，必背）
DM 會讓 FRAX **低估**真實風險，校正方法（任一）：
1. **年齡 +10 歲**輸入；或
2. 勾選 **secondary osteoporosis（次發性）**；或
3. **T-score 再降 0.5 SD**（實務上 FRAX 是 key BMD 值，需自行換算）。

- 🧠 記憶法：**「加十歲、打次發、減半個 SD」**（+10 / secondary / -0.5 SD）。
- 未來 **FRAX plus** 可自動校正（但需付費，目前多用免費 FRAX）。

> 📌 **High-Yield（必背）**：DM 用 <u>T ≤ -2.0</u> 當高風險門檻；FRAX 必校正＝**+10歲 / 勾secondary / -0.5 SD（三選一）**。

## 💊 三、糖尿病用藥對骨頭的影響（高頻比較題）

| 藥物 | BMD | 骨折風險 | 備註 |
|---|---|---|---|
| Metformin | neutral / 略↑ | neutral / ↓ | 安全 |
| DPP-4i | neutral | neutral | 安全 |
| GLP-1 RA | neutral / 略↑ | neutral / ↓ | 安全 |
| **Sulfonylurea** | 無影響 | **↑** ⚠️ | 因**低血糖→跌倒** |
| **TZD（PPARγ）** | **↓** | **↑** ⚠️ | 抑制 osteoblast |
| **Insulin** | neutral / ↓ | **↑** ⚠️ | 實際 study 骨折↑（含低血糖/跌倒） |
| SGLT2i | neutral | neutral | CANVAS（Canagliflozin）曾報骨折↑，後續 study 未證實 → 視為 neutral |

- 🧠 記憶法：**「TZD、SU、Insulin」三害骨**（T-S-I）＝會增加骨折風險的 DM 用藥。
- ⚠️ **易混陷阱**：SGLT2i 只有 **CANVAS（Canagliflozin）** 報過骨折增加，現多認為 neutral，別一律當成增加骨折。

> 📌 **High-Yield（必背）**：增加骨折風險的 DM 藥＝**TZD（直接抑制 osteoblast）＋ SU、Insulin（低血糖/跌倒）**。SGLT2i 現視為 neutral。

## 🩺 四、健保給付規則（台灣，超高頻臨床考點）

- ⚠️ **2024（去年）3月起新制**：糖尿病病人**即使尚未骨折（primary prevention）**也可給付抗骨鬆藥，但有兩個**硬條件**：
  1. **正在使用 insulin**（健保財務考量綁定 insulin）；且
  2. **BMD T-score < <u>-2.5</u>**。
- 可給付藥物：**Denosumab（Prolia / Xgeva）**等。
- 健保**只看 BMD**（不看 FRAX/TBS）；學會建議 DM 病人應額外做 FRAX 校正與骨品質評估。
- 已骨折者（secondary prevention）本就可積極治療。

> 📌 **High-Yield（必背）**：DM 未骨折要健保給付 Denosumab → **必須「有用 insulin」＋「T-score < -2.5」兩條件齊備**（2024/3 新制）。

## 🩺 五、找出高風險病人 & 評估工具

### ADA「特別注意」的高風險族群（必背清單）
- BMD **T < -2.0**
- **HbA1c > 8**（控制不良）
- **DM 病程 > 10 年**
- 使用 **insulin / SU / TZD**
- 有**低血糖**發作
- 有**大血管 / 小血管併發症**（neuropathy、retinopathy、CKD…）

### 評估工具（除了 DEXA 還要加做）
- **FRAX（校正版）**：未來10年骨折風險。
- **TBS（Trabecular Bone Score）**：新機型內建，評估骨小樑/骨品質，對**次發性骨鬆**特別有用。
- **VFA / IVA**（DEXA 內建，極低輻射）或 **lumbar spine 側位 X-ray**：找隱性 vertebral fracture。
- 臨床線索：**身高變矮（自年輕 ≥ <u>4 公分</u>有意義）**、背痛、跌倒史。

### 篩檢年齡
- **≥ 65 歲**：常規做 BMD。
- **≥ 50 歲 + DM 風險因子**：應做 BMD + FRAX（+ TBS / VFA 更佳）。

> 📌 **High-Yield（必背）**：DM 高風險＝<u>HbA1c>8、病程>10年、TZD/SU/Insulin、低血糖、micro/macrovascular 併發症、T<-2.0</u>。評估＝**DEXA + 校正FRAX + TBS + VFA/IVA**。

## 💊 六、治療策略：風險分層與 Anabolic-first

### AACE / 台灣骨鬆學會 風險分層 → 用藥順序

| 風險層級 | 定義（重點） | 首選治療 |
|---|---|---|
| **High risk** | T ≤ -2.5；或 osteopenia 合併脆弱性骨折（spine/hip/pelvic/distal forearm）；FRAX hip>3% 或 MOF>20% | **Antiresorptive**：Denosumab、bisphosphonate、SERM |
| **Very high risk（極高）** | 近期骨折、**多發性骨折**、BMD 極低（**T ≤ -3.0**）、治療中仍骨折、FRAX 再 +50%（hip 4.5%、MOF 30%） | **Anabolic first**：**Teriparatide / Romosozumab**，再 sequential 接 antiresorptive |

- 🧠 口訣：**極高風險 → Anabolic first（先建骨）→ 再 sequential 接抗吸收藥維持**。
- ⚠️ **必考停藥概念**：**Denosumab（及 anabolic agent）絕不可驟然停藥**，停藥後新增的 BMD 會**快速流失（回到原點，甚至反彈性骨折）**；停 Denosumab 必須接 **antiresorptive（如 bisphosphonate）銜接**保住骨密度。

### 各藥對 DM 病人療效（重點）
- **Bisphosphonate**：BMD↑、骨折↓（含 zoledronic acid）。
- **Denosumab**：BMD↑明顯（**total hip 與 lumbar spine 2年約↑<u>8%</u>**）、椎體骨折↓；持續治療 BMD **持續上升**（無 bisphosphonate 約3年後的 plateau）。
- **Teriparatide / Abaloparatide（PTHrP 類，台灣未上市）/ Romosozumab**：BMD↑、骨折↓（多為 post-hoc 分析）。

> 📌 **High-Yield（必背）**：**極高風險 → anabolic first（teriparatide/romosozumab）→ sequential**。**Denosumab 停藥必須接 antiresorptive**（驟停＝BMD快速流失＋椎體反彈骨折）。

## 💊 七、Denosumab（Prolia / Xgeva）重點（Amgen 段落）

- 🧪 機轉：**anti-RANKL monoclonal antibody** → 抑制 osteoclast 形成/活性 → 屬廣義骨鬆「**target therapy**」（RANKL＝osteoclast 的 growth factor）。
  - ⚠️ **老師口試題**：「Prolia 是 target therapy 嗎？對抗什麼？」→ **是；對抗 RANKL（monoclonal antibody）**。
- 給藥：**皮下注射，每 6 個月一次** → 方便、**adherence/persistence 最佳**（real-world 持續率高於 bisphosphonate）。
- 療效：2年 hip/spine BMD **約↑8%**；長期治療 BMD 持續上升。
- ⚠️ 停藥陷阱（同上）：停 Denosumab → 須 antiresorptive 銜接。
- 🧠 講者口訣：bisphosphonate **一年 + 接 Denosumab → 效果如同用 Denosumab 七年**（強調 sequential 維持）。
- 價格梗（臨床 SDM）：台灣半年約 **NT$4361（≈一天<25元）**，號稱「全世界最便宜的 monoclonal antibody / target therapy」，可與病人 **shared decision making** 討論自費。

> 📌 **High-Yield（必背）**：**Denosumab = anti-RANKL mAb，SC q6個月**，adherence最佳，2年BMD↑約8%；**停藥必接 antiresorptive**。

## ⚠️ 八、考前快速複習（老師明示／易混總表）

| 考點 | 重點答案 |
|---|---|
| DM 骨鬆本質 | low turnover + **bone quality 差**，BMD 會低估 |
| Type 2 陷阱 ⚠️ | BMD 正常/偏高 ≠ 骨頭好 |
| DM 的 BMD 門檻 | **T ≤ -2.0**（≈ 一般人 -2.5） |
| FRAX 校正 | **+10歲 / 勾secondary / -0.5 SD** |
| 害骨 DM 藥 | **TZD、SU、Insulin**（SGLT2i 現 neutral） |
| 健保給付（未骨折）| **insulin 使用中 + T < -2.5** |
| 極高風險用藥 | **Anabolic first → sequential** |
| Denosumab 停藥 | **必接 antiresorptive，否則 BMD 快速流失/反彈骨折** |
| Prolia 機轉 | **anti-RANKL monoclonal antibody（target therapy）** |
| 身高變矮 cutoff | 自年輕 **≥4 cm** 有意義 |
