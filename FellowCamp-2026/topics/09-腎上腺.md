# Practical Approach to Adrenal Disorders（Case-based）考試摘要
> 來源：FellowCamp 2026 投影片 · Practical Approach to Adrenal Disorders（case-based）· 吳婉禎醫師（臺大醫院內分泌科）

本講以 **5 個 case** 串起腎上腺疾病的高頻考點：**PA → Adrenal incidentaloma/MACS → Cushing → PPGL → CAH**。以下逐 case 整理「臨床線索 → 數據 → 診斷 → 處理 → board pearl」，並把所有可考 cutoff 抽出。

---

## 🧠 全章記憶骨架（先記這個）
- **每顆 adrenal mass 都問兩個問題**：① benign or malignant？② functioning or nonfunctioning？
- **功能性篩檢三大軸**：cortisol（1-mg DST）、aldosterone（ARR）、catecholamine（plasma free metanephrines）。視情況加 sex hormone/steroid precursors。
- 口訣「**PA 看 renin、Cushing 看 DST、PPGL 看 metanephrine、CAH 看 17-OHP**」。

---

# Case 1 — Primary Aldosteronism（PA）

## 🩺 臨床線索
- 63M，HTN since age 40，pre-DM、dyslipidemia。
- **吃了 MRA（Inspra/eplerenone 50 mg）+ Nebilet + Adalat OROS，BP 仍 140-160/90-100 且持續低血鉀**。

## 📊 數據（screening）
- PAC <u>36.73 ng/dL</u>、PRA <u>0.10 ng/ml/hr</u> → **ARR <u>367.3</u>**
- K <u>2.9</u> mmol/L、Na 141、Cre 0.7

> ⚠️ **Board Pearl**：**已在吃 MRA 也不能排除 PA**。即使在 MRA 下 **renin 仍被壓很低（low PRA）就是強陽性訊號**。Normokalemia 也不能排除 PA。

### ⚠️ 偽陽性/偽陰性陷阱（confounder）
- **β-blocker** 可造成偽陽性：當 aldosterone 落在 **10-15 ng/dL** 區間要小心；若 **aldosterone 高於此範圍，即使在 β-blocker 下仍 likely PA**。
- **CKD**：renin 隨 nephron loss 等比例下降、aldosterone 在 CKD 可升高 → **偽陽性**。

## 📊 Confirmation — Seated Saline Suppression Test（SST）
| 項目 | Before | After |
|---|---|---|
| PAC (ng/dL) | 19.13 | **13.99** |
| PRA (ng/ml/h) | 0.03 | — |
| Cortisol (μg/dL) | 6.10 | 3.90 |
| K (mmol/L) | 3.7 | 3.0 |

**SST 判讀 cutoff（seated）**
- Post-saline PAC <u>< 7.8 ng/dL → PA unlikely（排除）</u>
- Post-infusion PAC <u>> 6 ng/dL → confirms PA</u>
- <u>> 16 ng/dL（TAIPAI study group，最高 PPV）</u>
- 本例 after 13.99 → **> 6，確診 PA**（介於 6 與 16 之間）。

## 📊 共病篩檢（concurrent autonomous cortisol secretion）
- 1-mg overnight DST cortisol <u>1.20 μg/dL</u>（>1.8 才算 cortisol autonomy，本例屬正常/borderline）。AVS 前要先排除 cortisol 共分泌，否則會干擾 cortisol 校正的 SI/LI。

## 📊 定位 — Adrenal Venous Sampling（AVS）★最高頻考點

### 三大指數公式與 cutoff（必背表）
| 指數 | 公式 | Cutoff | 意義 |
|---|---|---|---|
| **Selectivity Index (SI)** | [cortisol]_AV / [cortisol]_IVC | 未刺激 <u>>1.4~3</u>；cosyntropin 刺激後 <u>>5</u> | 確認**插管成功** |
| **Lateralization Index (LI)** | (Aldo/Cortisol)_highAV / (Aldo/Cortisol)_lowAV | 未刺激或刺激後 <u>≥4</u>（投影片另列基礎>2、刺激>3-4） | 區分**單側 vs 雙側** |
| **Contralateral Suppression Index (CSI)** | (Aldo/Cortisol)_lowAV / (Aldo/Cortisol)_IVC | 未刺激或刺激後 <u><1</u> | 對側 aldosterone 被**壓抑**（支持單側病灶） |

### 本例 AVS 數值
| | 右 AV | 左 AV | IVC |
|---|---|---|---|
| **SI**（basal / ACTH-S） | 12.7 / 73.9 | 8.3 / 32.7 | — |
| ACR（basal） | 0.23 | 0.68 | 0.99 |

- **SI**：右 12.7、左 8.3（basal >3）；刺激後右 73.9、左 32.7（>5）→ **雙側插管成功** ✅
- **LI（Lt/Rt，左為 dominant）**：basal <u>2.98</u>（>2）、ACTH-S <u>3.94</u>（>3-4）→ **lateralization 到左側** ✅
- **CSI（Rt/IVC，非優勢側）**：basal <u>0.23</u>、ACTH-S <u>0.13</u>（皆 <1）→ **對側受壓抑** ✅

> 🧠 記憶法：**「S 看插管、L 看哪邊、C 看對側乖不乖」**。SI 要**夠大（>3 / >5）**才算抽對；LI **≥4** 才敢說單側；CSI **<1** 表示對側被關掉。

## 💊 處理與結果
- **Laparoscopic left adrenalectomy**（單側病灶 → 手術可能治癒）。
- Post-op 6M：PAC 8.80、PRA 0.75（ARR 12）、K 4.8 → **生化完全成功**。
- Post-op 12M：PAC 10.29、PRA 1.90（ARR 5.4）、K 4.4，仍需 Nebilet → **clinical partial success + biochemical complete success**。

> 📌 **High-Yield（必背）**：① **吃 MRA + 低 renin = 仍要查 PA**；② SST post-saline **<7.8 排除、>6 確診、>16 PPV 最高**；③ AVS：**SI 基礎>3/刺激>5（插管成功）、LI≥4（單側）、CSI<1（對側壓抑）**；④ **CT 不能可靠判斷 laterality，eligible 病人術前常需 AVS**；⑤ 2025 ES CPG 建議**所有 HTN 都篩 PA**；⑥ 單側可治癒、雙側用 MRA。生化成功與臨床成功要分開講。

# Case 2 — Adrenal Incidentaloma 合併 MACS

## 🩺 臨床線索
- 31F，無系統疾病，**右腎上腺 1.1 cm incidentaloma**。
- **不孕 2 年**、體重增 5 kg（50→55）、**無 Cushingoid 外觀**、BP 正常（115/69）。

## 📊 數據
| | ACTH (pg/mL) | Cortisol (μg/dL) |
|---|---|---|
| 8 AM | <5.00 | 7.20 |
| 4 PM | <5.00 | 10.20 |
| **1-mg DST** | | **3.5 與 3.8** |

- PAC 8.64、PRA 0.18、DHEA-S 1.59 → ARR 偏高但非主軸
- 尿 catecholamine：Dopamine 281.3、NE 22.8、Epi 1.2、VMA 3.0（排除 PPGL）

## 🩺 診斷：MACS（Mild Autonomous Cortisol Secretion）
- **1-mg DST cortisol 3.5/3.8 μg/dL** + **ACTH 被壓抑（<5）** + **失去日夜節律（4PM 反高於 8AM）** → **ACTH-independent cortisol autonomy**。
- **DST cortisol 判讀**：<u><1.8 μg/dL 排除自主分泌</u>；<u>1.8-5.0 = MACS（possible autonomy）</u>；<u>>5.0 = autonomous（明確）</u>。

> 🧠 **MACS 重點不只是 DST 數字，而是病人的「臨床負擔」**——本例的不孕、體重、之後發現的骨密度問題即為 cortisol comorbidity。

### 🩺 ESE 2023 incidentaloma 處理原則
- **Comorbidities attributable to cortisol**：DM、HTN、dyslipidemia、osteoporosis。
- **手術**通常只在 comorbidity **符合 ≥1 項**才考慮：progressive、難治、與年齡不符的 end-organ damage、與年齡/家族史不符、或多重共病；並綜合 age、sex、general health、DST 後 cortisol 持續不可壓抑的程度與病人意願。

## 💊 處理與結果
- **右側腹腔鏡 adrenalectomy** + 術後 cortisone replacement 2 週（因對側被抑制，需 GC cover）。
- Post-op 6 週：ACTH 回升（17.5 / 13.3）、cortisol 正常 → HPA 軸恢復。
- Post-op 2M：減重 5 kg、月經正常；3M：**自然懷孕（pregnancy test +）**。
- 後續 DST <1.0（完全恢復）、DHEA-S 回升、BMD 改善。

> 📌 **High-Yield（必背）**：① **無 Cushingoid 外觀 ≠ 無臨床意義的 cortisol excess**；② **1-mg DST 是 MACS 評估核心**，cutoff <u>1.8 / 5.0</u>；③ MACS 可造成**不孕、骨質流失、代謝共病**，手術後可逆；④ 手術需**個別化（MDT）**，不是每個 MACS 都開刀；⑤ 術後因對側 HPA 被抑制 → **需 GC 補充**。

# Case 3 — Cushing's Syndrome（ACTH-independent，腎上腺源）

## 🩺 臨床線索（phenotype-driven！）
- 23M 學生，**低背痛 4 個月**；21 歲起 HTN（Norvasc）、22 歲腿水腫/血尿。
- **HTN + dyslipidemia + DM**（年輕卻多重代謝病）。
- **雙側腎結石 + 左輸尿管結石**（s/p ESWL）。
- **Kyphosis：身高 170→157 cm，多處 compression fracture**。
- BMD：腰椎 L1-L4 total T-score <u>-4.1（osteoporosis）</u>、femoral neck T-score <u>-2.3（osteopenia）</u>。
- PE：**moon face、facial plethora、central obesity、purple striae**。

> 🩺 標題即考點：**「次發性骨鬆 workup 要 phenotype-driven，不能只看 lab」**——年輕男性嚴重骨鬆 + 多重代謝病 + 結石，要想到 Cushing。

## 📊 數據（確診內源性高皮質醇）
| | ACTH (pg/mL) | Cortisol (μg/dL) |
|---|---|---|
| 8 AM | 6.59 | 21.30 |
| 4 PM | <5.00 | 20.00 |
| 11 PM | | **18.40**（失去日夜節律） |
| **1-mg DST** | | <u>**21.40（不被壓抑）**</u> |
| **24-hr UFC** | | <u>**521.3 μg/24hr（顯著升高）**</u> |

- Ca 2.49、P 2.7、iPTH 36.6、25-OH D 19.1（鑑別其他骨鬆病因）。

## 🩺 診斷邏輯
- **先排除外源性 GC**（exogenous glucocorticoid 必須先排除）。
- **First-line 篩檢三選（任二陽性）**：1-mg overnight DST、late-night salivary cortisol（LNSC）、24-hr UFC。
- 確認內源性後，**ACTH 是關鍵分岔點**：本例 ACTH 低（<5~6.59）+ CT 右腎上腺 mass → **ACTH-independent，腎上腺源**。

## 💊 處理與結果
- **Laparoscopic right adrenalectomy**。
- 術後 **cortisone 補充 9 個月**（對側長期被抑制，恢復慢）；**DM、HTN、dyslipidemia 全部 resolved**。

> 📌 **High-Yield（必背）**：① 在**年輕脆性骨折/次發性骨鬆、近端肌無力、facial plethora、易瘀青、purple striae** 要想到 CS；② 篩檢必排**外源 GC**；③ **三大篩檢=DST/LNSC/UFC**；④ **DST 不被壓（本例 21.4）+ UFC 高（521）= 內源性 hypercortisolism**；⑤ **ACTH 低 → ACTH-independent → 找腎上腺**；⑥ 單側 adrenalectomy 術後**需長期 GC cover**（本例 9 個月）。

# Case 4 — Pheochromocytoma / PPGL

## 🩺 臨床線索
- 39F（acoustic neuroma s/p gamma knife、MVP、GERD）。
- **右 supra-renal 大腫瘤（5.2 cm，hypervascular，壓迫肝臟）**。
- 症狀**只有偶發 palpitation，BP 正常（96-112/70-80）** → ⚠️ **normotensive 也可能是 PPGL**。

## 📊 數據（catecholamine / metanephrine）
| 24h 尿 | 值 |
|---|---|
| Dopamine | 285.8 μg |
| **NE** | **438.8 μg ↑** |
| Epinephrine | <0.7 |
| **VMA** | **15.8 mg ↑** |
| **Free-NMN（尿）** | **1160.1 μg ↑↑** |
| Free-MN（尿） | 5.8 |

| 血漿 | Free-NMN (ppb) | Free-MN (ppb) |
|---|---|---|
| Seated | **9.156 ↑** | 0.033 |
| Supine | **6.313 ↑** | 0.033 |

- **Chromogranin A 1565.0 ng/mL ↑**（PPGL 標記）。
- 正常腎上腺皮質荷爾蒙；syndromic PPGL（MEN2、VHL、NF1）臨床評估陰性。
- 生化型態：**NE/NMN 為主（noradrenergic phenotype）**，與 SDHx 相關表現一致。

## 🩺 影像與定位
- CT 大右腎上腺腫瘤；**⁶⁸Ga-DOTATOC PET/CT** 定位（SSTR-based，對 SDHx 相關 PPGL 敏感）。
- 胰臟病灶 EUS-FNB → IPMN（非 NET，附帶發現）。

## 💊 處理與基因
- **Right laparoscopic adrenalectomy**；病理 pheochromocytoma，**mitotic 1/10 HPF、Ki-67 <1%**。
- 基因：**SDHD c.3G>C, p.M1I, heterozygous, pathogenic**（起始密碼子突變）；FH p.V139M = VUS。
- Pedigree 支持遺傳性（PPGL/HTN/MNG/DM/dyslipidemia 家族聚集）。

## 🩺 PPGL 一般原則
- 懷疑時機：adrenal mass ± headache、palpitation、diaphoresis、labile HTN（典型三聯：頭痛+心悸+流汗）。
- **初始生化檢查＝plasma free metanephrines 與 urinary fractionated metanephrines**（生化要先於 invasive procedure）。
- **2022 WHO 分類**：取消「pheochromocytoma vs paraganglioma」二分，**統一稱 paraganglioma**，腎上腺者為 adrenal paraganglioma（"pheochromocytoma"）；分 sympathetic abdominal / sympathetic H&N / parasympathetic（carotid body 等）。

> ⚠️ **Board Pearl（必背）**：
> - **Never biopsy an adrenal mass before excluding PPGL**（切片前必先排除 PPGL，否則可能誘發危象）。
> - **Negative family history 不能排除 hereditary PPGL** → 應做 **germline testing**（影響追蹤與家族篩檢）。
> - **所有 PPGL 都有 metastatic potential → 終生追蹤**。
> - 生化確診**先於** invasive procedure；**生化先行，基因很重要**。

> 📌 **High-Yield**：normotensive 也可有 PPGL；**plasma free metanephrines** 為首選；定位可用 CT/MRI + **DOTA / MIBG**；治療順序＝**α-blockade 先 → 再手術**。

# Case 5 — Congenital Adrenal Hyperplasia（CAH，21-OHD，非典型）

## 🩺 臨床線索
- 29F，**20 歲起月經不規則**，OCP/中藥反應差；GYN echo **無多囊卵巢**（鑑別 PCOS）。
- **Hirsutism**。

## 📊 數據
**Cosyntropin（250 μg）刺激試驗**
| | 0 min | 30 min | 60 min |
|---|---|---|---|
| Cortisol (μg/dL) | 8.50 | 9.90 | 10.00（刺激後上升不足） |
| **17-OHP (nmol/L)** | **>39.6** | **>39.6** | **>39.6** |

- Baseline：8AM ACTH 79.47、cortisol 8.00、DHEA-S 15.9、**Testosterone 1.82**、**17-OHP 39.6**、Androstenedione 39.86（hyperandrogenism）。
- FSH 2.73 / LH 2.62 / E2 81.07 / P4 4.35。

## 🩺 診斷判讀（17-OHP cutoff★必背）
- **Basal 8AM 17-OHP**：<u>< 6 nmol/L（~<200 ng/dL）→ unlikely CAH</u>；<u>> 6 nmol/L → 做 cosyntropin 刺激</u>。
- **Cosyntropin 刺激後 17-OHP**：<u>> 30 nmol/L（~>1000 ng/dL）→ 確診 classic/non-classic 21-OHD</u>。本例 **>39.6（爆表）→ 確診**。
- ⚠️ 換算：**1 ng/dL ≈ 0.03 nmol/L**；30 nmol/L ≈ 1000 ng/dL。

## 🧪 基因
- **CYP21A2 c.92C>T, p.P31L, pathogenic** + **CYP21A2 exon 1 large deletion**（MLPA 顯示 utr5/exon1/intron1 劑量 0.5 = 半合子缺失）→ compound heterozygous。

## 🩺 CAH 一般原則（成人 IM 重點）
- **最常見＝21-hydroxylase deficiency**，**genotype–phenotype correlation**。
- 成人 IM 最常遇到：**transition care、fertility、長期 GC 過度治療、代謝併發症**。
- ⚠️ **無症狀、非懷孕的 non-classic CAH 成人通常不需常規 GC 治療**（避免過度治療）；但孕前規劃與專科追蹤重要。

> 📌 **High-Yield（必背）**：① hyperandrogenism + 月經不規則 + **無多囊卵巢** + DHEA-S/17-OHP/androstenedione 升高 → 想 **non-classic CAH**；② **17-OHP basal >6 nmol/L 做刺激、刺激後 >30 nmol/L 確診**；③ **不要過度治療 non-classic disease**。

# Final Board Pearls 總表（一頁背完）

| Condition | Screening | Confirmation / Localization | Management | ⚠️ Pearl |
|---|---|---|---|---|
| **PA** | ARR（PAC/PRA） | CCT / SST；CT / **AVS** | Surgery 或 MRA | **多篩、不要只信 CT** |
| **Adrenal incidentaloma** | 臨床 + **1-mg DST**；ARR*、plasma free MNs*、sex hormone/steroid precursors* | 依影像特徵與荷爾蒙個別化 | Follow-up 或 surgery | 永遠問 **benign/malignant** 與 **functioning/nonfunctioning** |
| **MACS** | **1-mg DST** | 找 cortisol 相關共病 | 個別化、MDT 決定手術/追蹤 | **生化 cortisol autonomy 仍可有臨床意義** |
| **CS** | **1-mg DST / LNSC / UFC** | **ACTH 分岔點** / CT | Adrenalectomy + 術後 GC cover | 在**次發性骨鬆與非典型代謝負擔**要想到 |
| **PPGL** | **Plasma / urine free MNs** | CT / MRI；**DOTA / MIBG** | **α-blockade → surgery** | **生化先行、基因重要**；切片前先排除 |
| **CAH** | **17-OHP（21-OHD）** | **Cosyntropin 刺激試驗 / genetics** | 個別化、GC 或 anti-androgen | **不要過度治療 non-classic** |

## 🧠 終極記憶口訣
- **「篩檢一句話」**：PA→**ARR/低renin**；incidentaloma/MACS/CS→**1-mg DST**；PPGL→**free metanephrines**；CAH→**17-OHP**。
- **四個「先後」陷阱**：PPGL **先生化再切片/手術**、PPGL **先 α-block 再開刀**、CS **先排外源 GC 再驗**、PA **AVS 前先排 cortisol 共分泌**。
- **三個「不能排除」**：吃 MRA 不排除 PA、無 Cushingoid 不排除 cortisol excess、無家族史不排除 hereditary PPGL。

> 📌 **High-Yield（全章收斂）**：cutoff 必背＝**SST post-saline <7.8 排除 / >6 確診 / >16 PPV最高**；**AVS SI >3(基礎)·>5(刺激)、LI ≥4、CSI <1**；**DST <1.8 / 1.8-5.0(MACS) / >5.0**；**17-OHP basal >6 nmol/L→刺激、刺激後 >30 nmol/L 確診**；**Never biopsy adrenal mass before excluding PPGL**。
