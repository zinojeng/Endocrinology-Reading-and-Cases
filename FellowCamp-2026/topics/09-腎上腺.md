# 腎上腺疾病（Adrenal Disorders）考試精華摘要

> 來源：FellowCamp 2026 · Adrenal Disorders · 吳婉禎醫師

本摘要以講者 5 個 case 為主軸，涵蓋 **原發性醛固酮增多症（PA）、腎上腺偶發瘤（Adrenal Incidentaloma）、Cushing syndrome、嗜鉻細胞瘤／副神經節瘤（PPGL）、先天性腎上腺增生（CAH）**。腎上腺疾病多屬罕見，考試重點在「篩檢 → 確診 → 定位 → 治療」的流程與各 cutoff。

---

## 1️⃣ 原發性醛固酮增多症 Primary Aldosteronism（PA）

🩺 **典型 case**：63 歲男性，40 歲起高血壓，用 3 種降壓藥（含 MRA）仍控制不良 + 持續低血鉀。

### 🩺 何時要篩檢（screening indication）
- **Resistant hypertension**（即使已用 MRA 仍控制不良）
- **Hypertension + hypokalemia**（持續低血鉀）
- **Adrenal incidentaloma + 高血壓**
- **Early-onset hypertension**（年輕高血壓）
- ⚠️ <u>**真正 PA 會出現 hypokalemia 的比例不到 40%**</u>，所以**正常血鉀不能排除 PA**。
- ⚠️ Resistant hypertension 中約 **1/3** 測起來是 PA（過去低估）。
- 📌 **2025 新 guideline 精神**：傾向「**所有高血壓都篩 PA**」，且為求簡便 → **可在用藥下先篩一次**，不必先停藥 2–4 週；若用藥干擾判讀，再做 minimal/maximal switch strategy。（此「全篩」建議仍有爭議 ⚠️）

### 📊 篩檢（screening）= ARR
- 測 **Aldosterone（PAC）+ Renin（PRA/DRC）→ 算 ARR（aldosterone-renin ratio）**。
- 🧪 **Pre-analytic 條件（必背、易考）**：
  - <u>**早上、坐姿（站坐 ≥ 15 分鐘）**</u>
  - **不要限鹽**
  - **先校正低血鉀**（hypokalemia 會壓低 aldosterone → false negative）
- 🧠 **Renin 低 = 最重要的 key clue**（即使在 MRA 下，renin 仍低 + 鉀仍低 = 很強的 positive signal）。

### ⚠️ 藥物對 ARR 的干擾（易混考點）
- **β-blocker**：抑制 RAS → 壓低 renin → ARR 易 **false positive**；但 β-blocker 也會壓低 aldosterone，**若 aldosterone 仍很高 → 不能只歸因 β-blocker**，可繼續往下查。
- **MRA（spironolactone/eplerenone）**：會升高 renin；但講者強調 **即使用 MRA、鉀仍低且 renin 仍低 → 仍不能放棄篩 PA**。

### 📊 確診試驗（新名稱：Aldosterone Suppression Test）
- ⚠️ **舊稱 confirmation test，新 guideline 改稱 aldosterone suppression test**（考點）。
- 本 case 用 **saline infusion test**：輸注後 PAC 仍高 → 確認 aldosterone 未被抑制（autonomy）。
- 新舊 guideline 在 **cutoff 上有差異**（⚠️ 回查 2024/2025 PA guideline）。

### 🩺 定位（Subtyping）→ 決定治療
- **CT** 看到 nodule ≠ 一定是分泌醛固酮的元兇 → 需 **AVS（Adrenal Venous Sampling）** 證實。
- 📌 看到小 nodule 時，因 **PA 約 3 成合併 cortisol co-secretion** → 應同時做 **overnight DST** 排除 **MACS**。
- ⚠️ NP-59 scan 過去可輔助定位，**目前停做**。

### 📊 AVS 三大 Index（考點，必懂意義）

| Index | 看什麼 | 判讀 |
|---|---|---|
| **Selective index** | 導管有無勾到 adrenal vein（cannulation 成功）| 先過這關才判讀；**右側 adrenal vein 直接接 IVC、最難找** ⚠️ |
| **Lateralization index** | 高側 vs 低側的 aldosterone/cortisol ratio | 舊：baseline >2、ACTH 刺激 >3–4；<u>**新 guideline 一律 >4**</u> ⚠️ |
| **Contralateral suppression index** | 對側是否被抑制 | <u>**< 1**</u> 支持單側 lateralization |

### 💊 治療
- **Lateralization（unilateral / APA）→ 手術（adrenalectomy）**，可 curable 或 improve。
- **Non-lateralization / bilateral → MRA 藥物治療**。
- 📊 術後評估用 **PASO（Primary Aldosteronism Surgical Outcome）標準**：
  - **Clinical**：血壓改善但仍需藥 → partial success；完全停藥血壓正常 → complete。
  - **Biochemical**：鉀與 ARR 正常 → complete success。
- 本 case 術後 12 個月 ARR 完全正常 = complete success。

> 📌 **High-Yield（必背）**：高血壓 control 不良 / 持續低血鉀 / incidentaloma / early-onset → 篩 PA。<u>正常血鉀不能排除（PA 中低血鉀 <40%）</u>。篩檢 = 早上坐姿、不限鹽、校正低鉀後測 ARR；**renin 低是 key**。CT 看到瘤 ≠ 元兇 → 想開刀必做 **AVS**；<u>新 lateralization index 一律 >4，contralateral suppression <1</u>。單側 → 手術，雙側 → MRA。「confirmation test」已改名 **aldosterone suppression test** ⚠️。

## 2️⃣ 腎上腺偶發瘤 Adrenal Incidentaloma

🩺 **典型 case**：31 歲女性，腹痛做 CT 意外發現右側小腺瘤；不孕 2 年、體重略增、無 Cushingoid 外觀、血壓正常 → 查出 ACTH 低、cortisol 失去 diurnal change、兩次 overnight DST 均 >1.8、DHEA-S 低 → **MACS**。

### 🩺 遇到 incidentaloma 一定問兩個問題
1. **良性 or 惡性？** → 看 **影像特徵（HU 值）與大小**。
2. **有沒有 function？**

### 🧪 荷爾蒙 survey（2023 guideline）
- **一定先測 cortisol**（即使無 Cushingoid 外觀）→ **1 mg overnight DST**。
- **若有 hypertension** → 加測 **PPGL（metanephrines）+ PA（ARR）**。
- 講者實務：常常一次全 survey。

### 📊 MACS（Mild Autonomous Cortisol Secretion）
- 定義：**生化上 cortisol autonomy，但無 Cushingoid 臨床症狀**。
- 診斷依 **overnight DST**：<u>**1 mg DST 後 cortisol > 1.8 µg/dL**</u> 為 positive（無 Cushingoid 外觀 → 落在 MACS；本 case DHEA-S 低、ACTH 低為佐證）。
- ⚠️ **並非所有 MACS 都要開刀** → 要評估有無 cortisol-related comorbidity。
  - 相關 comorbidity：**DM、hypertension、dyslipidemia、osteoporosis**。
  - ⚠️ 陷阱：這些 comorbidity 在一般人 prevalence 本就高，**難確認是否真由 cortisol 造成** → 需 shared decision making。

### 💊 手術考量
- 綜合：comorbidity 嚴重度、**年齡、性別、general condition、病人意願**。
- 🩺 本 case 雖年輕、indication equivocal，但病人意願 → 開刀；術後 6 週 ACTH 回升、月經規則、後續懷孕 → 證實 MACS 由該瘤造成。

> 📌 **High-Yield（必背）**：偶發瘤問「良惡性 + 有無功能」。**一律先測 cortisol（即使無 Cushingoid）→ 1 mg overnight DST，cutoff <u>>1.8 µg/dL</u>**；有高血壓加測 PPGL + PA。MACS = 生化 autonomy 無症狀，**不一定開刀**，依 comorbidity + 病人因素 individualize。**不建議對 PPGL/aldosterone 瘤做 biopsy**（見下）。

## 3️⃣ Cushing Syndrome（腎上腺型）

🩺 **典型 case（秒殺題級）**：23 歲研究所學生，下背痛 4 個月；**21 歲就高血壓**（只開 Norvasc 未 survey）→ 一年後 leg edema、下背痛、hematuria、雙側腎結石、新發 DM + dyslipidemia；**身高 170→157 cm**、kyphosis、multiple compression fracture、Z-score 極低；門診見 **moon face、facial plethora、purple striae** → cortisol 全高、無 diurnal、UFC 高 → 右側 dark adenoma。

⚠️ **老師反覆強調的考點**：**年輕就高血壓（21 歲）若早 survey，就不會走到不可逆的 compression fracture**；compression fracture 治療後 **身高不可逆**（BMD 可恢復、身高不會）。

### 🩺 何時懷疑 Cushing
- 年輕就出現不符年齡的 **metabolic abnormality**（DM、HTN、dyslipidemia）
- 年輕 **osteoporosis / compression fracture、muscle weakness**
- 外觀：**moon face、facial plethora、purple striae、central obesity**

### 🧪 診斷流程
1. ⚠️ **第一件事一定先排除 exogenous（外源性）**！
   - 吃的、打的、吸的、**塗的**——各種途徑類固醇都會造成 Cushingoid。
   - 真正 endogenous Cushing 是 **rare disease**。
2. 📊 **三大初篩試驗**（至少 **兩項異常**即可診斷）：
   - **Overnight (1 mg) dexamethasone suppression test**
   - **Late-night salivary cortisol**
   - **24-hr urine free cortisol（UFC）**
3. 確診後測 **ACTH** 分流：
   - **ACTH-independent（低）→ 腎上腺問題** → 影像 + 手術。
   - ACTH-dependent → 上游（pituitary/ectopic）問題。

### 💊 術後處理（必考）
- 📌 **Adrenal Cushing 術後一定要補 cortisol**：因 ACTH 長期被抑制 → **對側腎上腺 atrophy** → 防 **steroid withdrawal / 腎上腺危象**。
- 補充時間因人而異：**數月～2–3 年**，視對側恢復而定。
- 術後 DM/HTN/dyslipidemia/BMD 可恢復；**身高（compression fracture）不可逆**。

> 📌 **High-Yield（必背）**：年輕 + 不符年齡的 osteoporosis/骨折/代謝異常 + Cushingoid 外觀 → 想 Cushing。**先排除外源性類固醇**。初篩三選項（**1 mg DST、late-night salivary cortisol、24h UFC**），<u>**≥2 項異常即診斷**</u> → 再測 ACTH 分 dependent/independent。**Adrenal Cushing 術後務必補充 cortisol（對側 atrophy）**。

## 4️⃣ 嗜鉻細胞瘤／副神經節瘤 PPGL

🩺 **典型 case**：39 歲女性，腹痛，echo→CT 見 **5 cm hypervascular** 腎上腺腫瘤；偶 palpitation、血壓正常 → 尿液以 **norepinephrine/normetanephrine 為主**、**Chromogranin A 很高** → laparoscopic adrenalectomy 確診 pheochromocytoma；基因 **SDHD** germline → hereditary paraganglioma syndrome（Cluster 1A）。

### 🧪 篩檢 / 診斷
- 📊 **首選初篩生化**：<u>**plasma free metanephrines** 或 **urine fractionated metanephrines**</u>。
- 📌 **生化即可診斷，不需 biopsy**；⚠️ **對 PPGL 做 adrenal biopsy 可能誘發 hypertensive crisis**（禁忌，常考）。
- **Chromogranin A**：neuroendocrine tumor marker，可輔助（非必高）。
- 影像：hypervascular；可做 functional imaging（本 case 用 **DOTATATE/DOTATOC + FDG PET**）。
  - 🧠 判讀小技巧：**Brain 不亮 = DOTA；Brain 很亮 = FDG**。

### 🧬 遺傳（高頻考點）
- ⚠️ <u>**約 1/3 的 PPGL 為 hereditary（germline pathogenic variant）**</u>，**即使無家族史、發病年齡不小也不能排除**。
- 📌 **現在建議所有 PPGL 病人都做 germline genetic testing**。
- Syndromic 範例：**MEN2、VHL、NF1**。
- **SDHx / Cluster 1A 特色**：
  - **Multifocal**、可長在 extra-adrenal（paraganglioma）也可在 adrenal
  - 荷爾蒙以 **norepinephrine（normetanephrine）為主**
  - ⚠️ **SDHD 有 parent-of-origin 效應**：<u>**父系遺傳才會發病，母系幾乎不發病**（僅 case report）</u> → **此非 imprinting，是特殊機制**（易混陷阱）。
  - germline (+) → 需 **family counseling**，父母／手足都應受測。

### 🩺 追蹤與命名
- 📌 **所有 PPGL 都有 metastatic potential → 需 lifelong surveillance**（講者親見 20 年後復發）。
- ⚠️ **命名（2022 WHO）**：建議**一律稱 paraganglioma**，再標部位（如 adrenal paraganglioma）；但臨床仍習慣稱 PPGL / pheochromocytoma。
- 📈 趨勢：以往多以症狀診斷，**現逾半數以 adrenal incidentaloma 被發現**。

> 📌 **High-Yield（必背）**：初篩 = **plasma free metanephrines 或 urine fractionated metanephrines**；**不可 biopsy（誘發危象）**。<u>**1/3 為 hereditary → 所有 PPGL 都做 germline testing**</u>，無家族史也不能排除。**SDHD = parent-of-origin（父系發病、非 imprinting）**。**所有 PPGL 終身追蹤**（有 metastatic potential）。2022 WHO 一律稱 paraganglioma。

## 5️⃣ 先天性腎上腺增生 CAH（Non-classic）

🩺 **典型 case**：29 歲女性，20 歲起月經不規則、hirsutism，避孕藥/中藥無效，**echo 無 PCO** → 測得 **DHEA、testosterone 高**，但 **ACTH 高、cortisol low-normal** → 17-OHP 與 androgen 高、**ACTH stimulation test 證實 primary adrenal insufficiency + 17-OHP 高** → **CYP21A2** 變異（一側 point mutation、一側 deletion）= **non-classic CAH**。

### 🧠 診斷邏輯（考點）
- 🧪 **同一腎上腺皮質「三層方向不一致」= 重要線索**：**glucocorticoid 不足，但 androgen 偏高** → 考慮 **CAH（即使成人）**。
- 最常見 = **21-hydroxylase deficiency（CYP21A2）**；**autosomal recessive**。
- 🧪 篩 / 確診：**17-OHP（基礎 + ACTH stimulation test）** 升高。
- **Genotype–phenotype correlation**：
  - **Salt-wasting（嚴重 enzyme deficiency）→ 小時候發病**。
  - **Non-classic → enzyme activity 仍有約 20–50% → 不一定小時候發病**，常以成人 hyperandrogenism（似 PCOS）表現。

### 💊 治療要點（易混）
- ⚠️ **Non-classic、nonpregnant、asymptomatic → 不一定需要 routine glucocorticoid replacement**（**避免 over-treat**）。
- 一旦有生育需求（女性結婚備孕）→ 再考慮荷爾蒙補充與治療。
- 🩺 因 **androgen 高 → 注意 cardiometabolic complication**；長大後需 **transitional care** 與 fertility 議題。

> 📌 **High-Yield（必背）**：女性 hyperandrogenism + 「**glucocorticoid 低、androgen 高（皮質各層方向不一致）**」→ 想 **non-classic CAH**。最常見 **21-OH（CYP21A2）缺乏、AR 遺傳**；測 **17-OHP（含 ACTH stimulation test）**。<u>**Non-classic 無症狀者不必常規補 glucocorticoid（勿 over-treat）**</u>，備孕時再治療。Salt-wasting 嚴重型小時候發病（genotype-phenotype correlation）。

## 📋 總整理：各腎上腺疾病 篩檢 → 確診 → 治療

| 疾病 | 篩檢（screening）| 確診 / 定位 | 治療要點 |
|---|---|---|---|
| **PA** | ARR（早上坐姿、不限鹽、校正低鉀）；renin 低為 key | Aldosterone suppression test（如 saline infusion）→ **AVS** 定位 | 單側→手術；雙側→MRA |
| **Adrenal incidentaloma** | **1 mg overnight DST**（必測 cortisol）；+HTN 加 PPGL、PA | DST <u>>1.8</u> → MACS；影像辨良惡 | MACS 視 comorbidity individualize |
| **Cushing** | 1 mg DST / late-night salivary / 24h UFC（先排外源性）| **≥2 項異常**→測 ACTH 分流 | Adrenal Cushing 術後**補 cortisol** |
| **PPGL** | **plasma free / urine fractionated metanephrines** | 生化即診斷（**勿 biopsy**）+ germline testing | adrenalectomy；終身追蹤 |
| **CAH (non-classic)** | 17-OHP（基礎 + ACTH stim） | CYP21A2 基因；皮質各層方向不一致 | 無症狀勿常規補 GC；備孕再治 |

## 🧠 記憶法 / 易混陷阱速記
- **「正常血鉀 ≠ 排除 PA」**（PA 低血鉀 <40%）。
- **PA 看到瘤 ≠ 元兇 → AVS**；新 lateralization index **一律 >4**，contralateral suppression **<1**。
- **DST cutoff 1.8 µg/dL**（MACS / Cushing 初篩）。
- **Cushing 先排外源性 → ≥2 項初篩異常診斷 → ACTH 分 dependent/independent**。
- **Adrenal Cushing 術後補 cortisol**（對側 atrophy）。
- **PPGL：1/3 hereditary、勿 biopsy、終身追蹤、SDHD 父系發病（非 imprinting）**。
- **CAH：glucocorticoid 低 + androgen 高 = 三層方向不一致 → 想 CAH；non-classic 勿 over-treat**。
