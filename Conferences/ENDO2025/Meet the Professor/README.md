## 使用說明

- 建議先從分類頁快速瀏覽同一主題下的 case，再回到原課程或原書進行完整學習。
- 摘要以「臨床問題、診斷路徑、治療決策與照護陷阱」為導向，不取代原始 CME 教材。
- 個別 case 頁的 recording-derived 摘要來自 website recording 字幕與 full-frame slide OCR；未取得 media 的 case 會在頁面中明確標示。
